盒子模型

  • 标准盒子模型（content-box，默认值）
    • 计算方式：总宽度 = width（内容区） + padding×2 + border×2

    总高度 = height（内容区） + padding×2 + border×2

    • 特点：width 和 height 仅表示内容区大小，padding 和 border 会增加盒子的整体尺寸。
    • 适用场景：需要精确控制内容区大小时（如图片容器）。
    box-sizing: content-box
  • 怪异盒子模型（border-box）（改变宽高计算方式）
    • 计算方式：总宽度 = width（内容区 + padding + border）

    总高度 = height（内容区 + padding + border）

    • 特点：width 和 height 包含内容区、padding 和 border，整体尺寸固定，不会因内边距或边框而撑开。
    • 适用场景：响应式布局或需要固定尺寸的容器（如自适应盒子）。
    box-sizing: border-box /* 可以自己计算 边距padding*/
  • 盒子模型的关键特性
    1. 背景显示范围
      • 背景色 / 背景图会覆盖内容区、内边距，但不会延伸到外边距。
      • 示例：设置 background-color: #f0f0f0; 时，内边距区域会显示该颜色，外边距区域透明。
    1. 外边距合并（Margin Collapsing）
      • 垂直方向：相邻块级元素的上下外边距会合并为两者中的最大值（父子元素也可能发生，需通过 padding 或 border 分隔）。
      • 水平方向：外边距不会合并，始终叠加。
      • 解决方法：
        • 给父元素添加 padding 或 border 避免父子合并。
        • 使用 overflow: hidden 触发 BFC（块级格式化上下文）。
    1. 行内元素的盒子模型限制
      • 行内元素（如 <span>）默认不支持 width/height，且垂直方向的 margin/padding 不会影响其他元素（但会显示背景色）。
      • 如需控制行内元素的尺寸，需设置 display: inline-block 或 display: block。
    • 问题：

      1.外边距折叠现象 - 01 合并现象

      垂直布局 和 块级元素，上下的margin会合并。最终结果为取margin最大的值。所以只给其中一个盒子margin值就行

      2.外边距折叠现象 - 02 塌陷现象

      互相嵌套 的 块级元素，子元素的margin-top会作用在父元素上。导致父元素一起移动。

      解决方法：

      1. 给父元素设置border-top或者padding-top（分离父子元素）

      2.给父元素设置overflow：hidden

      3.转换成行内块元素

      4.设置浮动

      3.行内元素 的内外边距问题（margin和padding）

      如果想要通过margin或padding改变行内标签的垂直位置，无法生效。即margin-top和botton不生效，padding-top和botton不生效。

      解决方法：

      1.添加line-height：100px

      2.改变布局非行内元素

  • 内边距

    内边距通过padding-top/right/bottom/left设置

    .box {
  padding-top: 10px;    /* 上内边距10px */
  padding-right: 20px;  /* 右内边距20px */
  padding-bottom: 15px; /* 下内边距15px */
  padding-left: 5px;    /* 左内边距5px */
}
--------------------------
.box { padding: 10px 20px 15px 5px; } 
/* 等价于：上10px、右20px、下15px、左5px */

.box { padding: 10px 20px 15px; } 
/* 等价于：上10px、左右20px、下15px */

.box { padding: 10px 20px; } 
/* 等价于：上下10px、左右20px */

.box { padding: 10px; } 
/* 等价于：上下左右均为10px */
  • 边框

    通过border-top/right/bottom/left控制

    属性作用取值示例
    border-width边框宽度thin（细）、medium（中，默认）、thick（粗）、10px、2em 等
    border-style边框样式（必须设置，否则默认无样式）solid（实线）、dashed（虚线）、dotted（点线）、double（双线）等
    border-color边框颜色颜色名（red）、十六进制（#ff0000）、RGB（rgb(255,0,0)）等，默认继承文本颜色
    .box {
  /* 上边框：2px 实线 红色 */
  border-top-width: 2px;
  border-top-style: solid;
  border-top-color: red;

  /* 右边框：4px 虚线 蓝色 */
  border-right: 4px dashed blue; /* 简写形式 */

  /* 下边框：仅设置样式（宽度和颜色用默认值） */
  border-bottom-style: dotted;

  /* 左边框：隐藏（宽度为0或样式为none） */
  border-left-width: 0;
}
  • 外边距
    • margin
      /* 单值语法：所有方向的margin相同 */
margin: 10px;      /* 上下左右均为10px */

/* 双值语法：垂直 | 水平 */
margin: 10px 20px; /* 上下10px，左右20px */

/* 三值语法：上 | 水平 | 下 */
margin: 10px 20px 30px; /* 上10px，左右20px，下30px */

/* 四值语法：上 | 右 | 下 | 左（顺时针） */
margin: 10px 20px 30px 40px; /* 上10px，右20px，下30px，左40px */

/* 负值：可使元素与其他元素重叠 */
margin: -5px;

/* 百分比：相对于父元素的宽度计算 */
margin: 5%;

/* 关键字 */
margin: auto; /* 水平居中（仅对块级元素有效） */

/* 单独定向 */
margin-top: 10px;    /* 上外边距 */
margin-right: 20px;  /* 右外边距 */
margin-bottom: 15px; /* 下外边距 */
margin-left: 5px;    /* 左外边距 */

/* 指定方向扩展，甚至可以覆盖其他元素 */
.box {
  margin-top: -20px; /* 元素向上移动20px，覆盖上方元素 */
}

/* 百分比计算 */、
.parent {
  width: 500px;
}
.child {
  margin-top: 10%; /* 实际为 500px × 10% = 50px */
}

/* 对块级元素可以实现水平居中 */
.container {
  width: 800px;
  margin: 0 auto; /* 左右自动均分剩余空间，实现居中 */
}

/* 外边距合并 */
/* 相邻块级元素的垂直 margin（上与下）会合并为一个，取其中较大的值。 */
<style>
  .box1 { margin-bottom: 20px; }
  .box2 { margin-top: 30px; }
</style>
<div class="box1">Box 1</div>
<div class="box2">Box 2</div>
<!-- 两元素实际间距为30px（较大值），而非20px+30px=50px -->

    外边距通过 margin-top/right/bottom/left控制

    .box {
  margin-top: 20px; /* 上外边距20px */
  margin-right: 1em; /* 右外边距1em（相对于父元素字体大小） */
  margin-bottom: 5%; /* 下外边距：父元素宽度的5% */
  margin-left: auto; /* 左外边距自动计算 */
}
-----------------------------------
1 个值：margin: 10px; → 四个方向均为 10px
2 个值：margin: 10px 20px; → 上下 10px，左右 20px
3 个值：margin: 10px 20px 30px; → 上 10px，左右 20px，下 30px
4 个值：margin: 10px 20px 30px 40px; → 上→右→下→左（顺时针顺序）

    1、兄弟关系：垂直方向外边距取最大值，水平外边距相加合并。

    2、父子关系：子margin-top⇒父的padding-top，需要手动计算高度变化。

    给父盒子设置边框，也需要算高度。

  • 盒子样式
    1、盒子阴影
box-shadow: 10px 10px 10px 10px #ff0 inset;          /* 盒子阴影 可以设置多个*/
/* 第四个参数为阴影大小， 最后一个参数为设置为内阴影 */

2、圆角边框
border-radius: 10px/70px 20px 30px 40px; /* 10px/70px 水平/垂直 盒子模型边的一半px时为最大，正方形此时为圆 */ /* 左上 右上 右下 左下*/
border-radius: 50%; /*自动占一半*/

3、透明度
ul li:not(:nth-child(7)){ /* 除了第7个，全部透明 */
    opacity: 0;
}
ul:hover li {
    opacity: 1;
