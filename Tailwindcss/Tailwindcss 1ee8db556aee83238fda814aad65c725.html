Tailwindcss

Tailwind CSS 是一款 实用优先（Utility-First） 的 CSS 框架，核心思想是通过预定义的原子化工具类直接编写 HTML 样式，无需编写传统 CSS 文件。结合 Vite + Vue 使用时，能极大提升开发效率，同时保持样式灵活性。

  • 安装（以vite/vue项目为例）
    npm create vite@latest my-tailwindcss
cd my-tailwindcss
npm install tailwindcss @tailwindcss/vite
// 项目启动
npm rum dev

    配置vite.config.js

    import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import tailwindcss from '@tailwindcss/vite'

// https://vite.dev/config/
export default defineConfig({
  plugins: [
    vue(),
    tailwindcss()
  ],
})

语法核心：Utility-First 工具类

Tailwind 的核心是 原子化工具类，每个类对应一个具体的 CSS 样式

所有工具类遵循统一命名规则：属性-值

  • 1.命名规则详解
    结构示例对应 CSS说明
    属性-值text-xlfont-size: 1.25rem;基础工具类
    属性-变体-值hover:text-blue-500hover { color: #3b82f6; }状态变体（hover/focus 等）
    属性-响应式-值md:text-2xl@media (min-width:768px) { font-size:1.5rem; }响应式断点
    属性-变体-响应式-值hover:md:bg-green-500@media (min-width:768px) { hover { background-color:#22c55e; } }
  • 2.基础工具类分类
    • 1.布局相关（Layout）

      控制盒模型、定位、浮动、弹性布局等。

      工具类功能示例对应 CSS
      block块级元素<div class="block">display: block
      inline行内元素<span class="inline">display: inline
      inline-block行内块元素<div class="inline-block">display: inline-block
      flex弹性布局<div class="flex">display: flex
      grid网格布局<div class="grid">display: grid
      static/relative/absolute/fixed/sticky定位方式<div class="absolute">position: absolute
      top-0/right-4/bottom-2/left-8定位偏移<div class="absolute top-0 right-4">top: 0; right: 1rem（1rem=4px）
      float-left/float-right浮动<img class="float-left">float: left
      clear-both清除浮动<div class="clear-both">clear: both
      overflow-hidden/overflow-auto/overflow-scroll溢出处理<div class="overflow-auto">overflow: auto

      Vue 示例（弹性布局）：

      <template>
  <div class="flex justify-center items-center h-64 bg-gray-100">
    <div class="w-20 h-20 bg-blue-500"></div>
  </div>
</template>
      • justify-center：主轴居中（flex 布局）
      • items-center：交叉轴居中
      • h-64：高度 16rem（h-{n} 中 n 对应 n*0.25rem，64=16rem）
    • 2.尺寸相关（Sizing）

      控制宽高、最小 / 最大宽高。

      工具类功能示例对应 CSS
      w-0~w-full/w-screen宽度<div class="w-32">width: 8rem（32*0.25rem）
      h-0~h-full/h-screen高度<div class="h-screen">height: 100vh
      min-w-0/min-w-full最小宽度<div class="min-w-full">min-width: 100%
      max-w-xs/max-w-sm/max-w-md.../max-w-full最大宽度<div class="max-w-md">max-width: 28rem（md 断点宽度）
      min-h-0/min-h-screen最小高度<div class="min-h-screen">min-height: 100vh
      max-h-full/max-h-screen最大高度<div class="max-h-96">max-height: 24rem

      Vue 示例（响应式宽度）：

      <template>
  <div class="w-full md:w-1/2 lg:w-1/3 mx-auto bg-gray-100 p-4">
    响应式宽度：移动端100%，平板50%，桌面33%
  </div>
</template>
    • 3.间距相关（Spacing）

      控制内边距（padding）和外边距（margin），命名规则：p{方向}-{大小}（padding）、m{方向}-{大小}（margin）。

      • 方向缩写：t（上）、r（右）、b（下）、l（左）、x（左右）、y（上下）、无方向（全方向）
      • 大小值：0~96（步长 4，对应 0~24rem）、auto（自动间距）、px（1px）
      工具类功能示例对应 CSS
      p-4全方向内边距 1rem<div class="p-4">padding: 1rem
      pt-6上内边距 1.5rem<div class="pt-6">padding-top: 1.5rem
      px-8左右内边距 2rem<div class="px-8">padding-left: 2rem; padding-right:2rem
      py-2上下内边距 0.5rem<div class="py-2">padding-top:0.5rem; padding-bottom:0.5rem
      m-3全方向外边距 0.75rem<div class="m-3">margin: 0.75rem
      mr-5右外边距 1.25rem<div class="mr-5">margin-right: 1.25rem
      mx-auto左右自动外边距（居中）<div class="mx-auto">margin-left: auto; margin-right: auto
      my-4上下外边距 1rem<div class="my-4">margin-top:1rem; margin-bottom:1rem

      Vue 示例（内外边距）：

      <template>
  <div class="m-6 p-4 bg-blue-100">
    外边距 m-6（1.5rem），内边距 p-4（1rem）
    <div class="mt-4 pl-2 bg-white">
      上外边距 mt-4（1rem），左内边距 pl-2（0.5rem）
    </div>
  </div>
</template>
    • 4.颜色相关（Colors）

      Tailwind 内置 19 种颜色（如 red、blue、green、gray 等），每种颜色有 10 个亮度等级（50~950，数值越大颜色越深），命名规则：属性-颜色-等级。

      ( 查询颜色表 )

      属性前缀功能示例对应 CSS
      bg-背景色bg-red-500background-color: #ef4444
      text-文字色text-blue-700color: #1d4ed8
      border-边框色border-green-300border-color: #86efac
      hover:bg-hover 背景色hover:bg-yellow-400hover { background-color: #facc15; }
      focus:ring-focus 状态环形颜色focus:ring-purple-500focus { --tw-ring-color: #a855f7; }

      Vue 示例（颜色组合）：

      <template>
  <div class="bg-gray-100 text-gray-800 p-4 border-2 border-gray-300 hover:bg-gray-200 transition-colors">
    背景色 bg-gray-100，文字色 text-gray-800，边框色 border-gray-300，hover 变色
  </div>
</template>
    • 5.文字相关（Typography）

      控制字体、字号、字重、对齐方式、行高、文本装饰等。

      工具类功能示例对应 CSS
      font-sans/font-serif/font-mono字体族<p class="font-mono">font-family: ui-monospace, SFMono-Regular, ...
      text-xs~text-9xl字号（响应式）<h1 class="text-4xl">font-size: 2.25rem
      font-light/font-normal/font-bold/font-black字重<span class="font-bold">font-weight: 700
      text-left/text-center/text-right/text-justify文本对齐<p class="text-center">text-align: center
      leading-none/leading-tight/leading-normal/leading-loose行高<p class="leading-loose">line-height: 2
      text-underline/text-line-through文本装饰<span class="text-underline">text-decoration: underline
      text-uppercase/text-lowercase/text-capitalize文本转换<p class="text-uppercase">text-transform: uppercase
      tracking-tight/tracking-wide字间距<p class="tracking-wide">letter-spacing: 0.05em
      truncate文本溢出省略（单行）<p class="truncate">white-space: nowrap; overflow: hidden; text-overflow: ellipsis
      line-clamp-3文本溢出省略（多行）<p class="line-clamp-3">限制 3 行，超出省略（需配合 overflow-hidden）

      Vue 示例（文字样式）：

      <template>
  <div class="max-w-md mx-auto">
    <h2 class="text-2xl font-bold text-center text-blue-600 mb-2">标题文字</h2>
    <p class="text-gray-700 leading-relaxed line-clamp-3">
      这是一段测试文本，用于演示 Tailwind 的文字样式。line-clamp-3 会限制文本最多显示 3 行，超出部分用省略号表示。
    </p>
    <p class="mt-4 text-sm text-gray-500 italic">
      小字号 + 斜体 + 灰色文字
    </p>
  </div>
</template>

    • 6.边框相关（Borders）

      控制边框宽度、颜色、圆角、样式。

      工具类功能示例对应 CSS
      border默认边框（1px 黑色）<div class="border">border: 1px solid #e5e7eb
      border-2/border-4边框宽度<div class="border-2">border-width: 2px
      border-t-2上边框宽度<div class="border-t-2">border-top-width: 2px
      border-red-500边框颜色<div class="border-red-500">border-color: #ef4444
      rounded-none/rounded-sm/rounded/rounded-md/rounded-lg/rounded-full圆角<div class="rounded-full">border-radius: 9999px（圆形）
      rounded-t-lg上圆角<div class="rounded-t-lg">border-top-left-radius: 0.5rem; border-top-right-radius:0.5rem
      border-dashed/border-dotted边框样式<div class="border-dashed">border-style: dashed
      ring-2/ring-blue-500环形边框（聚焦 / 高亮）<input class="ring-2 ring-blue-500">--tw-ring-width: 2px; --tw-ring-color: #3b82f6

      Vue 示例（边框与圆角）：

      <template>
  <div class="w-32 h-32 border-4 border-green-500 rounded-lg bg-green-50 flex items-center justify-center">
    边框 4px + 圆角 lg + 背景色
  </div>
</template>
    • 7.背景相关（Backgrounds）

      控制背景色、背景图片、背景大小、背景位置等。

      工具类功能示例对应 CSS
      bg-red-500背景色<div class="bg-red-500">background-color: #ef4444
      bg-opacity-50背景透明度<div class="bg-blue-500 bg-opacity-50">background-color: rgba(59, 130, 246, 0.5)
      bg-gradient-to-r线性渐变（右向）<div class="bg-gradient-to-r from-blue-500 to-purple-500">background-image: linear-gradient(to right, #3b82f6, #a855f7)
      bg-cover背景图片覆盖<div class="bg-cover bg-center" style="background-image: url('xxx.jpg')">background-size: cover; background-position: center
      bg-repeat/bg-no-repeat背景重复<div class="bg-no-repeat">background-repeat: no-repeat
      bg-fixed背景固定<div class="bg-fixed">

      Vue 示例（渐变背景）：

      <template>
  <div class="h-40 bg-gradient-to-br from-pink-400 via-purple-500 to-indigo-600 flex items-center justify-center text-white font-bold text-xl">
    从粉到紫到靛蓝的渐变背景
  </div>
</template>

    • 8.阴影相关（Shadows）

      控制阴影大小、颜色、透明度，默认支持 shadow-sm~shadow-xl，还有 shadow-inner（内阴影）、shadow-none（无阴影）。

      工具类功能示例对应 CSS（简化）
      shadow默认阴影<div class="shadow">box-shadow: 0 1px 3px 0 rgba(0,0,0,0.1)
      shadow-md中等阴影<div class="shadow-md">box-shadow: 0 4px 6px -1px rgba(0,0,0,0.1)

      shadow-inner内阴影<div class="shadow-inner">box-shadow: inset 0 2px 4px 0 rgba(0,0,0,0.05)
      shadow-blue-500蓝色阴影<div class="shadow-lg shadow-blue-500">box-shadow: 0 10px 15px -3px rgba(59,130,246,0.4)

      Vue 示例（阴影效果）：

      <template>
  <div class="w-40 h-40 bg-white rounded-lg shadow-xl hover:shadow-2xl transition-shadow mx-auto mt-10">
    hover 时阴影放大（过渡动画）
  </div>
</template>
    • 9.过度与动画（Transitions & Animations）

      Tailwind 内置过渡和动画工具类，简化交互效果开发。

      过度（Transitions）

      工具类功能示例对应 CSS
      transition默认过渡（all 0.3s）<div class="transition">transition: all 0.3s ease-in-out
      transition-colors仅颜色过渡<div class="transition-colors">transition-property: color, background-color, border-color...
      transition-all所有属性过渡<div class="transition-all">transition-property: all
      duration-300/duration-500过渡时长<div class="duration-500">transition-duration: 500ms
      ease-in/ease-out/ease-in-out过渡曲线<div class="ease-in-out">transition-timing-function: ease-in-out
      delay-200过渡延迟<div class="delay-200">transition-delay: 200ms

      动画（Animations）

      Tailwind 内置 spin（旋转）、pulse（呼吸）、bounce（弹跳）三种动画。

      工具类功能示例对应 CSS（简化）
      animate-spin旋转动画<div class="animate-spin">animation: spin 1s linear infinite
      animate-pulse呼吸动画（透明度变化）<div class="animate-pulse">animation: pulse 2s cubic-bezier(0.4,0,0.6,1) infinite
      animate-bounce弹跳动画<div class="animate-bounce">animation: bounce 1s cubic-bezier(0.175, 0.885, 0.32, 1.275) infinite

      Vue 示例（过渡与动画）：

      <template>
  <div class="flex gap-10 justify-center mt-10">
    <!-- 过渡效果 -->
    <div class="w-20 h-20 bg-blue-500 hover:bg-red-500 transition-colors duration-300 rounded"></div>
    <!-- 旋转动画 -->
    <div class="w-20 h-20 border-4 border-t-blue-500 rounded-full animate-spin"></div>
    <!-- 呼吸动画 -->
    <div class="w-20 h-20 bg-green-500 rounded animate-pulse"></div>
  </div>
</template>

  • 3.响应式设计（Responsive Design）

    Tailwind 内置 5 个响应式断点（移动优先），通过断点前缀控制不同屏幕尺寸的样式，前缀需放在工具类最前面。

    1.默认断点

    断点前缀屏幕尺寸（最小宽度）适用设备
    无前缀< 640px移动端（手机）
    sm:≥ 640px小屏幕手机
    md:≥ 768px平板（竖屏）
    lg:≥ 1024px平板（横屏）/ 桌面
    xl:≥ 1280px大屏幕桌面
    2xl:≥ 1536px超大屏幕桌面

    2.使用规则

    • 移动优先：无前缀的工具类默认作用于移动端，断点前缀仅作用于大于等于该尺寸的设备
    • 顺序无关：但建议按「从小到大」的顺序编写（便于阅读）
    • 可组合：多个断点前缀可叠加使用
    <template>
  <div class="grid grid-cols-1 sm:grid-cols-2 md:grid-cols-3 lg:grid-cols-4 gap-4 p-4">
    <!-- 移动端1列，小屏2列，平板3列，桌面4列 -->
    <div class="bg-gray-100 p-4 rounded-md shadow">卡片1</div>
    <div class="bg-gray-100 p-4 rounded-md shadow">卡片2</div>
    <div class="bg-gray-100 p-4 rounded-md shadow">卡片3</div>
    <div class="bg-gray-100 p-4 rounded-md shadow">卡片4</div>
  </div>
</template>
  • 4.状态变体（State Variants）

    Tailwind 支持为元素的不同状态（如 hover、focus、active 等）添加样式，通过状态前缀控制，前缀需放在工具类最前面（响应式前缀之后）。

    1.常用状态前缀

    状态前缀适用元素功能示例
    hover:所有元素鼠标悬停状态hover:bg-blue-500
    focus:表单元素、链接元素聚焦状态focus:ring-2
    active:所有元素元素激活状态（点击时）active:bg-blue-700
    disabled:表单元素、按钮元素禁用状态disabled:opacity-50
    visited:链接链接已访问状态visited:text-purple-600
    checked:复选框、单选框元素选中状态checked:bg-green-500
    focus-within:容器元素容器内有聚焦元素时focus-within:bg-gray-100
    dark:所有元素深色模式下dark:bg-gray-800（需启用深色模式）

    2.使用规则

    • 状态前缀可与响应式前缀组合（响应式前缀在前，状态前缀在后）：md:hover:bg-red-500
    • 多个状态前缀可叠加：hover:focus:ring-4
    • 部分状态需配合特定元素（如 checked: 仅用于表单控件）
    <template>
  <button class="px-6 py-3 bg-blue-500 text-white rounded-md hover:bg-blue-600 active:bg-blue-700 focus:outline-none focus:ring-2 focus:ring-blue-300 disabled:opacity-50 transition-colors">
    按钮状态示例
  </button>
  <input type="checkbox" class="w-5 h-5 rounded checked:bg-green-500">
</template>

高级用法：自定义配置与扩展

Tailwind 支持通过 tailwind.config.js 扩展默认主题、添加自定义工具类，满足项目个性化需求。

  • 扩展默认主体（theme.extend）

    在 theme.extend 中添加的配置会与默认主题合并（而非覆盖），常用扩展项：colors、fonts、spacing、breakpoints、borderRadius 等。

    示例：扩展颜色，字体，间距

    // tailwind.config.js
export default {
  content: ["./index.html", "./src/**/*.{vue,js,ts}"],
  theme: {
    extend: {
      // 扩展颜色（添加自定义品牌色）
      colors: {
        primary: "#165DFF", // 品牌主色
        secondary: "#FF7D00", // 品牌辅助色
        neutral: {
          100: "#F5F7FA",
          200: "#E4E7ED",
          900: "#1D2129",
        },
      },
      // 扩展字体（添加自定义字体）
      fontFamily: {
        sans: ["Inter", "system-ui", "sans-serif"],
        display: ["Montserrat", "sans-serif"], // 自定义标题字体
      },
      // 扩展间距（添加自定义尺寸）
      spacing: {
        "128": "32rem", // w-128 = 32rem
        "144": "36rem",
      },
      // 扩展响应式断点（添加 3xl 断点）
      screens: {
        "3xl": "1920px",
      },
      // 扩展圆角
      borderRadius: {
        "xl": "1rem",
        "2xl": "1.5rem",
      },
    },
  },
  plugins: [],
};

    使用自定义主题：

    <template>
  <div class="bg-primary text-white p-8 rounded-2xl font-display">
    自定义品牌色 bg-primary + 自定义字体 font-display + 自定义圆角 rounded-2xl
  </div>
</template>

  • 自定义工具类（@layer utilities）

    通过 @layer utilities 在 CSS 文件中添加自定义原子化工具类，Tailwind 会自动处理前缀（如响应式、状态变体）。

    示例：添加自定义工具类

    /* src/assets/main.css */
@tailwind base;
@tailwind components;
@tailwind utilities;

/* 自定义工具类（放在 @tailwind utilities 之后） */
@layer utilities {
  /* 文字渐变 */
  .text-gradient {
    background-clip: text;
    -webkit-background-clip: text;
    -webkit-text-fill-color: transparent;
  }
  /* 内容垂直居中（需父容器为 flex） */
  .content-center {
    align-content: center;
  }
  /* 自定义阴影 */
  .shadow-custom {
    box-shadow: 0 4px 20px rgba(0, 0, 0, 0.15);
  }
}

    使用自定义工具类：

    <template>
  <div class="flex h-64 bg-gray-100 content-center">
    <h2 class="text-3xl font-bold text-gradient bg-gradient-to-r from-primary to-secondary shadow-custom">
      文字渐变 + 自定义阴影
    </h2>
  </div>
</template>
  • 自定义组件类（@layer components）

    通过 @layer components 提取可复用的组件样式（如按钮、卡片），避免重复编写工具类组合。

    示例：自定义组件类

    /* src/assets/main.css */
@layer components {
  /* 自定义按钮组件 */
  .btn {
    @apply px-6 py-3 rounded-md font-bold transition-colors focus:outline-none focus:ring-2 focus:ring-offset-2;
  }
  .btn-primary {
    @apply bg-primary text-white hover:bg-primary/90 focus:ring-primary/50;
  }
  .btn-secondary {
    @apply bg-secondary text-white hover:bg-secondary/90 focus:ring-secondary/50;
  }
  /* 自定义卡片组件 */
  .card {
    @apply bg-white rounded-xl shadow-md p-6 hover:shadow-lg transition-shadow;
  }
}

    使用自定义组件类：

    <template>
  <div class="card max-w-md mx-auto">
    <h3 class="text-xl font-bold mb-4">自定义卡片</h3>
    <p class="text-gray-600 mb-6">使用 @layer components 提取的卡片样式</p>
    <div class="flex gap-4">
      <button class="btn btn-primary">主按钮</button>
      <button class="btn btn-secondary">辅助按钮</button>
    </div>
  </div>
</template>

    注：@apply 用于在自定义样式中引入 Tailwind 工具类，简化代码。

  • 启用神色模式（Dark Mode）

    Tailwind 支持深色模式，需在 tailwind.config.js 中配置，然后通过 dark: 前缀控制深色样式。

    步骤 1：配置深色模式

    // tailwind.config.js
export default {
  darkMode: "class", // 基于 class 切换（推荐，灵活控制）
  // darkMode: 'media' // 基于系统主题自动切换
  content: ["./index.html", "./src/**/*.{vue,js,ts}"],
  theme: { extend: {} },
  plugins: [],
};

    步骤 2：在 Vue 中控制深色模式

    通过给根元素（如 html 或 body）添加 dark 类切换深色模式：

    <template>
  <div class="min-h-screen bg-gray-50 dark:bg-gray-900 text-gray-900 dark:text-gray-100 transition-colors">
    <h1 class="text-2xl font-bold p-6">深色模式示例</h1>
    <button @click="toggleDarkMode" class="px-4 py-2 bg-primary text-white rounded-md">
      {{ isDarkMode ? "切换浅色模式" : "切换深色模式" }}
    </button>
  </div>
</template>

<script setup>
import { ref, watchEffect } from "vue";

// 初始化深色模式（从本地存储读取）
const isDarkMode = ref(localStorage.getItem("darkMode") === "true");

// 切换深色模式
const toggleDarkMode = () => {
  isDarkMode.value = !isDarkMode.value;
  localStorage.setItem("darkMode", isDarkMode.value);
};

// 监听深色模式变化，给 html 标签添加/移除 dark 类
watchEffect(() => {
  if (isDarkMode.value) {
    document.documentElement.classList.add("dark");
  } else {
    document.documentElement.classList.remove("dark");
  }
});
</script>
  • 使用插件（Plugins）

    Tailwind 生态有丰富的插件，可扩展更多功能（如表单样式、图标、动画等）。

    示例：安装并使用 @tailwindcss/forms（优化表单样式）

    安装插件：

    npm install -D @tailwindcss/forms

    配置插件：

    // tailwind.config.js
export default {
  content: ["./index.html", "./src/**/*.{vue,js,ts}"],
  theme: { extend: {} },
  plugins: [require("@tailwindcss/forms")], // 引入插件
};

    使用效果：表单元素（输入框、下拉框、复选框等）会自动应用优化后的样式，无需手动编写。

  • 动态绑定工具类（Vue 模板语法）

    在 Vue 中通过 :class 动态绑定工具类，支持对象、数组、计算属性语法。

    示例 1：对象语法（条件绑定）

    <template>
  <div :class="{
    'bg-green-500': isSuccess,
    'bg-red-500': !isSuccess,
    'text-white': true,
    'p-4 rounded-md': true
  }">
    {{ isSuccess ? "操作成功" : "操作失败" }}
  </div>
</template>

<script setup>
const isSuccess = ref(true);
</script>

    示例 2：数组语法（组合绑定）

    <template>
  <div :class="[
    'p-4 rounded-md',
    isSuccess ? 'bg-green-500 text-white' : 'bg-red-500 text-white'
  ]">
    动态样式组合
  </div>
</template>

    示例 3：计算属性（复杂逻辑）

    <template>
  <div :class="cardClass">复杂动态样式</div>
</template>

<script setup>
const type = ref("primary");
const size = ref("large");

const cardClass = computed(() => {
  return [
    "rounded-md shadow-md p-4",
    type.value === "primary" ? "bg-primary text-white" : "bg-gray-100 text-gray-800",
    size.value === "large" ? "text-xl h-24" : "text-base h-16",
  ];
});
</script>

  • 提取可复用组件（Vue SFC）

    结合 Tailwind 工具类和 Vue 组件化，提取通用组件（如按钮、卡片、表单），提升复用率。

    示例：通用按钮组件（src/components/Button.vue）

    <template>
  <button
    :class="[
      'px-4 py-2 rounded-md font-bold transition-colors focus:outline-none focus:ring-2 focus:ring-offset-2',
      variant === 'primary' ? 'bg-primary text-white hover:bg-primary/90 focus:ring-primary/50' : '',
      variant === 'secondary' ? 'bg-secondary text-white hover:bg-secondary/90 focus:ring-secondary/50' : '',
      variant === 'outline' ? 'border border-gray-300 text-gray-700 hover:bg-gray-100' : '',
      disabled ? 'opacity-50 cursor-not-allowed' : '',
      size === 'large' ? 'px-6 py-3 text-lg' : '',
      size === 'small' ? 'px-3 py-1 text-sm' : '',
    ]"
    :disabled="disabled"
    @click="$emit('click')"
  >
    <slot></slot>
  </button>
</template>

<script setup>
const props = defineProps({
  variant: {
    type: String,
    default: "primary",
    validator: (v) => ["primary", "secondary", "outline"].includes(v),
  },
  size: {
    type: String,
    default: "default",
    validator: (v) => ["large", "default", "small"].includes(v),
  },
  disabled: {
    type: Boolean,
    default: false,
  },
});

defineEmits(["click"]);
</script>

    使用组件：

    <template>
  <Button variant="primary" size="large" @click="handleClick">
    主按钮（大尺寸）
  </Button>
  <Button variant="outline" size="small" class="ml-4">
    轮廓按钮（小尺寸）
  </Button>
</template>

<script setup>
const handleClick = () => alert("按钮点击");
</script>
  • 优化打包体积

    Tailwind 3 默认启用 JIT（即时编译）模式，仅打包项目中实际使用的工具类，体积优化极佳。若需进一步优化：

    • 确保 tailwind.config.js 的 content 配置正确（仅扫描必要文件）
    • 避免过度使用 @apply 提取大量未复用的样式
    • 使用 purgecss（Tailwind 3 已内置，无需额外配置）
