方块属性

  1. strength(float destroyHardness, float explosionResistance)
    • 第一参数：挖掘硬度，数值越大挖掘越慢
    • 第二参数：爆炸抗性，数值越高越难被炸碎
    // 硬度1.5，防爆3
.of().strength(1.5F, 3.0F)
  1. lightLevel(BlockState state -> int)

    亮度范围 0~15，15 为最亮

    // 固定亮度7
.of().lightLevel(s -> 7)
  1. dropsLike(Block block)

    沿用指定方块的掉落逻辑

    // 掉落规则同原石
.of().dropsLike(Blocks.COBBLESTONE)
  1. requiresCorrectToolForDrops()

    必须对应工具才能掉落物品，徒手挖不掉

    // 必须镐子挖掘才掉落
.of().requiresCorrectToolForDrops()
  1. soundType(SoundType type)

    常用原版声音枚举

    // 石头音效
.of().soundType(SoundType.STONE)
// 木质音效
.of().soundType(SoundType.WOOD)
// 沙子、玻璃、金属、泥土同理
  1. friction(float value)

    默认 0.6F，数值越小越滑

    // 类似冰面打滑
.of().friction(0.98F)
  1. jumpFactor(float value)

    调整玩家跳跃高度

    // 跳得更高
.of().jumpFactor(1.2F)
  1. speedFactor(float value)

    调整方块上移动速度

    // 行走变慢
.of().speedFactor(0.7F)
  1. noPush()

    活塞无法推动该方块

    .of().noPush()
  1. opaque() / transparent()
    • opaque()：不透明，遮挡光线
    • transparent()：透明方块，如水、玻璃
    .of().transparent()
  1. noCollission()

    无碰撞，可直接穿过（花朵、栅栏装饰）

    .of().noCollission()
  1. indestructible()

    无敌方块，不能破坏

    .of().indestructible()
  1. offsetType(BlockBehaviour.OffsetType type)

    NONE无偏移，XZ地面随机偏移（花草通用）

    .of().offsetType(BlockBehaviour.OffsetType.XZ)
  1. slippery()

    冰类打滑专属属性

    .of().slippery()
  1. air()

    当作空气方块，无实体体积

    .of().air()
  1. fireProperties(BlockFireProperties props)

    设置易燃、火势蔓延速度

    .of().fireProperties(BlockFireProperties.NONE)
// 发光、高防爆、需镐子挖掘的自定义方块
BlockBehaviour.Properties.of()
        .strength(2F, 5F)
        .lightLevel(s -> 10)
        .requiresCorrectToolForDrops()
        .soundType(SoundType.METAL);
