    • 案例
      <!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>基础功能表单</title>
</head>
<body>
    <h2>用户信息表单</h2>
    <!-- 表单核心：action提交地址，method提交方式 -->
    <form action="submit.php" method="post">
        <!-- 1. 文本输入框（单行文本） -->
        <p>
            用户名：
            <input type="text" name="username" required placeholder="请输入用户名">
            <!-- required：必填项；placeholder：提示文本 -->
        </p>

        <!-- 2. 密码输入框 -->
        <p>
            密码：
            <input type="password" name="password" required minlength="6">
            <!-- type="password"：输入内容隐藏；minlength：最小长度限制 -->
        </p>

        <!-- 3. 单选按钮组（只能选一个） -->
        <p>
            性别：
            <input type="radio" name="gender" value="male" checked> 男
            <input type="radio" name="gender" value="female"> 女
            <!-- name相同为一组；checked：默认选中 -->
            <!-- value="male" 是提交给后台的值，如果选择男，后台会受到 male -->
        </p>

        <!-- 4. 复选框（可多选） -->
        <p>
            爱好：
            <input type="checkbox" name="hobby[]" value="reading"> 阅读
            <input type="checkbox" name="hobby[]" value="sports"> 运动
            <input type="checkbox" name="hobby[]" value="music"> 音乐
            <!-- name带[]：后端接收为数组 -->
        </p>

        <!-- 5. 下拉选择框 -->
        <p>
            学历：
            <select name="education">
                <option value="">请选择</option>
                <option value="highschool">高中</option>
                <option value="college">大专</option>
                <option value="bachelor" selected>本科</option>
                <option value="master">硕士</option>
                <!-- selected：默认选中项 -->
            </select>
        </p>

        <!-- 6. 多行文本框 -->
        <p>
            个人简介：
            <br>
            <textarea name="intro" rows="3" cols="30" placeholder="请输入个人简介"></textarea>
            <!-- rows：行数；cols：列数 -->
        </p>

        <!-- 7. 数字输入框 -->
        <p>
            年龄：
            <input type="number" name="age" min="0" max="120" value="18">
            <!-- type="number"：只允许输入数字；min/max：范围限制；value：默认值 -->
        </p>

        <!-- 8. 隐藏域（用户不可见，用于传递数据） -->
        <input type="hidden" name="form_id" value="user_info_123">

        <!-- 9. 按钮 -->
        <p>
            <input type="submit" value="提交">
            <input type="reset" value="重置">
            <button type="button" onclick="alert('这是普通按钮')">帮助</button>
        </p>
    </form>
</body>
</html>
    • <form>

      用于用于收集用户输入并提交数

      属性名作用描述常用取值示例代码
      action指定表单数据提交的目标 URL（后端处理程序）相对路径（如 submit.php）、绝对路径（如 https://api.example.com）、空值（当前页面）<form action="/user/login">
      method定义数据提交的 HTTP 方法get（默认，数据在 URL 中，适合查询）、post（数据在请求体，适合敏感数据）<form method="post">
      enctype设置表单数据的编码方式（仅 method="post" 有效）application/x-www-form-urlencoded（默认，普通数据）、multipart/form-data（文件上传）、text/plain（纯文本）<form enctype="multipart/form-data">
      target指定响应结果显示的窗口 / 框架_self（默认，当前窗口）、_blank（新窗口）、_parent（父框架）、_top（顶层窗口）<form target="_blank">
      autocomplete控制浏览器是否自动填充表单数据on（默认，启用）、off（禁用，适合敏感字段）<form autocomplete="off">
      novalidate禁用浏览器的 HTML5 自动验证（如 required、pattern 验证）布尔属性（存在即生效，无需赋值）<form novalidate>
      rel定义当前页面与 action 目标的关系（增强安全性）noopener（防止新页面访问原页面）、noreferrer（隐藏来源）<form rel="noopener noreferrer">
      name为表单定义名称，用于 JavaScript 访问（document.forms.name）自定义名称（如 loginForm）<form name="loginForm">
      id表单唯一标识符，用于 CSS 或 JavaScript 定位（getElementById）自定义唯一 ID（如 userForm）<form id="userForm">
    • <input>
      <input type="控件类型" name="键名" 属性="值"> 
必须属性：
// name：用于后端接收数据时得到“键名”，缺少无法提交数据
// type：决定空间类型，默认为text
      type 值控件名称功能说明示例代码
      text单行文本框接收普通文本输入（默认类型），可通过 maxlength 限制长度。<input type="text" name="username" placeholder="请输入用户名" maxlength="20">
      password密码框输入内容以圆点 / 星号隐藏，保护敏感信息。<input type="password" name="pwd" required minlength="6">
      radio单选按钮多个同 name 的单选按钮组成一组，只能选择其中一个，用 checked 设默认值。<input type="radio" name="gender" value="male" checked> 男
      checkbox复选框可多选，同 name 建议加 []（如 hobby[]），后端接收为数组。<input type="checkbox" name="hobby[]" value="reading"> 阅读
      number数字输入框只允许输入数字（包括正负号、小数点），通过 min/max 限制范围。<input type="number" name="age" min="0" max="120" value="18">
      email邮箱输入框自带邮箱格式验证（如必须包含 @），提交时不符合格式会提示错误。<input type="email" name="email" required>
      tel电话输入框移动端会唤起数字键盘，PC 端无特殊限制，需配合 pattern 自定义验证。<input type="tel" name="phone" pattern="\d{11}" placeholder="请输入11位手机号">
      url网址输入框验证输入是否为合法网址（需包含 http:// 或 https://）。<input type="url" name="website" placeholder="如：https://example.com">
      date日期选择器弹出日历选择年月日（格式 YYYY-MM-DD），可通过 min/max 限制日期范围。<input type="date" name="birthday" min="1900-01-01" max="2023-12-31">
      time时间选择器选择小时和分钟（格式 HH:MM）。<input type="time" name="meet_time" min="09:00" max="18:00">
      file文件上传框用于选择本地文件，multiple 允许多选，accept 限制文件类型。<input type="file" name="avatar" accept="image/*" multiple>
      hidden隐藏域用户不可见，用于传递额外数据（如表单 ID、用户 ID 等）。<input type="hidden" name="form_id" value="user_123">
      submit提交按钮点击后提交表单数据到 form 标签的 action 地址。<input type="submit" value="提交表单">
      reset重置按钮点击后将表单所有控件恢复为默认值。<input type="reset" value="重新填写">
      button普通按钮无默认功能，需配合 onclick 绑定事件（如弹窗）。<input type="button" value="点击提示" onclick="alert('Hello')">
      input属性名作用范围功能说明
      name所有类型后端接收数据的 “键名”（如 name="username"，后端获取 username 的值）。
      value所有类型控件的默认值（如文本框的初始内容、单选 / 复选框的提交值）。
      placeholder文本类控件输入框为空时显示的提示文本（如 “请输入用户名”），不影响实际提交值。
      required大部分控件标记为必填项，提交时若为空，浏览器会提示 “请填写此字段”。
      disabled所有类型禁用控件（不可编辑、不可点击），禁用状态的控件数据不会被提交。
      readonly文本类控件控件只读（不可编辑，但可选中复制），数据会正常提交。
      checked单选 / 复选框设置默认选中状态（单选按钮组中只能有一个 checked）。
      min/max数字 / 日期 / 时间限制输入范围（如 min="0" 表示数字不能小于 0，max="2023-12-31" 限制日期）。
      maxlength文本类控件限制最大输入长度（如 maxlength="11" 限制手机号为 11 位）。
      multiple文件上传 / 邮箱允许多选（文件上传时可选择多个文件，邮箱输入时可填多个邮箱用逗号分隔）。
      accept文件上传限制可上传的文件类型（如 accept="image/*" 只允许图片，accept=".pdf" 只允许 PDF）。
    • <button>

      <button> 标签用于创建可点击的按钮，功能灵活，可用于提交表单、重置表单或触发自定义 JavaScript 事件。

      与 <input type="button"> 相比，<button> 是双标签，内部可包含文本、图片等内容，样式控制更灵活。

      <button type="按钮类型" 属性>按钮内容（文本/图片等）</button>
// type：决定按钮行为，默认值为submit
      type 值按钮类型行为说明适用场景
      submit提交按钮点击后提交表单数据到 <form> 标签的 action 地址（默认类型）。表单提交（如 “登录”“注册”）
      reset重置按钮点击后将表单中所有控件恢复为默认值（需放在 <form> 内才有效）。表单重置（如 “重新填写”）
      button普通按钮无默认行为，需配合 onclick 等事件绑定自定义逻辑（如弹窗、切换内容）。触发 JavaScript 操作
      <button>属性名作用描述示例代码
      type定义按钮类型（submit/reset/button），默认 submit。<button type="button">点击</button>
      name为按钮定义名称，用于后端区分多个按钮（提交表单时会将 name=value 传递给后端）。<button type="submit" name="action" value="save">保存</button>
      value按钮的提交值（仅在 type="submit" 或 type="reset" 时有效，后端通过 name 获取）。<button type="submit" name="op" value="delete">删除</button>
      disabled禁用按钮（不可点击，样式变灰），禁用状态的按钮不会触发事件或提交数据。<button type="submit" disabled>提交（禁用）</button>
      autofocus页面加载时自动聚焦到该按钮（一个页面只能有一个元素用此属性）。<button type="button" autofocus>自动聚焦</button>
      form指定按钮关联的表单 ID（即使按钮在 <form> 外部，也能控制该表单）。<button type="submit" form="userForm">提交表单</button>（表单需设置 id="userForm"）
      formaction覆盖表单的 action 属性，指定当前按钮的提交地址（仅 type="submit" 有效）。<button type="submit" formaction="save.php">保存</button>
      formmethod覆盖表单的 method 属性，指定当前按钮的提交方式（get/post，仅 type="submit" 有效）。<button type="submit" formmethod="get">查询</button>
      <form id="userForm" action="submit.php" method="post">
  <input type="text" name="username" placeholder="用户名">
  
  <!-- 1. 提交按钮（默认 type="submit"） -->
  <button>默认提交（type="submit"）</button>
  
  <!-- 2. 明确指定提交类型，带自定义提交值 -->
  <button type="submit" name="action" value="save">保存（提交）</button>
  
  <!-- 3. 重置按钮 -->
  <button type="reset">重置表单</button>
  
  <!-- 4. 普通按钮（触发JavaScript） -->
  <button type="button" onclick="alert('普通按钮被点击')">点击弹窗</button>
</form>

<!-- 5. 表单外部的按钮，通过 form 属性关联表单 -->
<button type="submit" form="userForm" formaction="update.php" formmethod="post">
  外部提交按钮（修改数据）
</button>

<!-- 6. 禁用状态的按钮 -->
<button type="submit" disabled>禁用的提交按钮（不可点击）</button>
    • <select>&&<option>

      创建下拉菜单。

      <select>属性名作用描述示例代码
      name定义下拉框的 “键名”，后端通过该名称接收选中的值（必填）。<select name="city">
      multiple布尔属性，允许同时选择多个选项（按住 Ctrl 键点击多选），默认只能选一个。<select name="hobby" multiple>
      size规定可见选项的数量（默认值为 1，即折叠为下拉框；值 >1 则显示为列表）。<select name="subject" size="3">
      disabled禁用下拉框（不可选择，数据不会提交）。<select name="city" disabled>
      required标记为必填项，提交表单时若未选择任何选项，浏览器会提示错误。<select name="city" required>
      autofocus页面加载时自动聚焦到该下拉框（一个页面只能有一个元素用此属性）。<select name="city" autofocus>
      <option>属性名作用描述示例代码
      value选项被选中时，提交给后端的实际值（若省略，默认提交 “选项文本”）。<option value="beijing">北京</option>
      selected布尔属性，设置该选项为默认选中项（一个 <select> 中可多个，但通常只设一个）。<option value="shanghai" selected>上海</option>
      disabled禁用该选项（不可选中）。<option value="guangzhou" disabled>广州</option>
      label为选项定义更简短的标签（优先于选项文本显示，较少用）。<option value="sz" label="深">深圳</option>

    • <textarea>

      创建多行文本输入框的元素。

      与单行文本输入框 <input type="text"> 相比，它支持换行和多行显示，且输入长度通常不受限（可通过属性控制）。

      <textarea name="键名" 属性>默认文本内容</textarea>
      属性名作用描述示例代码
      name定义文本框的 “键名”，后端通过该名称接收输入的文本（必填）。<textarea name="intro"></textarea>
      rows规定可见的行数（高度），默认值为 2（实际高度受字体大小等影响）。<textarea name="comment" rows="4"></textarea>
      cols规定可见的列数（宽度），默认值为 20（实际宽度受字体大小等影响）。<textarea name="comment" cols="30"></textarea>
      placeholder输入框为空时显示的提示文本（如 “请输入评论”），不影响实际提交值。<textarea name="feedback" placeholder="请描述您遇到的问题"></textarea>
      required标记为必填项，提交表单时若未输入内容，浏览器会提示错误。<textarea name="agreement" required></textarea>
      disabled禁用文本框（不可编辑、不可点击），禁用状态的数据不会被提交。<textarea name="note" disabled></textarea>
      readonly文本框只读（不可编辑，但可选中复制），数据会正常提交。<textarea name="log" readonly></textarea>
      maxlength限制最大输入字符数（包括换行符），超过则无法输入。<textarea name="reply" maxlength="500"></textarea>
      wrap控制提交时文本的换行方式：- soft（默认）：提交时忽略换行，仅保留内容。- hard：提交时保留换行（需配合 cols 使用）。<textarea name="article" wrap="hard" cols="50"></textarea>
      autofocus页面加载时自动聚焦到该文本框（一个页面只能有一个元素用此属性）。<textarea name="comment" autofocus></textarea>
      <form action="submit.php" method="post">

    • <fieldset>

      <fieldset> 是 HTML 中用于对表单元素进行分组和关联的标签.

      常与 <legend> 配合使用，用于在表单中创建一个具有边框的区域，将相关的表单控件（如输入框、复选框等）组合在一起，提升表单的结构性和可读性。

      <fieldset>
  <legend>分组标题</legend>
  <!-- 相关的表单控件 -->
  <label>用户名: <input type="text"></label><br>
  <label>密码: <input type="password"></label>
</fieldset>

      常见属性：

      disabled：禁用整个字段集内的所有表单控件（不可交互，样式会变灰）

      <fieldset disabled>
  <legend>禁用的分组</legend>
  <input type="text" value="不可编辑">
</fieldset>

      form：指定字段集所属的表单（值为目标表单的id），用于关联不在<form>标签内的字段集。

      name：为字段集定义名称，可用于脚本中标识该分组。

      完整实例：

      <form action="/submit" method="post">
  <!-- 个人信息分组 -->
  <fieldset>
    <legend>个人信息</legend>
    <label>姓名: <input type="text" name="name"></label><br>
    <label>年龄: <input type="number" name="age"></label>
  </fieldset>

  <!-- 联系方式分组 -->
  <fieldset>
    <legend>联系方式</legend>
    <label>电话: <input type="tel" name="phone"></label><br>
    <label>邮箱: <input type="email" name="email"></label>
  </fieldset>

  <button type="submit">提交</button>
</form>

// auto：浏览器决定（默认）

// anonymous：跨域请i去不发送凭据（cookie）
// use-credentials：跨域请求发送凭据

  • <table>

    <table> 标签用于创建表格，用于展示结构化数据，配合多个标签（如 <tr>、<td>、<th> 等）可构建复杂的表格结构。

    <table>：表格容器，所有表格内容都需放在此标签内。
<thead>：表头区域，用于定义表格的列标题，视觉上通常会加粗显示。
<tbody>：表格主体，包含实际的数据行，是表格的核心内容区域。
<tfoot>：表尾区域，用于显示汇总信息（如总计、平均值），浏览器会自动将其放在表格底部。
<tr>：表格行（Table Row），每个 <tr> 代表一行，包含多个单元格。
<th>：表头单元格（Table Header），用于列标题，默认加粗居中。
<td>：数据单元格（Table Data），用于存放具体数据。
    <table>
  <thead> <!-- 表头区域（可选，通常包含列标题） -->
    <tr> <!-- 表头行 -->
      <th>列标题1</th>
      <th>列标题2</th>
    </tr>
  </thead>
  <tbody> <!-- 表格主体（必选，包含数据行） -->
    <tr> <!-- 数据行 -->
      <td>单元格内容1</td>
      <td>单元格内容2</td>
    </tr>
    <tr>
      <td>单元格内容3</td>
      <td>单元格内容4</td>
    </tr>
  </tbody>
  <tfoot> <!-- 表尾区域（可选，通常包含汇总信息） -->
    <tr>
      <td>总计</td>
      <td>xxx</td>
    </tr>
  </tfoot>
</table>
    table属性作用示例
    border表格边框宽度（像素），border="0" 表示无边框（默认）。<table border="1">
    width表格宽度（像素或百分比），建议用 CSS 替代。<table width="800"> 或 width="80%"
    height表格高度（像素或百分比），建议用 CSS 替代。<table height="400">
    cellspacing单元格之间的间距（像素），建议用 CSS border-spacing 替代。<table cellspacing="5">
    cellpadding单元格内容与边框的内边距（像素），建议用 CSS padding 替代。<table cellpadding="10">
    align表格在页面中的对齐方式（left/center/right），建议用 CSS 替代。<table align="center">
    bgcolor表格背景色，建议用 CSS background-color 替代。<table bgcolor="#f5f5f5">
    tr属性作用示例
    align行内所有单元格内容的水平对齐（left/center/right）。<tr align="center">
    valign行内所有单元格内容的垂直对齐（top/middle/bottom）。<tr valign="top">
    bgcolor行的背景色，建议用 CSS 替代。<tr bgcolor="#eee">
    th和td属性作用示例
    width/height单元格宽度 / 高度（像素或百分比），建议用 CSS 替代。<td width="150">
    align/valign单元格内容的水平 / 垂直对齐，同 <tr> 的 align/valign。<td align="right" valign="middle">
    bgcolor单元格背景色，建议用 CSS 替代。<td bgcolor="#fff">
    colspan横向合并单元格（跨列），值为合并的列数。<td colspan="2">（合并当前列和右侧 1 列）
    rowspan纵向合并单元格（跨行），值为合并的行数。<td rowspan="3">（合并当前行和下方 2 行）
    nowrap单元格内容不换行（布尔属性），建议用 CSS white-space: nowrap 替代。<td nowrap>

    横向合并（colspan）

    <table border="1">
  <tr>
    <th colspan="2">学生信息表</th> <!-- 横向合并2列 -->
  </tr>
  <tr>
    <th>姓名</th>
    <th>年龄</th>
  </tr>
  <tr>
    <td>张三</td>
    <td>18</td>
  </tr>
</table>

    纵向合并（rowspan）

    <table border="1">
  <tr>
    <th>班级</th>
    <th>姓名</th>
  </tr>
  <tr>
    <td rowspan="3">高一(1)班</td> <!-- 纵向合并3行 -->
    <td>张三</td>
  </tr>
  <tr>
    <td>李四</td>
  </tr>
  <tr>
    <td>王五</td>
  </tr>
</table>
  • <audio>

    音频标签

    <audio src="音频文件路径" controls>
	您的浏览器不支持 audio 标签
</audio>
    属性名作用说明
    src指定音频文件的 URL 路径（必填，支持相对路径或绝对路径）。
    controls显示浏览器默认的播放控件（包含播放 / 暂停、音量、进度条、播放速度等）。
    autoplay音频加载完成后自动播放（注意：多数浏览器（如 Chrome、Safari）为避免打扰用户，默认禁止自动播放，需配合 muted 使用）。
    muted音频默认静音播放（可与 autoplay 配合，绕过自动播放限制）。
    loop音频播放结束后自动循环播放。
    preload指定音频预加载方式（可选值：auto 自动预加载；metadata 只预加载元数据；none 不预加载）。
    volume音频音量（0.0~1.0，可通过 JavaScript 控制，如 audio.volume = 0.5）。

    视频标签

    <video src="视频文件路径" controls>
  您的浏览器不支持 video 标签。 <!-- 浏览器不支持时显示的备用文本 -->
</video>
    属性名作用说明
    src指定视频文件的 URL 路径（必填，支持相对路径或绝对路径）。
    controls显示浏览器默认的播放控件（包含播放 / 暂停、进度条、音量、全屏等）。
    autoplay视频加载完成后自动播放（注意：多数浏览器要求配合 muted 使用，否则会被阻止）。
    muted视频默认静音播放（可与 autoplay 配合，绕过自动播放限制）。
    loop视频播放结束后自动循环播放。
    preload指定预加载方式：- auto：自动预加载视频（默认）- metadata：只预加载元数据（时长、尺寸等）- none：不预加载
    width/height设置视频播放器的宽度 / 高度（单位：px），建议同时设置以避免页面布局抖动。
    poster指定视频加载前或未播放时显示的封面图（URL 路径）。
    playsinline允许视频在页面内播放（而非全屏，尤其适用于 iOS 设备）。
  • <datalist>

    用于给输入框（如input）提供预设选项列表，它可以让用户在输入时看到推荐选项，同时允许自由输入内容。

    datalist本身不会直接显示，而是通过id与<input>的list属性关联，为输入框提供可选值。

    <input type="text" list="optionslist" placeholder="请输入选择...">
<datalist id="optionslist">
    <option value="手表"></option>
    <option value="表带"></option>
    <option value="袋子"></option>
    <option value="儿子"></option>
</datalist>

    option标签可以添加label属性，用于显示不同于value的提示文本（但实际提交的值仍然为value）

    <datalist id="cities">
  <option value="Beijing" label="北京">
  <option value="Shanghai" label="上海">
</datalist>
<input list="cities" placeholder="输入城市...">
