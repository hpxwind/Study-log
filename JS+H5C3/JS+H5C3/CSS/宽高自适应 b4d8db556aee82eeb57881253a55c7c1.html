宽高自适应

  • 宽度自适应：元素宽度随外部容器或浏览器窗口变化而自动调整。

    1、块级元素默认特性：

    div，p，section等，默认情况下会根据父元素的全部可用宽度（width：100%），并且会自动换行

    .box {
  /* 无需设置width，默认占满父元素宽度 */
  background: #f0f0f0;
}

    2、通过width：百分比 让元素宽度相对于父元素宽度自适应

    .child {
  width: 50%; /* 宽度为父元素宽度的50% */
}

    3、使用max-width 和min-width 限制范围

    max-width：设置元素的最大宽度，避免元素过宽（如在大屏幕上内容过度拉伸）。

    min-width：设置元素的最小宽度，避免元素过窄（如在小屏幕上内容挤压变形）。

    .container {
  width: 80%;
  max-width: 1200px; /* 最大不超过1200px */
  min-width: 320px; /* 最小不小于320px（适配手机） */
  margin: 0 auto; /* 居中显示 */
}

    4、浮动与overflow配合，

    浮动元素（float: left/right）的宽度默认由内容决定，但可通过父元素的 overflow: hidden 触发 BFC（块级格式化上下文），让父元素自适应包含浮动子元素的宽度。

  • 高度自适应：元素高度默认由内容撑开，若需随父元素或窗口变化，需特殊处理。

    1、若元素未设置固定高度时，高度会默认由内部内容（文本，子元素等）撑开，内容增加时，高度自动变大。

    .content {
  /* 不设置height，高度随内容变化 */
  padding: 20px;
  background: #eee;
}

    2、百分比限制

    与宽度不同，height: 百分比 生效的前提是父元素必须有明确的高度（不能是默认的 auto）。

    .parent {
  height: 500px; /* 父元素明确高度 */
}
.child {
  height: 50%; /* 子元素高度为父元素的50%（即250px） */
}

    3、全屏高度自适应

    要让元素高度占满整个浏览器窗口，可借助 html 和 body 传递高度

    html, body {
  height: 100%; /* 让html和body占满窗口高度 */
  margin: 0; /* 清除默认边距 */
}
.full-screen {
  height: 100%; /* 继承body的100%高度，即全屏 */
}

    4、使用min-height和max-height 限制范围

    min-height：设置元素最小高度，内容不足时保持该高度；内容超出时自动撑开。

    max-height：设置元素最大高度，内容超出时会出现滚动条（需配合 overflow: auto）。

    .scroll-box {
  min-height: 100px;
  max-height: 300px;
  overflow: auto; /* 内容超出max-height时显示滚动条 */
}

    5、弹性布局（Flexbox）

    弹性容器中，子元素，可以通过flex属性自动分配父元素的剩余高度

    .flex-container {
  display: flex;
  flex-direction: column;
  height: 500px; /* 父容器固定高度 */
}
.flex-child {
  flex: 1; /* 自动占满父容器剩余高度 */
}

  • 浮动元素的父元素（高度塌陷）高度自适应

    使用伪元素清除（清除浮动）

    .box::after {
	  content: "";
	  clear: both;
	  display: block;
	  width: 0; /* 避免显示内容 */
	  height: 0; /* 避免显示内容 */
	  visibility: hidden; /* 占位隐藏 */
    /* display: none; 不占位隐藏 */ 
}
  • 窗口自适应

    正常设置div等盒子时，设置height，width设置自适应，会相对于祖先盒子html和body。

    但默认html，body的height为0，所以div盒子为高度height为0，更改为如下即可

    html, body {
		height: 100%;
}
  • 特殊场景

    1、图片自适应

    图片默认按照原始尺寸，可能导致溢出容器，通过以下方式自适应:

    img {
	max-width: 100%; /* 宽度不超过父容器，自动缩小 */
	height: auto; /* 高度按比例自适应，避免变形 */
}

    2、表格自适应

    表格（table）可以通过width：100% 自适应父容器宽度，同时table-layout：fixed 让列宽均匀分配：

    table {
  width: 100%;
  table-layout: fixed; /* 列宽按比例分配，不随内容变化 */
}

    3、视窗单位（vw/vh）

    使用视窗单位可让元素宽高直接相对于浏览器窗口尺寸自适应：

    vw：1vw = 视窗宽度的 1%

    vh：1vh = 视窗高度的 1%

    .header {
  width: 100vw; /* 宽度等于窗口宽度 */
  height: 10vh; /* 高度为窗口高度的10% */
}
