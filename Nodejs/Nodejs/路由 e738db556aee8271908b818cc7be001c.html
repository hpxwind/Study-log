路由

一、路由是什么

路由 = 地址 + 请求方式 → 对应执行代码

浏览器访问不同网址，后端匹配路由，返回对应内容。

二、基础环境

npm init -y
npm install express

三、最简路由完整代码

const express = require('express')
const app = express()

// 1.根路由 GET
app.get('/', (req, res) => {
  res.send('首页')
})

// 2.普通路由
app.get('/news', (req, res) => {
  res.send('新闻页面')
})

// 3.动态路由（可变参数）
app.get('/user/:id', (req, res) => {
  let id = req.params.id
  res.send(`用户ID：${id}`)
})

// 4.POST路由
app.post('/login', (req, res) => {
  res.send('登录请求')
})

// 404兜底路由
app.all('*', (req, res) => {
  res.send('页面不存在')
})

app.listen(3000, () => {
  console.log('服务启动：http://localhost:3000')
})

四、路由核心三要素

  1. 请求方式：get、post、put、delete
  1. 请求地址：/ /news /user/123
  1. 回调函数：处理逻辑、返回数据

五、常用路由写法

1. 查询参数路由

地址：/search?name=张三&age=20

app.get('/search', (req, res) => {
  console.log(req.query) // {name:'张三',age:'20'}
  res.send(req.query)
})

2. 动态参数路由

地址：/goods/1001

app.get('/goods/:gid', (req, res) => {
  console.log(req.params.gid)
  res.send(`商品编号：${req.params.gid}`)
})

3. 多请求统一路由

app.route('/test')
.get((req,res)=>res.send('get请求'))
.post((req,res)=>res.send('post请求'))

六、模块化路由（项目真实用法）

1. 新建路由文件 routes/user.js

const express = require('express')
const router = express.Router()

router.get('/list', (req, res) => {
  res.send('用户列表')
})

module.exports = router

2. 主入口引入挂载

const userRouter = require('./routes/user')
// 统一前缀 /user
app.use('/user', userRouter)

访问：/user/list

七、路由执行顺序

从上到下依次匹配，匹配成功就不再往下走

兜底 404 必须写在所有路由最后

八、常用请求对象

  • req.params：获取动态路由参数
  • req.query：获取问号查询参数
  • req.body：获取 post 表单数据（需配置中间件）

九、路由中间件拦截

// 全局拦截所有路由
app.use((req, res, next) => {
  console.log('请求地址：', req.url)
  next() // 放行
})
