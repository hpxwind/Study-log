洛谷P1990 覆盖墙壁（dp推理）

  • 题目

    题目描述

    你有一个长为 N 宽为 2 的墙壁，给你两种砖头：一个长 2 宽 1，另一个是 L 型覆盖 3 个单元的砖头。如下图：

    0 0
    0 00

    砖头可以旋转，两种砖头可以无限制提供。你的任务是计算用这两种来覆盖 N×2 的墙壁的覆盖方法。例如一个 2×3 的墙可以有 5 种覆盖方法，如下：

    012 002 011 001 011
    012 112 022 011 001

    注意可以使用两种砖头混合起来覆盖，如 2×4 的墙可以这样覆盖：

    0112
    0012

    给定 N，要求计算 2×N 的墙壁的覆盖方法。由于结果很大，所以只要求输出最后 4 位。例如 2×13 的覆盖方法为 13465，只需输出 3465 即可。如果答案少于 4 位，就直接输出就可以，不用加前导 0，如 N=3 时输出 5。

    输入格式

    一个整数 N，表示墙壁的长。

    输出格式

    输出覆盖方法的最后 4 位，如果不足 4 位就输出整个答案。

    输入输出样例

    输入 #1复制运行

    13

    输出 #1复制运行

    3465
  • 推导

    我们定义：

    • dp[i] = 铺满 2×i 墙壁的总方法数

    第一步：写出原始递推关系

    我们从最后一块区域怎么铺来分类：

    情况 1：最后放 1 个竖的 2×1 砖头

    前面剩下

    2×(i-1)

    贡献：

    dp[i-1]

    情况 2：最后放 2 个横的 1×2 砖头

    刚好铺满 2×2，前面剩下

    2×(i-2)

    贡献：

    dp[i-2]

    情况 3：最后用 L 型砖头（关键！）

    L 型砖头

    必须两个一起用

    ，才能拼成完整的 2×3 区域！而且 L 型有

    两种摆法

    ：

    • 凸在左边
    • 凸在右边

    所以：

    • 用 L 型铺满 2×3 → 2 种方法
    • 用 L 型铺满 2×4 → 2*dp [1] 种
    • 用 L 型铺满 2×5 → 2*dp [2] 种
    • ...
    • 用 L 型铺满 2×i → 2 * dp[i-3]

    第二步：合并所有情况

    把上面三类加起来：

    plaintext

    dp[i] = dp[i-1] + dp[i-2] + 2*(dp[i-3] + dp[i-4] + ... + dp[0])

    第一种：       第二种：
□  □            □  □
□  □ □        □ □  □

    这是原始公式，但我们可以简化成更短的式子！

    第三步：两式相减，推导出最简公式

    我们写出 i 和 i-1 的式子：

    ① i 式

    plaintext

    dp[i] = dp[i-1] + dp[i-2] + 2*(dp[i-3] + ... + dp[0])

    ② i-1 式

    plaintext

    dp[i-1] = dp[i-2] + dp[i-3] + 2*(dp[i-4] + ... + dp[0])

    ① - ② 得到：

    plaintext

    dp[i] - dp[i-1] = dp[i-1] + dp[i-3]

    移项后：

    ✔ 最终公式

    plaintext

    dp[i] = 2 * dp[i-1] + dp[i-3]

import java.util.*;

public class Main{
  static int  MOD = 10000;
  
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);
    int n = sc.nextInt();

    int[] dp = new int[n+1];
    if (n>=0) dp[0] = 1;
    if (n>=1) dp[1] = 1;
    if (n>=2) dp[2] = 2;

    for (int i=3; i<=n; i++) {
        dp[i] = (2*dp[i-1] + dp[i-3]) % MOD;
    }

    System.out.println(dp[n]);
  }
}
