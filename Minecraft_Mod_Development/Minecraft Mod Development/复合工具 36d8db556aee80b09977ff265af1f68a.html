复合工具

这里实现了一个斧头+镐子的工具

  • 类名：PickaxeAxeItem（镐斧合一工具）
  • 继承：AxeItem（原版斧子）

自定义工具类

  • com.hpxwind.tutorial.item.custom
    • PickaxeAxeItem.java
      package com.hpxwind.tutorial.item.custom;

import com.hpxwind.tutorial.tag.ModBlockTags;
import net.minecraft.world.item.AxeItem;
import net.minecraft.world.item.Item;
import net.minecraft.world.item.ItemStack;
import net.minecraft.world.item.Tier;
import net.minecraft.world.level.block.state.BlockState;

public class PickaxeAxeItem extends AxeItem {
    public PickaxeAxeItem(Tier pTier, float pAttackDamageModifier, float pAttackSpeedModifier, Properties pProperties) {
        super(pTier, pAttackDamageModifier, pAttackSpeedModifier, pProperties);
    }

    @Override
    public float getDestroySpeed(ItemStack pStack, BlockState pState) {
        return pState.is(ModBlockTags.PICKAXE_AXE_MINEABLE) ? this.speed : 1.0F;
    }

    @Override
    public boolean isCorrectToolForDrops(ItemStack stack, BlockState state) {
        return state.is(ModBlockTags.PICKAXE_AXE_MINEABLE);
    }
}
    1. 构造函数（固定）
      public PickaxeAxeItem(Tier pTier, float pAttackDamageModifier, float pAttackSpeedModifier, Properties pProperties) {
    super(pTier, pAttackDamageModifier, pAttackSpeedModifier, pProperties);
}
      1. 参数：
        • pTier：工具材质（如钻石、铁、自定义材质）
        • pAttackDamageModifier：攻击伤害（斧子默认比镐高）
        • pAttackSpeedModifier：攻击速度
        • pProperties：物品属性（ Creative 标签、耐久等）
      1. super(...)：调用原版斧子的构造，保留斧子基础特性（砍树、去皮、高伤害
    1. 重写 getDestroySpeed（挖掘速度）
      @Override
public float getDestroySpeed(ItemStack pStack, BlockState pState) {
    return pState.is(ModBlockTags.PICKAXE_AXE_MINEABLE) ? this.speed : 1.0F;
}
      1. 作用：控制挖掘方块的速度
      1. 逻辑：
        • 如果方块属于 ModBlockTags.PICKAXE_AXE_MINEABLE（自定义标签，比如石头、矿石）→ 用镐的速度（this.speed，来自 Tier）
        • 否则 → 速度 1.0F（徒手速度，等于不能快速挖）
      1. 效果：对石头 / 矿石快，对其他方块慢
    1. 重写 isCorrectToolForDrops（是否能掉落物品）
      @Override
public boolean isCorrectToolForDrops(ItemStack stack, BlockState state) {
    return state.is(ModBlockTags.PICKAXE_AXE_MINEABLE);
}
      1. 作用：判断工具是否能让方块掉落物品（比如挖钻石矿掉钻石）
      1. 逻辑：
        • 方块在自定义标签里 → 能掉落（像镐一样）
        • 否则 → 不能掉落（比如用它挖泥土，不掉泥土）
      1. 效果：只有指定方块才掉东西，防止滥用
    1. 与原版工具对比
    特性原版斧子（Axe）原版镐（Pickaxe）你的 PickaxeAxeItem
    砍树速度快慢快（继承斧子）
    挖石头 / 矿石速度慢快快（自定义逻辑）
    挖矿石掉物品❌✅✅（自定义逻辑）
    攻击伤害高低高（继承斧子）

定义完工具还需要定义一个自定义方块标签，用来存 “能被你的镐斧快速挖并掉落物品” 的所有方块。

  • com.hpxwind.tutorial.tag
    • ModBlockTags.java
      public static final TagKey<Block> PICKAXE_AXE_MINEABLE = createForgeTag("tools/pickaxe_axe_mineable");

    1. 类型：TagKey<Block> → 一个 “方块标签的键”，用来标记一组方块。
    1. 名字：PICKAXE_AXE_MINEABLE → 常量名，意思是 “可被镐斧挖掘的方块”。
    1. 值来源：createForgeTag(...) → 模组自己封装的一个创建 Forge 标签的工具方法。
    1. 路径："tools/pickaxe_axe_mineable" → 标签的注册名（资源路径）。
