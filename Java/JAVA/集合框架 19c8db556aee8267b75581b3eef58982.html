集合框架

什么是集合？

一个长度可变，可以保存任何数据类型的动态数组

在java中由一组接口和类共同构成的框架体系（而非一个类）

大致可以分为三层：最上层是一组接口，继而是接口的实现类，接下来就是对集合各种操作的工具类

接口描述
Collection结合框架最基础的接口，一个Collection存储一组无需，不唯一的对象，一般不直接使用该接口
ListCollection的子接口，存储一组有序，不唯一的对象，开发中常用
SetCollection的子接口，存储一组无序，唯一的对象
Map独立于Collection的另外一个接口，存储一组键值对，提供键到值的映射
Iterator专门用来输出集合元素的接口，一般适用于无序集合，从前往后输出元素
ListIteratorIterator的子接口，可以双向输出集合中的元素
Enumeration传统的输出接口，已经被Iterator取代
SortedSetSet的子接口，可以对集合元素进行排序
SortedMapMap的子接口，可以对集合中的键元素进行排序
Queue队列接口，此接口实现类可以实现队列操作
Map.EntryMap的内部接口，描述Map中的一个键值对元素

ArrayList 实现了长度可变的数组

优点：查找快，因为元素是连续的，可以快速求出下标从而取出元素

缺点：增删慢，因为元素是连续的，需要顺序找到增删的元素再增删

int类型ArrayList第n个元素的地址 = n*0004

LinkedList采用链表的形式来存储数据，何ArrayList恰好相反

优点：增删快

缺点：查询慢

Set接口

set接口是Collection的子接口，Set接口以散列的形式存储数据，所以元素是无序的，可以存储一组无序且唯一的对象。

Set接口的实现类

HashSet存储一组无序且唯一的元素，这里的无序是指元素的存储顺序和遍历顺序不一致

  • 实现
    public class HashSetDemo {
    public static void main(String[] args) {
        Set<String> set = new HashSet<>();
        set.add("banana");
        set.add("apple");
        set.add("orange");
        set.add("apple"); // 重复，不会被添加

        System.out.println("HashSet 遍历结果：");

            System.out.println(s);

    }
}

    类的重写

    import java.util.Set;
import java.util.TreeSet;

class Student implements Comparable<Student> {
    String name;
    int score;

    Student(String name, int score) {
        this.name = name;
        this.score = score;
    }

    @Override
    public int compareTo(Student o) {
        // 按分数升序，分数相同则按姓名
        if (this.score != o.score) {
            return Integer.compare(this.score, o.score);
        }
        return this.name.compareTo(o.name);
    }

    @Override
    public String toString() {
        return name + ":" + score;
    }
}

public class TreeSetCustom {
    public static void main(String[] args) {
        Set<Student> set = new TreeSet<>();
        set.add(new Student("Bob", 85));
        set.add(new Student("Alice", 90));
        set.add(new Student("Charlie", 85)); // 分数相同，姓名不同，视为不同元素

        for (Student s : set) {
            System.out.println(s);
        }
        // 输出顺序：Bob:85, Charlie:85, Alice:90（按分数升序，分数相同时按姓名）
    }
}

泛型

泛型是指在类定义时不指定类中信息的具体数据类型，而是用一个标识符来替代，当外部实例化对象的时候再来指定具体的数据。

多态的思想

  • 例子
    // 定义一个泛型类，类型参数为 T
public class Box<T> {
    private T content;

    public void set(T content) {
        this.content = content;
    }

    public T get() {
        return content;
    }

    public static void main(String[] args) {
        // 存放 String
        Box<String> stringBox = new Box<>();
        stringBox.set("Hello");
        String str = stringBox.get();   // 不需要强制类型转换
        System.out.println(str);

        // 存放 Integer
        Box<Integer> intBox = new Box<>();
        intBox.set(123);
        Integer num = intBox.get();     // 自动拆装箱，也不需强转
        System.out.println(num);
    }
}

枚举

Enum 是一种有确定取值区间的数据类型，本质上是一种类，具有简洁，高校，安全，方便的特点

枚举的值被约束到一个特定的范围，只能从该范围中取值。

  • 例子
    // 定义一个表示季节的枚举
public enum Season {
    SPRING, SUMMER, AUTUMN, WINTER
}

// 使用枚举
public class Main {
    public static void main(String[] args) {
        // 1. 直接使用枚举常量
        Season s = Season.SPRING;
        System.out.println(s);  // 输出: SPRING

        // 2. 遍历所有枚举值
        for (Season season : Season.values()) {
            System.out.println(season + " 的序号是 " + season.ordinal());
        }

        // 3. 在 switch 中使用枚举
        switch (s) {
            case SPRING:
                System.out.println("春天");
                break;
            case SUMMER:
                System.out.println("夏天");
                break;
            case AUTUMN:
                System.out.println("秋天");
                break;
            case WINTER:
                System.out.println("冬天");
                break;
        }
    }
}

    public enum Planet {
    MERCURY(3.303e23, 2.4397e6),
    VENUS(4.869e24, 6.0518e6),
    EARTH(5.976e24, 6.37814e6);

    private final double mass;   // 质量
    private final double radius; // 半径

    Planet(double mass, double radius) {
        this.mass = mass;
        this.radius = radius;
    }

    public double getMass() { return mass; }
    public double getRadius() { return radius; }

    // 计算表面重力加速度
    public double surfaceGravity() {
        final double G = 6.67430e-11;
        return G * mass / (radius * radius);
    }
}

System.out.println(earth.surfaceGravity());  // 约 9.80

日期类

Java 的日期时间 API 经历了从 java.util.Date → java.util.Calendar → java.time

java.util.Date

Date 类代表一个特定的时间点，内部以 long 值存储自 1970-01-01 00:00:00 UTC 以来的毫秒数。它试图同时表示日期和时间，但设计上存在诸多问题。

// 当前时间
Date now = new Date();

// 根据时间戳创建
Date date = new Date(1672531200000L); // 2023-01-01 00:00:00 UTC

// 获取/设置时间戳（毫秒）
long time = date.getTime();
date.setTime(time + 86400000); // 加一天

// 比较
boolean before = date.before(now);
boolean after = date.after(now);
int cmp = date.compareTo(now);

java.util.Calendar

Calendar 是一个抽象类，提供了日期字段（年、月、日、时、分、秒等）的运算和转换功能，弥补了 Date 的部分不足。通常使用 GregorianCalendar 子类。

// 获取实例（默认时区和区域）
Calendar cal = Calendar.getInstance();

// 设置特定日期
cal.set(2026, Calendar.MARCH, 29); // 月份使用常量，但注意 Calendar.MARCH = 2

// 获取字段
int year = cal.get(Calendar.YEAR);
int month = cal.get(Calendar.MONTH);     // 3月 = 2
int day = cal.get(Calendar.DAY_OF_MONTH);

// 日期运算
cal.add(Calendar.DAY_OF_MONTH, 10);      // 加10天
cal.add(Calendar.MONTH, -1);             // 减1个月

// Date 与 Calendar 互转
Date date = cal.getTime();
cal.setTime(date);

java.time（Java 8+，JSR-310）

  • 合理的索引：月份 1~12，星期⼀~⽇。

File类

import java.io.File;
import java.io.IOException;

public class FileDemo {
    public static void main(String[] args) {
        // 1. public File(String path)  - 根据路径创建文件对象（不创建实际文件）
        File file = new File("test.txt");
        File dir = new File("myFolder");

        // 2. public String getName()    - 获取文件名
        System.out.println("文件名：" + file.getName());   // test.txt

        // 3. public String getParent()  - 获取文件所在的目录（字符串形式）
        System.out.println("父目录：" + file.getParent()); // 若构造时未指定父路径，返回 null

        // 4. public File getParentFile() - 获取文件所在的目录对应的 File 对象
        File parentFile = file.getParentFile();           // 同上，返回 File 对象

        // 5. public String getPath()     - 获取文件路径（构造时传入的路径）
        System.out.println("路径：" + file.getPath());     // test.txt

        // 6. public boolean exists()     - 判断对象是否存在
        System.out.println("是否存在：" + file.exists());  // false（还未创建）

        // 7. public boolean isDirectory() - 判断是否为目录
        System.out.println("是否为目录：" + file.isDirectory()); // false

        // 8. public boolean isFile()      - 判断是否为文件
        System.out.println("是否为文件：" + file.isFile());      // false

        // 9. public long length()         - 获取文件大小（字节）
        System.out.println("文件大小：" + file.length());       // 0（不存在或目录返回0）

        // 10. public boolean createNewFile() - 创建文件（如果不存在）
        try {
            if (file.createNewFile()) {
                System.out.println("文件创建成功");
            }
        } catch (IOException e) {
            e.printStackTrace();
        }

        // 11. public boolean delete()       - 删除文件或空目录
        if (file.delete()) {
            System.out.println("文件已删除");
        }

        // 12. public boolean mkdir()         - 创建单级目录
        if (dir.mkdir()) {
            System.out.println("目录创建成功");
        }

        // 13. public boolean renameTo(File file) - 重命名（或移动）文件
        File renamed = new File("renamed.txt");
        if (file.renameTo(renamed)) {
            System.out.println("重命名成功");
        }
    }
}

    import java.io.*;

public class InputStreamDemo {
    public static void main(String[] args) {
        // 假设有一个文件 test.txt，内容为 "Hello"
        try (InputStream is = new FileInputStream("test.txt")) {

            // 1. public int read() —— 读取一个字节，返回 int（0-255），读到末尾返回 -1
            int b = is.read();
            System.out.println("第一个字节: " + (char) b); // 输出 H

            // 2. public int read(byte[] bytes) —— 将数据存入 byte 数组，返回实际读取的字节数
            byte[] buffer = new byte[10];
            int len = is.read(buffer); // 读取剩余内容 "ello"
            System.out.println("读取了 " + len + " 个字节: " + new String(buffer, 0, len)); // 输出 ello

            // 重新打开流演示第三个方法（因为上面已经读完）
        } catch (IOException e) {
            e.printStackTrace();
        }

        // 重新打开流来演示 read(byte[], int, int)
        try (InputStream is = new FileInputStream("test.txt")) {
            byte[] buffer = new byte[10];
            // 3. public int read(byte[] bytes, int off, int len)
            //    将数据存入 byte 数组的指定区间，返回实际读取长度
            int len = is.read(buffer, 2, 3); // 从索引2开始放，最多放3个字节
            System.out.println("读取了 " + len + " 个字节");
            System.out.println("数组内容: " + Arrays.toString(buffer)); // 前两个元素为0，后三个为 'H','e','l'
            System.out.println("字符串表示: " + new String(buffer, 2, len)); // Hel

            // 4. public int available() —— 返回当前流中未读取的字节数（估计值）
            int remaining = is.available();
            System.out.println("剩余未读取字节数: " + remaining); // 因为文件共5字节，已读3个，剩余2个

            // 5. public void close() —— 关闭流，释放资源（已在 try-with-resources 中自动关闭）
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

序列化与反序列化

如果把一个Java对象存入文件中，则需要对该对象进行序列化处理，而需要从文件输出出来就需要反序列化处理

可序列化类

import java.io.Serializable;

// 类必须实现 Serializable 接口
public class Person implements Serializable {
    // 建议显式声明 serialVersionUID，保证版本一致性
    private static final long serialVersionUID = 1L;

    private String name;
    private int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    @Override
    public String toString() {
        return "Person{name='" + name + "', age=" + age + "}";
    }
}

序列化与反序列化过程

import java.io.*;

public class SerializationDemo {
    public static void main(String[] args) {
        String fileName = "person.ser";

        // 创建一个 Person 对象
        Person originalPerson = new Person("张三", 25);
        System.out.println("原始对象: " + originalPerson);

        // 序列化：将对象写入文件
        try (ObjectOutputStream oos = new ObjectOutputStream(new FileOutputStream(fileName))) {
            oos.writeObject(originalPerson);
            System.out.println("序列化完成，对象已写入 " + fileName);
        } catch (IOException e) {
            e.printStackTrace();
        }

        // 反序列化：从文件读取对象
        try (ObjectInputStream ois = new ObjectInputStream(new FileInputStream(fileName))) {
            Person deserializedPerson = (Person) ois.readObject();
            System.out.println("反序列化得到的对象: " + deserializedPerson);
        } catch (IOException | ClassNotFoundException e) {
            e.printStackTrace();
        }
    }
}

Class类是反射的基础

Class是专门用来描述其他类的类，每一个Class对象都是对某个类的具体描述

获取class类的三种方法：

  1. 调用forName方法
  1. 通过目标类的类字面量获取
  1. 通过目标类的实例化对象获取
package test;

public class Main {
    public static void main(String[] args) throws ClassNotFoundException {
        // forName
        Class clazz1 = Class.forName("test.User");
        System.out.println(clazz1); // class User
        // 类字面量
        Class clazz2 = User.class;
        System.out.println(clazz2);
        // 实例化对象
        User user = new User();
        Class clazz3 =  user.getClass();
        System.out.println(clazz3);

        System.out.println(clazz1 == clazz2); // true
        System.out.println(clazz2 == clazz3); // true
    }
}
上述三种方法获取的class类是同一个，因为每一个类在内存中只有一份，对应的对象也只有一个，不是有它创建出来的实例化对象，而是描述它内部结果的对象

    import java.lang.reflect.*;
import java.util.Arrays;

public class ReflectionMethodsDemo {
    public static void main(String[] args) throws Exception {
        // 获取 Class 对象
        Class<?> clazz = DemoClass.class;

        System.out.println("=== 基本信息 ===");
        System.out.println("isInterface(): " + clazz.isInterface());          // false
        System.out.println("isArray(): " + clazz.isArray());                  // false
        System.out.println("isAnnotation(): " + clazz.isAnnotation());        // false
        System.out.println("getName(): " + clazz.getName());                  // DemoClass
        System.out.println("getClassLoader(): " + clazz.getClassLoader());
        System.out.println("getSuperclass(): " + clazz.getSuperclass().getSimpleName()); // Parent
        System.out.println("getPackage(): " + clazz.getPackage());
        System.out.println("getPackageName(): " + clazz.getPackageName());
        System.out.println("getInterfaces(): " + Arrays.toString(clazz.getInterfaces()));
        System.out.println("getModifiers(): " + Modifier.toString(clazz.getModifiers())); // public

        System.out.println("\n=== 字段相关 ===");
        // public 字段（包括继承的）
        System.out.println("getFields():");
        for (Field f : clazz.getFields()) {
            System.out.println("  " + f.getName());
        }
        // 本类声明的所有字段（不限访问权限）
        System.out.println("getDeclaredFields():");
        for (Field f : clazz.getDeclaredFields()) {
            System.out.println("  " + f.getName() + " (" + Modifier.toString(f.getModifiers()) + ")");
        }
        // 通过名称获取 public 字段（包括继承的）
        Field fPublic = clazz.getField("publicField");
        System.out.println("getField(\"publicField\"): " + fPublic.getName());
        // 通过名称获取本类声明的字段（包括 private）
        Field fPrivate = clazz.getDeclaredField("privateField");
        System.out.println("getDeclaredField(\"privateField\"): " + fPrivate.getName());

        System.out.println("\n=== 方法相关 ===");
        // 所有 public 方法（包括继承的）
        System.out.println("getMethods():");
        for (Method m : clazz.getMethods()) {
            System.out.println("  " + m.getName());
        }
        // 本类声明的所有方法（不限访问权限）
        System.out.println("getDeclaredMethods():");
        for (Method m : clazz.getDeclaredMethods()) {
            System.out.println("  " + m.getName() + " (" + Modifier.toString(m.getModifiers()) + ")");
        }
        // 通过名称+参数获取 public 方法（包括继承）
        Method mPublic = clazz.getMethod("publicMethod");
        System.out.println("getMethod(\"publicMethod\"): " + mPublic.getName());
        // 通过名称+参数获取本类声明的方法（包括 private）
        Method mPrivate = clazz.getDeclaredMethod("privateMethod");
        System.out.println("getDeclaredMethod(\"privateMethod\"): " + mPrivate.getName());

        System.out.println("\n=== 构造器相关 ===");
        // 所有 public 构造器
        System.out.println("getConstructors():");
        for (Constructor<?> c : clazz.getConstructors()) {
            System.out.println("  " + c);
        }
        // 本类声明的所有构造器（不限访问权限）
        System.out.println("getDeclaredConstructors():");
        for (Constructor<?> c : clazz.getDeclaredConstructors()) {
            System.out.println("  " + c);
        }
        // 通过参数获取 public 构造器
        Constructor<?> cPublic = clazz.getConstructor(String.class, int.class);
        System.out.println("getConstructor(String.class, int.class): " + cPublic);
        // 通过参数获取本类声明的构造器（包括 private）
        Constructor<?> cPrivate = clazz.getDeclaredConstructor(int.class);
        System.out.println("getDeclaredConstructor(int.class): " + cPrivate);
    }
}
