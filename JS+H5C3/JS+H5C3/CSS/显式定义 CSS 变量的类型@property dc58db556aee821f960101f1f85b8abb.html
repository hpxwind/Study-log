显式定义 CSS 变量的类型@property

@property 是 CSS Houdini 规范中的一个强大特性，用于显式定义 CSS 变量的类型、初始值和继承行为，让 CSS 变量从 “无类型的值” 升级为 “带类型的可控变量”。

  • 语法结构与作用
    @property --变量名 {
  syntax: '<类型>';       /* 定义变量的合法值类型（必填） */
  initial-value: 值;      /* 初始值（可选，但建议设置） */
  inherits: 布尔值;       /* 是否允许继承（默认 false） */
}
    • 类型约束：明确变量的合法值类型（如颜色、长度、数字等），浏览器会拒绝不符合类型的值。
    • 支持动画 / 过渡：带类型的变量可直接用于 transition 或 animation，实现平滑动画（普通 CSS 变量不行）。
    • 明确初始值和继承性：规范变量的默认行为，避免样式意外继承。
  • syntax（必填属性）
    1. 定义变量允许的值类型，决定了变量能接受哪些 CSS 值。常见类型：
    1. <color>：颜色（如 #f00、rgb(255,0,0)、blue）。
    1. <length>：长度（如 10px、2em、5rem）。
    1. <number>：数字（如 10、3.14，无单位）。
    1. <percentage>：百分比（如 50%）。
    1. <length-percentage>：长度或百分比（如 10px 或 50%）。
    1. <angle>：角度（如 90deg、1rad）。
    1. 复合类型：用 | 分隔（如 <length> | <percentage> 表示长度或百分比）。
    1. 通配符： 表示接受任意类型（不推荐，失去类型约束意义）。
    /* 仅接受颜色值 */
@property --primary-color {
  syntax: '<color>';
  initial-value: #4285f4;
  inherits: false;
}

/* 接受长度或百分比 */
@property --size {
  syntax: '<length-percentage>';
  initial-value: 100px;
  inherits: true;
}
  • inherits（可选，默认false）
    • true：变量会从父元素继承值（同普通 CSS 变量的继承行为）。
    • false：变量不继承，使用自身定义的值或 initial-value。
    /* 不继承，子元素需显式定义或使用初始值 */
@property --border-width {
  syntax: '<length>';
  initial-value: 1px;
  inherits: false;
}
  • 支持动画与过度

    普通 CSS 变量（未用 @property 定义）无法直接参与 transition 或 animation，因为浏览器不知道其类型，无法计算中间值。而用 @property 定义的变量，因类型明确，可实现平滑动画。

    • 示例1：颜色过度

      效果：鼠标悬停时，盒子背景色从红色平滑过渡到蓝色（普通 CSS 变量无法实现）。

      /* 定义带类型的颜色变量 */
@property --accent {
  syntax: '<color>';
  initial-value: red;
  inherits: false;
}

.box {
  width: 200px;
  height: 200px;
  background: var(--accent);
  transition: --accent 0.3s; /* 对变量本身过渡 */
}

.box:hover {
  --accent: blue; /* hover 时变量变色，触发过渡 */
}
    • 示例2：数字动画

      效果：进度条宽度平滑从 0% 过渡到 100%。

      /* 定义数字变量 */
@property --progress {
  syntax: '<number>';
  initial-value: 0;
  inherits: false;
}

.progress-bar {
  height: 20px;
  width: calc(var(--progress) * 1%); /* 用变量计算宽度 */
  background: green;
  transition: --progress 1s;
}

/* 触发动画：通过 JS 修改 --progress */
      // 点击按钮让进度条从 0% 涨到 100%
document.getElementById('start').addEventListener('click', () => {
  document.documentElement.style.setProperty('--progress', 100);
});
  • 与普通CSS变量的区别
    特性普通 CSS 变量@property 定义的变量
    类型约束无（任意值，浏览器不校验）有（必须符合 syntax 类型）
    动画支持不支持（无法过渡 / 动画）支持（可直接用于 transition）
    初始值管理需手动在 :root 定义内置 initial-value 属性
    继承控制默认继承（需手动覆盖）可通过 inherits 精确控制
    浏览器解析纯字符串替换按类型解析，支持计算中间值
  • 实际案例
    1. 主题色平滑切换
    @property --theme-color {
  syntax: '<color>';
  initial-value: #fff;
  inherits: true;
}

body {
  background: var(--theme-color);
  transition: --theme-color 0.5s;
}

/* 切换主题时，JS 只需修改 --theme-color */
    1. 动态进度指示器
    @property --angle {
  syntax: '<angle>';
  initial-value: 0deg;
  inherits: false;
}

.spinner {
  width: 50px;
  height: 50px;
  border: 5px solid #eee;
  border-top-color: blue;
  border-radius: 50%;
  animation: spin 1s linear infinite;
}

@keyframes spin {
  to {
    --angle: 360deg; /* 用变量控制角度动画 */
    transform: rotate(var(--angle));
  }
}
    1. 响应式动态调整

    结合媒体查询，动态修改带类型的变量，实现更精细的响应式控制：

    @property --content-width {
  syntax: '<length>';
  initial-value: 1200px;
  inherits: false;
}

.container {
  width: min(var(--content-width), 100% - 20px);
}

@media (max-width: 768px) {
  :root {
    --content-width: 700px; /* 小屏幕缩小最大宽度 */
  }
}
  • 兼容性与注意事项

    兼容性

    • 支持浏览器：Chrome 85+、Edge 85+、Firefox 97+（需开启 layout.css.properties.enabled 实验性开关）、Safari 15.4+。
    • IE 完全不支持，需做好降级处理。

    2. 注意事项

    • @property 必须定义在全局（如 :root 外）或选择器内，但通常建议放在全局，确保变量在所有地方可用。
    • syntax 类型必须严格匹配，否则变量会被视为无效（使用 initial-value 或继承值）。
    • 不要滥用 syntax: '*'，会失去类型约束的优势，无法支持动画。
    • 可通过 CSS.registerProperty() 用 JS 动态注册变量：
    CSS.registerProperty({
  name: '--custom-color',
  syntax: '<color>',
  initialValue: 'pink',
  inherits: false
});
