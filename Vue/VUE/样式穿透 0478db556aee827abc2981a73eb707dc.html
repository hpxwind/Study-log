样式穿透

样式穿透是Vue开发中处理样式作用域的重要技术，用于在scoped样式中修改子组件或深层DOM元素的样式。
样式穿透是Vue中用于打破scoped样式限制的技术，允许父组件修改子组件内部的样式。

::v-deep 是专门为 scoped 样式设计的穿透语法。它的作用是：

  • 在保持父组件样式隔离的前提下
  • 允许选择性地穿透到子组件内部
  • vue样式作用域原理
    <!-- 子组件 Child.vue -->
<template>
  <div class="child">
    <button class="btn">子组件按钮</button>
  </div>
</template>

<!-- 编译后 -->
<div class="child" data-v-123456>
  <button class="btn" data-v-123456>子组件按钮</button>
</div>

<style scoped>
/* 编译前 */
.child .btn { color: red; }

/* 编译后 */
.child .btn[data-v-123456] { color: red; }
</style>

不同版本的穿透样式

  • 原生CSS/SCSS
    // 父组件
<style scoped>
li {list-style: none;}

.active {
  background-color: red;
  color: #fff;
}

/* 【样式穿透】使用 ::v-deep */
/* Vue2 和 Vue3 都推荐使用 ::v-deep 语法 */

/* 穿透修改子组件的 .child-text 样式 */
::v-deep .child-text {
  color: red !important;
  font-weight: bold;
}

/* 穿透修改子组件的 .child-box 样式 */
::v-deep .child-box {
  background-color: lightgreen !important;
  border: 2px dashed orange;
}
</style>

// 子组件

<template>
    <div class="child-wrapper">
        <h3>我是子组件ChuanTou</h3>
        <p class="child-text">这段文字的样式可以被父组件穿透修改</p>
        <div class="child-box">
            <span>盒子内容</span>
        </div>
    </div>
</template>

<script>
export default {
    name: 'ChuanTou'
}
</script>

<style scoped>
/* scoped 作用域样式：默认只在当前组件生效 */
.child-wrapper {
    border: 2px solid #333;
    padding: 20px;
    margin: 10px 0;
}

.child-text {
    color: blue;
    font-size: 16px;
}

.child-box {
    background-color: lightgray;
    padding: 10px;
    margin-top: 10px;
}
</style>
    <template>
  <div class="parent">
    <ChildComponent class="child-wrapper" />
  </div>
</template>

<style scoped>
/* Vue 2的写法 - 已废弃但仍支持 */
.parent >>> .child-wrapper .inner-element {
  color: red;
}

/* 或使用/deep/ */
.parent /deep/ .child-wrapper .inner-element {
  background: blue;
}
</style>
  • Sass/SCSS
    <style lang="scss" scoped>
/* Vue 2的写法 */
.parent {
  /* 使用::v-deep */
  ::v-deep .child-wrapper {
    .inner-element {
      color: red;
    }
  }
  
  /* 或使用/deep/ */
  /deep/ .child-wrapper {
    .inner-element {
      background: blue;
    }
  }
}
</style>
  • vue3推荐

    Vue 3统一使用:deep()、:slotted()、:global()选择器：

    <style scoped>
/* 修改子组件样式 */
:deep(.child-wrapper) {
  .inner-element {
    color: red;
  }
}

/* 等同于 */
:deep(.child-wrapper .inner-element) {
  color: red;
}
</style>

各种使用场景

  • 修改子组件根元素
    <template>
  <div class="parent">
    <MyButton class="custom-button" />
  </div>
</template>

<style scoped>
/* 修改子组件根元素 */
:deep(.custom-button) {
  padding: 20px;
  font-size: 16px;
  border-radius: 8px;
}
</style>
  • 修改子组件内部元素
    <template>
  <div class="container">
    <DataTable class="my-table" />
  </div>
</template>

<style scoped>
/* 深入修改表格内部样式 */
:deep(.my-table) {
  .table-header {
    background: linear-gradient(45deg, #409EFF, #67C23A);
    color: white;
    font-weight: bold;
  }
  
  .table-row:hover {
    background-color: #f5f7fa;
  }
  
  .table-cell {
    padding: 12px 16px;
  }
}
</style>
  • 修改第三方组件样式
    <template>
  <div class="page">
    <!-- Element Plus组件 -->
    <el-dialog
      v-model="dialogVisible"
      title="提示"
      width="30%"
      class="custom-dialog"
    >
      <span>这是一段信息</span>
    </el-dialog>
    
    <!-- Ant Design Vue组件 -->
    <a-button type="primary" class="custom-btn">
      自定义按钮
    </a-button>
  </div>
</template>

<script setup>
import { ref } from 'vue'

const dialogVisible = ref(false)
</script>

<style scoped>
/* 修改Element Plus对话框样式 */
:deep(.custom-dialog) {
  .el-dialog__header {
    background: linear-gradient(90deg, #409EFF, #67C23A);
    
    .el-dialog__title {
      color: white;
      font-size: 18px;
    }
    
    .el-dialog__headerbtn {
      .el-dialog__close {
        color: white;
        
        &:hover {
          color: #f0f0f0;
        }
      }
    }
  }
  
  .el-dialog__body {
    padding: 30px 20px;
  }
  
  .el-dialog__footer {
    text-align: center;
  }
}

/* 修改Ant Design按钮样式 */
:deep(.custom-btn) {
  &.ant-btn-primary {
    background: linear-gradient(45deg, #FF6B6B, #4ECDC4);
    border: none;
    border-radius: 25px;
    padding: 0 30px;
    height: 40px;
    
    &:hover {
      transform: translateY(-2px);
      box-shadow: 0 4px 12px rgba(255, 107, 107, 0.3);
    }
  }
}
</style>
  • 插槽内容样式
    <template>
  <div class="card-wrapper">
    <CardComponent>
      <!-- 插槽内容 -->
      <template #header>
        <h3 class="custom-header">自定义标题</h3>
      </template>
      
      <template #default>
        <p class="custom-content">自定义内容区域</p>
      </template>
    </CardComponent>
  </div>
</template>

<style scoped>
/* 修改插槽内容的样式 */
:deep(.custom-header) {
  color: #409EFF;
  border-bottom: 2px solid #409EFF;
  padding-bottom: 10px;
  margin-bottom: 20px;
}

:deep(.custom-content) {
  line-height: 1.8;
  color: #606266;
  font-size: 14px;
}
</style>
  • :slotted()选择器

    专门用于修改插槽内容的样式：

    <!-- ParentComponent.vue -->
<template>
  <div class="parent">
    <slot>
      <span class="default-slot">默认内容</span>
    </slot>
    
    <slot name="footer">
      <div class="default-footer">默认页脚</div>
    </slot>
  </div>
</template>

<style scoped>
/* 修改插槽内容的样式 */
:slotted(.slot-content) {
  color: #f56c6c;
  font-weight: bold;
}

/* 修改具名插槽的样式 */
:slotted(.footer-content) {
  background: #f0f0f0;
  padding: 10px;
  border-radius: 4px;
}

/* 默认插槽内容 */
.default-slot {
  color: #909399;
}

.default-footer {
  color: #c0c4cc;
}
</style>

<!-- 使用组件 -->
<template>
  <ParentComponent>
    <!-- 会被:slotted选择器选中 -->
    <span class="slot-content">插槽内容</span>
    
    <template #footer>
      <div class="footer-content">自定义页脚</div>
    </template>
  </ParentComponent>
</template>
  • :global()选择器

    用于在scoped样式中添加全局样式：

    <template>
  <div class="page-container">
    <h1>页面标题</h1>
    <div class="content">
      <p>页面内容</p>
    </div>
  </div>
</template>

<style scoped>
.page-container {
  padding: 20px;
}

/* 全局样式 - 不受scoped限制 */
:global(.global-class) {
  color: red;
  font-size: 20px;
}

/* 全局修改body样式 */
:global(body) {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  margin: 0;
  font-family: Arial, sans-serif;
}

/* 修改其他组件的全局样式 */
:global(.some-external-component) {
  .internal-element {
    background: blue;
  }
}
</style>

<style>
/* 另一种添加全局样式的方式 */
.global-button {
  padding: 10px 20px;
  background: #409EFF;
  color: white;
  border: none;
  border-radius: 4px;
}
</style>
