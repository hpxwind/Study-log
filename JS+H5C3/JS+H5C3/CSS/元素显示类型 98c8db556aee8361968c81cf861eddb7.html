元素显示类型

  • 块级元素（Block Level Elements）

    特性：

    1. 独占一行：默认会占据父容器的整个宽度，换行显示。
    1. 可设置宽高：可以通过 width、height、padding、margin 等属性控制尺寸和间距。
    1. 背景可见：背景色、边框等会填充整个盒子模型。

    常见标签：

    • <div>、<p>、<h1~h6>、<ul>、<li>、<form> 等。
    div {
  display: block; /* 块级元素（默认值） */
  width: 200px;
  height: 100px;
  background-color: #f0f0f0;
}
  • 行内元素（Inline Elements）

    特性：

    1. 同行显示：多个行内元素会排列在同一行，直到空间不足时才换行。
    1. 无法直接设置宽高：宽高由内容自动撑开，width、height 属性无效（但可通过 line-height 调整行高）。
    1. 盒子模型受限：padding、margin 仅左右方向有效，上下方向会影响相邻行内元素，但不会改变布局。

    常见标签：

    • <span>、<a>、<strong>、<em>、<img>、<button> 等。
    span {
  display: inline; /* 行内元素（默认值） */
  padding: 10px; /* 上下 padding 会显示，但不影响布局 */
  background-color: #ffd700;
}
  • 行内块元素（Inline-Block Elements）

    特性：

    1. 同行显示 + 可设宽高：兼具行内元素和块级元素的特点，多个元素可同行显示，同时支持设置 width、height、padding、margin。
    1. 基线对齐：默认会基于文字基线对齐，可能产生底部间隙（可通过 vertical-align 或父元素 font-size: 0 解决）。

    常见标签：

    • <img>、<input>、<textarea> 等表单元素默认是 inline-block（部分浏览器需手动设置）。
    img {
  display: inline-block; /* 行内块元素 */
  width: 100px;
  height: 100px;
  margin: 5px;
}
  • 其他常用显示模式

    1. display: none

    • 特性：元素完全隐藏，不占据任何空间（脱离文档流）。
    • 应用场景：配合 JavaScript 实现动态显示 / 隐藏效果（如下拉菜单隐藏时）。

    2. display: visibility: hidden

    • 特性：元素隐藏，但仍占据空间（占位隐藏）。
    • 对比：与 display: none 的区别在于是否保留占位。

    3. display: flex

    • 特性：将元素设为 Flex 容器，子元素成为 Flex 项，可通过 Flex 布局属性（如 justify-content、align-items）灵活控制子元素的排列方式。
    • 应用场景：现代响应式布局（如水平垂直居中、等高列、弹性分布等）。

    4. display: grid

    • 特性：将元素设为 Grid 容器，通过网格布局系统定义二维网格，精确控制子元素的行列位置。
    • 应用场景：复杂网格布局（如相册、仪表盘等）。

    5. display: inline-flex / display: inline-grid

    • 特性：行内版本的 Flex/Grid 容器，容器本身为行内元素，可与其他行内元素同行显示，内部子元素遵循 Flex/Grid 布局规则。
  • 模式转换

    通过 display 属性可以强制改变元素的默认显示模式，例如：

    • 行内元素转块级：display: block（如让 <span> 像 <div> 一样独占一行）。
    • 块级元素转行内：display: inline（如让多个 <div> 同行显示）。
    • 块级元素转行内块：display: inline-block（如按钮组同行显示且可设尺寸）。
    /* 让 <a> 标签变为块级元素（常用于导航栏按钮） */
a {
  display: block;
  width: 100px;
  height: 40px;
  line-height: 40px;
  text-align: center;
}

/* 让 <li> 标签变为行内块元素（横向导航栏） */
li {
  display: inline-block;
  margin: 0 10px;
}
