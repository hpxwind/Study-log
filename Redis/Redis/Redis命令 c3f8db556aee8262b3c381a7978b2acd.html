Redis命令

数据结构

  1. 字符串（String）
  1. 哈希（Hash）
  1. 列表（List）
  1. 集合（Set）
  1. 有序集合（Sorted Set）
  1. 扩展数据结构（位图（Bitmap）,HyperLogLog,地理空间（Geospatial）,流（Stream））

通用命令

  • HELP [commods]: 遇到不会的就查看一下
  • KEYS: 查看符合薄板的所有key
    Redis是单线程，keys内置模糊查询，如果数据过多，会阻塞redis服务
    除非是主从线程，将keys放置在从线程中执行
  • DEL: 删除一个或多个key
  • EXISTS: 判断key是否存在
  • EXPIRE: 给一个key设置有效期，有效期到期时该key会被自动删除
  • TTL: 查看一个key的剩余有效期( 值为-2 的时候key被删除; -1为永久有效)

Key的层级格式

Redis的key允许多个单词形成层级结构，中间用”:”分割

项目名:业务名:类型:id

例如:

user:1001:name          → "Alice"
user:1001:email         → "alice@example.com"
user:1001:profile       → JSON 字符串或 Hash key

# 或者整体存储为 Hash
user:1001               → {name: "Alice", email: "..."}
order:2023001:user_id   → 1001
order:2023001:status    → "shipped"
order:2023001:items     → Set 或 List 存储商品ID列表

String类型

最简单的存储类型

其value是字符串，根据字符串的格式不同，可以分为三类：

  • string: 普通字符串
  • int: 整数类型，可以自增，自减操作
  • float: 浮点类型，可以自增，自减操作
不管是那种，最大长度不能超过512字节

常见命令:

命令描述示例
SET设置指定 key 的值，可附加过期时间或条件（NX=不存在时设置，XX=存在时设置）SET name "Alice"
GET获取指定 key 的值GET name → "Alice"
MSET（批量增）同时设置多个 key-value 对MSET a 1 b 2 c 3
MGET（批量获取）获取所有指定 key 的值MGET a b c → ["1","2","3"]
SETNX仅当 key 不存在时设置值（SET if Not eXists）SETNX lock 1
SETEX设置值并指定过期时间（秒）SETEX session 60 "abc123"
PSETEX设置值并指定过期时间（毫秒）PSETEX token 5000 "xyz"
GETSET设置新值并返回旧值（原子操作）GETSET counter 10
INCR将 key 中存储的数字值加 1INCR views
INCRBY将 key 中存储的数字值增加指定整数INCRBY score 5
DECR将 key 中存储的数字值减 1DECR stock
DECRBY将 key 中存储的数字值减少指定整数DECRBY balance 100
INCRBYFLOAT将 key 中存储的数字值增加指定浮点数INCRBYFLOAT price 0.99
APPEND将值追加到原值的末尾APPEND log "new line"
STRLEN返回 key 存储的字符串值的长度STRLEN name
GETRANGE获取字符串的子串（闭区间，支持负数索引）GETRANGE name 0 2
SETRANGE从指定偏移量开始覆盖原值的一部分SETRANGE name 5 "Alice"
DEL删除一个或多个 key（通用命令，也适用于其他类型）

Hash类型

散列，其value为无序字典，类似于java中的HashMap结构

与String类型不同，String是将对象序列化为Json字符串后存储，当需要修改对象某个字符按时很不方便
而Hash结构可以将对象中的每个字段独立存储，可以针对单个字段做CRUD:

Hash类型 value中又分为field:value

常见命令:

命令描述示例
HSET设置 hash 中一个或多个字段的值。若字段已存在则覆盖，返回新增字段个数。HSET user:1001 name "Alice" age 30
HGET获取 hash 中指定字段的值。HGET user:1001 name → "Alice"
HMSET（批量新增）同时设置多个字段（已废弃，推荐使用 HSET）。HMSET user:1001 name "Bob" age 25
HMGET（批量获取）获取多个字段的值，返回列表。HMGET user:1001 name age → ["Alice","30"]
HGETALL返回 hash 中所有字段和值（字段-值交替排列）。HGETALL user:1001 → ["name","Alice","age","30"]
HDEL删除一个或多个字段，返回成功删除的个数。HDEL user:1001 age
HEXISTS判断字段是否存在。存在返回 1，否则 0。HEXISTS user:1001 name → 1
HLEN返回 hash 中字段的数量。HLEN user:1001 → 2
HINCRBY为指定字段的整数值增加 increment（可为负数）。HINCRBY user:1001 age 1
HINCRBYFLOAT为指定字段的浮点数值增加 increment。HINCRBYFLOAT product:1001 price 0.99
HKEYS返回 hash 中所有字段名。HKEYS user:1001 → ["name","age"]
HVALS返回 hash 中所有值。HVALS user:1001 → ["Alice","30"]
HSETNX仅当字段不存在时设置该字段的值（原子操作）。成功返回 1，失败返回 0。HSETNX user:1001 email "alice@xx.com"
HSTRLEN返回字段值的字符串长度（字节数）。HSTRLEN user:1001 name → 5
HSCAN增量迭代 hash 中的字段-值对，用于避免 HGETALL 阻塞。HSCAN user:1001 0 MATCH name*

List类型

类似java中的LinkedList，可以看作是一个双向链表结构。即可以支持正向检索也可以反向检索：

特征也类似：

  • 有序
  • 元素可以复制
  • 插入和删除快
  • 查询速度一般
命令描述示例
LPUSH将一个或多个值插入到列表头部（左侧），返回列表长度。LPUSH queue "task1" "task2"
RPUSH将一个或多个值插入到列表尾部（右侧），返回列表长度。RPUSH queue "task3"
LPOP移除并返回列表头部的一个或count个元素。LPOP queue → "task2"
RPOP移除并返回列表尾部的一个或count个元素。RPOP queue → "task3"
LLEN返回列表长度（元素个数）。LLEN queue → 1
LRANGE返回列表中指定区间内的元素（索引从0开始，支持负数表示倒数）。LRANGE queue 0 -1 → 所有元素
LINDEX通过索引获取列表中的元素。LINDEX queue 0 → "task1"
LSET通过索引设置列表元素的值（索引存在时成功）。LSET queue 0 "newtask"
LINSERT在列表中某个参考值pivot之前或之后插入元素。LINSERT queue BEFORE "task1" "urgent"
LREM从列表中删除与value相等的元素。count>0从左删除，=0删除全部，<0从右删除。LREM queue 1 "task1"
LTRIM修剪列表，只保留指定区间内的元素，其他删除。LTRIM queue 0 1 保留前两个元素
RPOPLPUSH将source列表尾部的元素弹出并推入destination列表头部，原子操作。RPOPLPUSH queue backup
BLPOP从非空列表头部弹出元素，若所有列表为空则阻塞等待timeout秒（0表示永久）。BLPOP queue 0
BRPOP与BLPOP类似，但从尾部弹出。BRPOP queue 10
BRPOPLPUSHRPOPLPUSH的阻塞版本。BRPOPLPUSH queue backup 5

Set类型

与Java中HashSet类似，可以看作是一个value为null的HashMap

同样具备Hash的特性

  • 无序
  • 元素不可重复
  • 查找快
  • 支持交集，并集，差集等功能
命令描述示例
SADD向集合添加一个或多个成员，返回新增成员的数量（忽略已存在的成员）。SADD fruits "apple" "banana" "orange"
SREM移除集合中一个或多个成员，返回成功移除的数量。SREM fruits "banana"
SMEMBERS返回集合中的所有成员。SMEMBERS fruits → ["apple","orange"]
SISMEMBER判断成员是否是集合的元素。是返回1，否返回0。SISMEMBER fruits "apple" → 1
SCARD返回集合的元素个数（基数）。SCARD fruits → 2
SPOP随机移除并返回一个或多个成员。SPOP fruits → "apple"
SRANDMEMBER随机返回一个或多个成员（不移除）。SRANDMEMBER fruits 2
SINTER返回所有给定集合的交集。SINTER set1 set2
SINTERSTORE求交集并将结果存储到新集合destination。SINTERSTORE common set1 set2
SUNION返回所有给定集合的并集。SUNION set1 set2
SUNIONSTORE求并集并将结果存储到新集合。SUNIONSTORE all set1 set2
SDIFF返回第一个集合与其他集合的差集（属于第一个但不属于后续的成员）。SDIFF set1 set2
SDIFFSTORE求差集并将结果存储到新集合。SDIFFSTORE diff set1 set2
SMOVE将成员从源集合移动到目标集合（原子操作）。SMOVE fruits tropical "banana"
SSCAN增量迭代集合中的元素，避免SMEMBERS阻塞。SSCAN fruits 0 MATCH a* COUNT 10

SortedSet类型

有序集合，类似Java中的TreeSet类似（底层实现不同）

sortedSet中每个元素都带有一个score属性，可以基于score属性对元素排序，底层的实现是一个跳表（SkipList）加Hash表。（常用于排行榜等功能）具有如下特性：

  • 可排序
  • 元素不可重复
  • 查询速度快
命令描述示例
ZADD向有序集合添加一个或多个成员，或更新已存在成员的分数。返回新增/修改的数量（取决于选项）。ZADD leaderboard 100 "Alice" 200 "Bob"
ZREM移除有序集合中的一个或多个成员，返回成功移除的数量。ZREM leaderboard "Bob"
ZSCORE返回成员的分数。ZSCORE leaderboard "Alice" → 100
ZINCRBY对指定成员的分数增加增量（可为负数），返回新分数。ZINCRBY leaderboard 50 "Alice" → 150
ZCARD返回有序集合的元素个数。ZCARD leaderboard → 1
ZCOUNT返回分数在 min 和 max 区间内的成员数量（默认闭区间）。ZCOUNT leaderboard 100 200 → 1
ZRANGE按索引范围返回成员（升序）。WITHSCORES 同时返回分数。ZRANGE leaderboard 0 -1 WITHSCORES
ZREVRANGE按索引范围返回成员（降序）。ZREVRANGE leaderboard 0 10 WITHSCORES
ZRANGEBYSCORE按分数范围返回成员（升序）。min/max 支持 -inf/+inf，( 表示开区间。ZRANGEBYSCORE leaderboard 100 300 WITHSCORES
ZREVRANGEBYSCORE按分数范围返回成员（降序，注意参数顺序：先 max 后 min）。ZREVRANGEBYSCORE leaderboard 300 100
ZRANK返回成员在有序集合中的排名（升序，从0开始）。ZRANK leaderboard "Alice" → 0
ZREVRANK返回成员在有序集合中的排名（降序，从0开始）。ZREVRANK leaderboard "Alice" → 1
ZREM移除指定成员（已在上方列出）。
ZREMRANGEBYRANK移除指定排名范围内的成员（按升序排名）。ZREMRANGEBYRANK leaderboard 0 0 移除最低分
ZREMRANGEBYSCORE移除指定分数范围内的成员。ZREMRANGEBYSCORE leaderboard 100 200
ZUNIONSTORE计算多个有序集合的并集，存储到 destination。ZUNIONSTORE union 2 zset1 zset2 WEIGHTS 2 3
ZINTERSTORE计算多个有序集合的交集，存储到 destination。ZINTERSTORE intersect 2 zset1 zset2
ZSCAN增量迭代有序集合中的元素（成员+分数）。ZSCAN leaderboard 0 MATCH a* COUNT 10
ZPOPMIN移除并返回分数最低的一个或多个成员（弹出）。ZPOPMIN leaderboard → ["Alice","100"]
ZPOPMAX移除并返回分数最高的一个或多个成员。ZPOPMAX leaderboard
BZPOPMIN阻塞版本的 ZPOPMIN，若无成员则阻塞等待。BZPOPMIN leaderboard 5
BZPOPMAX阻塞版本的 ZPOPMAX。BZPOPMAX leaderboard 5
