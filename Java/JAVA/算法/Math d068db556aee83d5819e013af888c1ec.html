取整与舍入

方法描述返回类型示例
ceil(a)向上取整，返回大于等于 a 的最小整数doubleMath.ceil(4.2) → 5.0
floor(a)向下取整，返回小于等于 a 的最大整数doubleMath.floor(4.7) → 4.0
rint(a)返回最接近 a 的整数，若距离相等则返回偶数doubleMath.rint(4.5) → 4.0
round(a)四舍五入（等价于 floor(a+0.5)）int (float) 或 long (double)Math.round(4.5) → 5
floorDiv(x, y)返回 x 除以 y 的商，结果向下取整int 或 longMath.floorDiv(-7, 3) → -3
floorMod(x, y)返回 x 除以 y 的余数，结果符号与 y 相同int 或 longMath.floorMod(-7, 3) → 2
