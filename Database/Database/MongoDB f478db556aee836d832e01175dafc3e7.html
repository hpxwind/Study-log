  • 文档操作方法
    方法功能描述示例
    insertOne()插入单个文档db.users.insertOne({ name: "Alice", age: 30 })
    insertMany()插入多个文档（参数为数组）db.users.insertMany([{ name: "Bob" }, { name: "Charlie" }])
    insert()（不推荐）插入单个或多个文档（已弃用，用上面两个）db.users.insert({ name: "Dave" })
    • 文档-插入
      功能描述示例
      insertOne()插入单个文档
      insertMany()插入多个文档（参数为数组）
      insert()（不推荐）插入单个或多个文档（已弃用，用上面两个）
      db.<集合名称>.insertOne(<文档>)
db.<集合名称>.insertMany([<文档1>, <文档2>,...])
    • 文档-查找
      方法功能描述示例
      find()查询匹配条件的所有文档（返回游标）db.users.find({ age: { $gt: 25 } }) → 年龄 > 25 的文档
      findOne()查询匹配条件的第一条文档db.users.findOne({ name: "Alice" })
      find().toArray()将游标转换为数组（方便查看结果）db.users.find().toArray()
      find().limit(n)限制返回结果数量db.users.find().limit(10) → 最多返回 10 条
      find().skip(n)跳过前 n 条文档（用于分页）db.users.find().skip(20).limit(10) → 第 21-30 条
      find().sort({ key: 1 })排序（1：升序，-1：降序）db.users.find().sort({ age: -1 }) → 按年龄降序
      find().countDocuments()统计匹配文档数量（MongoDB 3.6+）db.users.find({ age: { $gt: 25 } }).countDocuments()
      db.<集合名称>.find().toArray()

db.<集合名称>.find({<条件>}).toArray()
如db.users.find({age: {$gt: 30}}).toArray()表示查找年龄大于 30 的文档

db.<集合名称>.find({}, {<字段1>: 1, <字段2>: 1,...}).toArray()
如db.users.find({}, {_id: 0, name: 1, age: 1}).toArray()表示只显示 name 和 age 字段，不显示_id 字段。
    • 文档-更新
      方法功能描述示例
      updateOne()更新匹配的第一条文档db.users.updateOne({ name: "Alice" }, { $set: { age: 31 } })
      updateMany()更新所有匹配的文档db.users.updateMany({ city: "北京" }, { $inc: { age: 1 } })
      replaceOne()替换匹配的第一条文档（全文档替换）db.users.replaceOne({ name: "Bob" }, { name: "Bobby", age: 28 })
      db.<集合名称>.updateOne({<查询条件>}, {<更新操作>})
如db.users.updateOne({name: "Alice"}, {$set: {age: 31}})

db.<集合名称>.updateMany({<查询条件>}, {<更新操作>})
如db.users.updateMany({city: "New York"}, {$inc: {age: 1}})。
    • 文档-删除
      方法功能描述示例
      createIndex()创建索引（单字段、复合、地理空间等）db.users.createIndex({ name: 1 })（单字段升序）
      复合索引db.users.createIndex({ city: 1, age: -1 })
      getIndexes()查看集合所有索引db.users.getIndexes()
      dropIndex()删除指定索引db.users.dropIndex({ name: 1 })
      dropIndexes()删除所有索引（保留默认 _id 索引）db.users.dropIndexes()
      db.<集合名称>.deleteOne({<条件>})
如db.users.deleteOne({name: "Alice"})

db.<集合名称>.deleteMany({<条件>})
如db.users.deleteMany({city: "Los Angeles"})

      索引类型：

      • 单字段索引：{ key: 1 }
      • 复合索引：{ key1: 1, key2: -1 }（遵循最左前缀原则）
      • 文本索引：{ title: "text", content: "text" }（用于全文搜索）
      • 地理空间索引：{ location: "2dsphere" }（用于地理位置查询）
  • 运算符
    • 比较：$eq（等于）、$gt（大于）、$lt（小于）、$in（包含）例：db.users.find({ age: { $in: [20, 30, 40] } })
    • 逻辑：$and、$or、$not例：db.users.find({ $and: [{ age: { $gt: 25 } }, { city: "北京" }] })

    更新操作符：

    • $set：修改字段值（推荐，只改指定字段）
    • $inc：增减数值（如 $inc: { age: 1 } 年龄 + 1）
    • $unset：删除字段（如 $unset: { city: 1 }）
    • $push：向数组添加元素（如 $push: { hobbies: "reading" }）
  • 索引操作

    创建索引

    创建升序索引：db.<集合名称>.createIndex({<字段>: 1})，如db.users.createIndex({name: 1})。
创建降序索引：db.<集合名称>.createIndex({<字段>: -1})，如db.users.createIndex({age: -1})。
创建复合索引：db.<集合名称>.createIndex({<字段1>: 1, <字段2>: -1,...})。

    查看索引

    db.<集合名称>.getIndexes()，如db.users.getIndexes()
用于列出集合上已有的索引。

    删除索引

    db.<集合名称>.dropIndex({<字段>: 1})
如db.users.dropIndex({name: 1})，可移除不再需要的索引。
  • 聚合框架
    阶段功能描述示例场景
    $match筛选文档（类似 find）{ $match: { status: "已支付" } }
    $group分组统计（$sum、$avg 等累加器）{ $group: { _id: "$city", total: { $sum: "$amount" } } }
    $lookup关联其他集合（类似 SQL JOIN）{ $lookup: { from: "users", localField: "user_id", foreignField: "_id", as: "user" } }
    $sort排序{ $sort: { total: -1 } }
    $limit限制结果数量{ $limit: 10 }
    $project筛选 / 重命名字段{ $project: { _id: 0, name: 1, age: 1 } }（不显示 _id）
    $unwind拆分数组字段{ $unwind: "$tags" }（将 tags 数组拆分为多条记录）
    db.<集合名称>.aggregate([{<阶段1>}, {<阶段2>},...])
如db.users.aggregate([{$match: {city: "New York"}}, 
                      {$group: {_id: "$city", totalAge: {$sum: "$age"}}}, 
                      {$sort: {totalAge: -1}}])
可进行过滤、分组、排序等复杂分析任务。
  • 用户管理和权限控制

    创建用户

    命令：use admin；db.createUser({user: "<用户名>", pwd: "<密码>", roles: [{role: "<角色>", db: "<数据库>"},...]})。

    db.dropUser("<用户名>")。
  • 事物方法
    方法功能描述示例片段
    startSession()创建事务会话const session = db.getMongo().startSession()
    session.startTransaction()开启事务session.startTransaction()
    session.commitTransaction()提交事务（所有操作生效）session.commitTransaction()
    session.abortTransaction()回滚事务（所有操作撤销）session.abortTransaction()

    转账事务（扣减 A 的余额，增加 B 的余额）：

    const session = db.getMongo().startSession();
session.startTransaction();
try {
  db.users.updateOne({ _id: "A" }, { $inc: { balance: -100 } }, { session });
  db.users.updateOne({ _id: "B" }, { $inc: { balance: 100 } }, { session });
  session.commitTransaction(); // 成功则提交
} catch (e) {
  session.abortTransaction(); // 失败则回滚
} finally {
  session.endSession();
}
