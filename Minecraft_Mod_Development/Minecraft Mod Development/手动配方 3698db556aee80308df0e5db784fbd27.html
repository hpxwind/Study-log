手动配方

需要再data.tutorial_mod.recipes下放置模组文件

  • ice_ether.json 无序合成配方

    ice_ether_block → ice_ether*9

    {
  "type": "minecraft:crafting_shapeless",  // 合成类型
  "category": "redstone",                 // 配方分类
  "ingredients": [                        // 合成材料
    {
      "item": "tutorial_mod:ice_ether_block"
    }
  ],
  "result": {                             // 合成结果
    "count": 9,
    "item": "tutorial_mod:ice_ether"
  }
}
    1. type:材料在工作台里随便放，不需要固定形状 / 顺序, 有序合成是 minecraft:crafting_shaped
    1. category:配方在JEI/REI等合成查看器里显示的分类
      1. building 建筑
      1. redstone 红石
      1. equipment 装备
      1. misc 杂项
    1. ingredients:合成需要的物品
      1. item：物品 ID
      1. tutorial_mod: 是你的模组 ID
      1. ice_ether_block 是物品名
    1. result:合成结果
      1. item：输出物品 ID
      1. count：输出数量
  • ice_ether_block.json 有序合成配方

    ice_ether*9 → ice_ether_block

    {
  "type": "minecraft:crafting_shaped", // 有序合成，必须按指定格子摆放
  "category": "building",              // 合成界面归类为建筑类
  "group": "bark",                     // 物品分组，同类配方归类展示
  "key": {                             // 字符映射物品
    "#": {"item": "tutorial_mod:ice_ether"}
  },
  "pattern": [                         // 3×3工作台摆放样式，三行三列全填满
    "###",
    "###",
    "###"
  ],
  "result": {                          // 合成产出
    "count": 3,                        // 产出数量
    "item": "tutorial_mod:ice_ether_block" // 产出物品ID
  },
  "show_notification": true            // 解锁配方时弹出提示通知
}
    1. type:参数类型
    1. category:合成书 / 查询器分类标签
      1. building:建筑类, redstone 红石、equipment 装备、misc 杂项
    1. group:配方分组标识,同组配方会在合成界面折叠归类，仅界面展示作用，不影响合成逻辑
    1. key:符号与物品映射字典,自定义占位字符，绑定对应物品 ID
    1. pattern:3×3 工作台摆放模板,数组每行对应工作台一行格子,字符对应 key 绑定物品，空格代表空置格子,本例 9 格全部放置对应材料.
    1. result:合成输出结果
      1. item：产出物品唯一 ID
      1. count：单次合成产出数量
    1. show_notification:配方解锁提示
      1. true：玩家解锁配方弹出提示弹窗
      1. false：无解锁通知
  • ice_ether_from_smelting_raw_ice_ether.json 熔炉烧制配方

    raw_ice_ether → ice_ether

    {
  "type": "minecraft:blasting",
  "category": "misc",
  "cookingtime": 100,
  "experience": 0.7,
  "group": "ice_ether",
  "ingredient": {
    "item": "tutorial_mod:raw_ice_ether"
  },
  "result": "tutorial_mod:ice_ether"
}
    1. type:配方类型，blasting 代表高炉烧制配方
    1. category:合成界面分类，misc 归类为杂项
    1. cookingtime:烧制耗时，单位游戏刻，20 刻 = 1 秒，此处耗时 5 秒
    1. experience:烧制完成给予玩家的经验值
    1. group:配方分组标识，仅界面归类展示
    1. ingredient:烧制原材料，指定原始冰以太物品
    1. result:烧制成品物品 ID
  • ice_ether_ore.json 无序合成配方

    raw_ice_ether + stone → ice_ether_ore

    {
  "type": "minecraft:crafting_shapeless",
  "category": "misc",
  "ingredients": [
    {"item": "tutorial_mod:raw_ice_ether"},
    {"item": "minecraft:stone"}
  ],
  "result": {"item": "tutorial_mod:ice_ether_ore"}
}
    1. type:合成类型，crafting_shapeless 无序合成，材料摆放位置不限
    1. category:合成界面分类，misc 归属杂项类目
    1. ingredients:合成原材料列表，需粗冰以太 + 原石两种物品
    1. result:合成产出物品，默认数量 1，成品为冰以太矿石
