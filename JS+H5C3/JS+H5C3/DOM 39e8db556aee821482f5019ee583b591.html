1. checkValidity()	如果 input 元素中的数据是合法的返回 true，否则返回 false。
2. setCustomValidity()  设置 input 元素的 validationMessage 属性，用于自定义错误提示信息的方法。
	使用 setCustomValidity 设置了自定义提示后，validity.customError 就会变成 true，checkValidity 总是会返回 false。如果要重新判断需要取消自定义提示，方式如下：

setCustomValidity(null) 
setCustomValidity(undefined)

    • 通过标签名获取 / document.getElementsByTagName()

      通过 HTML 标签名（如 div、p）获取，返回HTMLCollection 集合（类数组，实时更新）。

      获取文档中所有指定标签
const divs = document.getElementsByTagName('div');

在父元素内获取（更高效）
const parent = document.getElementById('parent');
const childSpans = document.getElementsByTagName('span');
    • 通过元素的clas属性获取 / document/father.getElementByClassName()

      通过元素的 class 属性获取，返回HTMLCollection 集合（类数组）

      多个类名用空格分隔（如 'active menu'）

      文档中所有含指定类名的元素
const activeTtems = document.getElementsByClassName('active');

在父元素内获取
const list = document.getElementById('list');
const listItems = list.getElementsByClassName('item');
    • 通过CSS选择器获取 / document.querySelector() / document.querySelectorAll()
      1. 获取第一个匹配的元素。

      document.querySelector(selector)

      const firstItem = dcument.querySelector('.list .item'); // 第一个匹配.list .item 元素
const input = document.querySelector('input[type="text"]'); // 桉属性选择

      返回：单个元素（若不存在则返回 null）。

      选择器支持：所有 CSS 选择器（如 ID、类、标签、属性、伪类等）。

      1. 获取所有匹配的元素。

      document.querySelectorAll(selector)

      const allItems = document.querySelectorAll('.list .item'); // 所有匹配的元素

      返回：NodeList 集合（类数组，静态集合，DOM 变化时不会更新）。

      特点：支持复杂选择器，返回的集合是 “静态的”（性能更好）。

  • 操作节点方式
    <div id="box">
    <div id="child">11111111111</div>
</div>
<script>
    var odiv = document.createElement("div"); // 创建一个节点
    odiv.innerHTML = "new created"; // 给创建的节点添加内容
    odiv.className = "myClass"; // 添加类属性
    odiv.id = "myId"; // 添加id
    console.log(odiv);

    // appendChild(节点对象) 追加
    box.appendChild(odiv); // 给id = box追加一个孩子

    // insertBefore(要插入的节点， 在谁的前面插入) 任意位置插入
    box.insertBefore(odiv, child);

    // box.removeChild(节点对象) 删除
    box.removeChild(child);

    // replaceChild(新节点， 老节点) 替换
    var odiv2 = document.createElement("div");
    odiv2.innerHTML = "122222222";
    box.replaceChild(odiv2, odiv);

    // cloneNode(false/true) 是否克隆后代，默认false
    var ocloneBox = box.cloneNode(true);
    console.log(ocloneBox);
    ocloneBox.id = "box2";

    document.body.appendChild(ocloneBox);
</script>

    • 1、每种类型对应不同的内容（整数），通过nodeType可以查看（只读）
      节点类型nodeType 值描述示例
      文档节点（Document）9整个文档的根节点document 对象
      元素节点（Element）1HTML 标签（如 div、p 等）<div>...</div>
      文本节点（Text）3标签内的文本内容Hello World
      注释节点（Comment）8注释内容<!-- 这是注释 -->
      属性节点（Attr）2元素的属性（已逐渐被弃用，推荐通过元素节点操作）class="box"
      const div = document.querySelector('div');
const text = div.firstChild; // 文本节点
const comment = document.createComment('注释');

console.log(div.nodeType);    // 1（元素节点）

console.log(comment.nodeType);// 8（注释节点）
console.log(document.nodeType);// 9（文档节点）
