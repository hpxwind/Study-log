PyQuery

PyQuery 是一个强大的 Python HTML/XML 解析库，它采用类似 jQuery 的语法，让你能够轻松提取和操作网页数据。

pip install pyquery

基本初始化：

  • 从HTML字符串初始化

    直接传入字符串即可构建文档对象模型（DOM）

    from pyquery import PyQuery as pq

html = '''
<div id="container">
  <ul class="list">
    <li class="item-0">first item</li>
    <li class="item-1"><a href="link2.html">second item</a></li>
  </ul>
</div>
'''
doc = pq(html)
print(doc('#container .list li'))  # 使用CSS选择器
  • 从URL初始化

    PyQuery 能直接请求网址并初始化，非常方便

    doc = pq(url='https://example.com')
# 或者更常用的是结合requests库，便于控制请求过程
import requests
response = requests.get('https://example.com')
doc = pq(response.text)
  • 从本地文件初始化

    指定文件名即可读取并解析

    doc = pq(filename='path/to/your/file.html')

PyQuery 的核心是 CSS 选择器，用法与 jQuery 高度一致：

  • 基本选择器

    通过标签名、类名、ID 进行选择。

    doc('a')           # 选择所有 <a> 标签
doc('.title')      # 选择所有 class 为 "title" 的元素
doc('#main')       # 选择 id 为 "main" 的元素
doc('div.product > h2')  # 组合选择器
  • 伪类选择器

    用于精细定位，如选择特定序位的元素

    doc('li:first-child')   # 第一个 li 元素
doc('li:last-child')    # 最后一个 li 元素
doc('li:nth-child(2)')  # 第二个 li 元素

定位到一组元素后，经常需要进一步遍历或查找：

  • 遍历节点

    使用 .items()方法获取可遍历的 PyQuery 对象序列

    lis = doc('li').items()  # 返回一个生成器
for li in lis:
    print(li.text())     # 逐个输出 li 的文本
  • 查找子节点

    find()方法查找所有子孙节点，children()方法仅查找直接子节点

    items = doc('.list')
items.find('li')        # 查找 .list 下的所有 li 标签（任何层级）
items.children()         # 仅查找 .list 的直接子节点
items.children('.active') # 仅查找直接子节点中 class 为 active 的
  • 查找父节点与祖先节点

    parent()获取直接父节点，parents()获取所有祖先节点

    item = doc('.item-0.active')
item.parent()           # 直接父节点
item.parents('div')      # 所有是 div 标签的祖先节点
  • 查找兄弟节点

    siblings()方法获取所有兄弟节点

    item.siblings()          # 所有兄弟节点
item.siblings('.active') # 兄弟节点中 class 为 active 的

成功定位元素后，下一步是提取所需信息：

  • 获取属性

    使用 attr()方法或其属性形式

    a = doc('a')
href = a.attr('href')   # 获取第一个 a 标签的 href 属性
# 或
href = a.attr.href       # 效果同上

# 遍历获取多个元素的属性
for a in doc('a').items():
    print(a.attr('href'))

    注意：如果选择器匹配多个元素，直接调用 attr()仅返回第一个元素的属性。遍历 .items()是安全可靠的做法.

  • 获取文本

    使用 text()方法获取元素及其后代的文本内容（合并为一个字符串），使用 html()方法获取元素的内部 HTML 内容

    li_text = doc('li').text()  # 获取所有 li 的文本内容，合并成一个字符串
li_html = doc('li').html()  # 获取第一个 li 标签内部的 HTML 内容

进阶技巧与实战：

  • 链式调用

    PyQuery 支持链式操作，让代码更紧凑

    # 获取所有商品名称并去重
names = doc(".product-item .name").text().split()
unique_names = list(set(names))
  • 处理缺失元素

    在提取信息时，元素可能不存在，建议做判断.

    price_elem = item(".price")
price = price_elem.text() if price_elem else "N/A"  # 如果元素不存在，赋默认值
  • 结合XPath

    对于特别复杂的结构，可以转换为 lxml 对象使用 XPath.

    from lxml import etree
root = etree.HTML(html)
prices = root.xpath('//span[@class="price"]/text()')

案例：

  • 实战案例：提取商品信息
    <div class="product-item">
  <h2 class="name">无线蓝牙耳机</h2>
  <span class="price">¥199.00</span>
  <a href="/product/101" class="detail-link">查看详情</a>
</div>

    提取代码可以这样写：

    def extract_products(url):
    import requests
    from pyquery import PyQuery as pq
    
    response = requests.get(url)
    doc = pq(response.text)
    
    products = []
    for item in doc(".product-item").items():  # 遍历每个商品块
        name = item(".name").text()           # 提取商品名称
        price = item(".price").text()         # 提取价格
        link = item(".detail-link").attr("href")  # 提取链接
        
        products.append({
            "name": name,
            "price": price,
            "link": link
        })
    
    return products

常见问题：

  • 编码问题

    遇到乱码时，可显式指定编码

    response.encoding = "utf-8"  # 或根据实际情况指定 "gbk" 等
  • 反爬机制

    设置请求头模拟浏览器访问是基本策略

    headers = {
    "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36..."
}
response = requests.get(url, headers=headers)
