四、分步代码编写

1. 启动类（开启Mapper扫描）

  • Springboot03Application.java
    package com.hpc.mybatis;

import org.mybatis.spring.annotation.MapperScan;
import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
// 精准扫描Mapper接口所在包，避免误扫Service
@MapperScan(basePackages = "com.hpc.mybatis.mapper")
public class Springboot03Application {
    public static void main(String[] args) {
        SpringApplication.run(Springboot03Application.class, args);
    }
}

3. Mapper 接口 VipMapper.java

  • VipMapper.java
    package com.hpc.mybatis.mapper;

import com.hpc.mybatis.entity.Vip;
import java.util.List;

public interface VipMapper {
    // 查询全部数据
    List<Vip> selectAll();
}

4. Mapper 映射文件 resources/mapper/VipMapper.xml

  • resources/mapper/VipMapper.xml
    <?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE mapper PUBLIC "-//mybatis.org//DTD Mapper 3.0//EN"
        "<http://mybatis.org/dtd/mybatis-3-mapper.dtd>">
<!-- namespace 必须和Mapper接口全类名完全一致 -->
<mapper namespace="com.hpc.mybatis.mapper.VipMapper">
    <!-- id对应接口方法名，resultType写实体类 -->
    <select id="selectAll" resultType="Vip">
        select * from `user_msg`
    </select>
</mapper>

5. Service 层

VipService 接口

  • sercvice/VipService.java
    package com.hpc.mybatis.service;

import com.hpc.mybatis.entity.Vip;
import java.util.List;

public interface VipService {
    List<Vip> getAll();
}

VipServiceImpl 实现类

  • service/impl/VipServiceImpl.java
    package com.hpc.mybatis.service.impl;

import com.hpc.mybatis.entity.Vip;
import com.hpc.mybatis.mapper.VipMapper;
import com.hpc.mybatis.service.VipService;
import lombok.RequiredArgsConstructor;
import org.springframework.stereotype.Service;
import java.util.List;

@Service
@RequiredArgsConstructor
public class VipServiceImpl implements VipService {
    private final VipMapper mapper;

    @Override
    public List<Vip> getAll() {
        return mapper.selectAll();
    }
}

五、常见问题&避坑总结（你之前遇到的全部问题）

  1. Invalid bound statement (not found)
    • namespace 和 Mapper 类名不一致
    • mybatis.mapper-locations 路径配置错误
    • @MapperScan 扫描范围过大误扫Service
  1. Access denied 账号密码报错
    • 配置混用yml缩进+properties平铺格式，解析错乱
  1. Table 不存在
    • 连接不同MySQL实例、编译缓存残留，执行 mvn clean 清理
  1. 查询字段值为null
    • 实体属性名和数据库字段名不匹配，开启驼峰自动转换
