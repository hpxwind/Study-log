Vue-cli

Vue CLI（Vue 命令行工具）是 Vue.js 官方提供的脚手架工具，用于快速搭建 Vue 项目、配置开发环境、打包部署等，极大简化了 Vue 项目的开发流程。

  • 安装与卸载
    # npm 安装
npm install -g @vue/cli

# yarn 安装（需先安装 yarn：npm install -g yarn）
yarn global add @vue/cli

如果使用vue init webpack hello  
的化
需要安装全局插件（专门用于启动vue init）

    安装后验证：

    vue --version  # 或 vue -V（大写 V）

    卸载：

    npm uninstall -g @vue/cli
npm uninstall -g vue-cli
  • 创建新项目（vue create）

    交互式命令创建项目，可以自定义配置：

    vue create my-vue-project  # my-vue-project 为项目名

    选择配置方式：

    • Default ([Vue 3] babel, eslint)：默认配置（Vue 3 + Babel + ESLint）。
    • Default ([Vue 2] babel, eslint)：默认 Vue 2 配置。
    • Manually select features：手动选择功能（推荐自定义）。

    手动选择功能（常用选项）：

    • Babel：转译 ES6+ 语法到 ES5（兼容旧浏览器）。
    • TypeScript：支持 TypeScript 类型检查。
    • Router：Vue 路由（管理页面跳转）。
    • Vuex/Pinia：状态管理（Vuex 是旧版，Pinia 是 Vue 3 推荐）。
    • CSS Pre-processors：CSS 预处理器（Less/Sass/Stylus）。
    • Linter / Formatter：代码规范检查（ESLint）。
    • Unit Testing/E2E Testing：测试工具（Jest/Cypress 等）。

    后续配置：

    • 选择 Vue 版本（Vue 2 或 Vue 3）。
    • 是否使用历史模式路由（history mode，需后端配合）。
    • 选择 CSS 预处理器（如 Sass/SCSS）。
    • 选择 ESLint 规则（如 ESLint + Prettier）。
    • 配置文件存放位置（单独文件或 package.json 中）。
    • 是否保存配置为预设（方便下次复用）。

  • 启动开发服务（serve）
    cd my-vue-project
npm run serve  # 或 yarn serve

    默认地址：http://localhost:8080，修改代码会自动刷新页面

  • 打包生产环境（build）

    将项目打包为可部署的静态文件（HTML/CSS/JS）

    npm run build  # 或 yarn build

    打包后文件生成在 dist 目录，可直接部署到服务器（需注意路径配置）

  • 配置项目（vue.config.js）

    项目根目录下的 vue.config.js 是核心配置文件，用于自定义构建、开发服务器等选项，常用配置：

    module.exports = {
  publicPath: './',  // 部署路径（默认 '/'，子目录部署需改为 './'）
  outputDir: 'dist', // 打包输出目录
  devServer: {       // 开发服务器配置
    port: 8081,      // 端口号
    open: true,      // 自动打开浏览器
    proxy: {         // 接口代理（解决跨域）
      '/api': {
        target: 'http://localhost:3000', // 后端接口地址
        changeOrigin: true,
        pathRewrite: { '^/api': '' }
      }
    }
  },
  css: {             // CSS 配置
    loaderOptions: { // 预处理器配置（如 Sass 全局变量）
      sass: {
        additionalData: `@import "@/styles/variables.scss";`
      }
    }
  }
};
  • 安装插件（vue add）

    通过官方插件扩展项目功能（如 Vue Router、Vuex 等）

    vue add router  # 安装 Vue Router
vue add vuex    # 安装 Vuex（Vue 2）
vue add pinia   # 安装 Pinia（Vue 3）
  • 检查项目依赖（vue inspect）

    查看 Webpack 配置（Vue CLI 基于 Webpack 封装）

    vue inspect > webpack.config.js  # 将配置输出到文件
