表单组件渲染

在 Vue 中，表单组件绑定的核心是 v-model 指令，它实现了表单元素与组件数据之间的双向绑定：当表单元素的值变化时，绑定的数据会自动更新；反之，当数据变化时，表单元素的值也会同步更新。v-model 本质是语法糖，简化了 v-bind:value（绑定值）和 v-on:input（监听输入事件）的组合写法。

v-model的核心是根据元素类型确定绑定值的类型

  • 1、输入文本(input[type=”text”], textarea)
    • 绑定值类型：字符串（String）。
    • 原理：监听 input 事件，将输入框的 value 与数据同步。
    <template>
  <div>
    <!-- 单行文本 -->
    <input type="text" v-model="username" placeholder="请输入用户名">
    <p>用户名：{{ username }}</p>

    <!-- 多行文本（textarea） -->
    <textarea v-model="desc" placeholder="请输入描述"></textarea>
    <p>描述：{{ desc }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      username: '', // 初始为空字符串
      desc: ''
    }
  }
}
</script>

    注意：textarea 无需使用 v-text 或插值表达式，v-model 会自动同步内容。

  • 2、复选框(input[type=”checkbox”])
    • 单个复选框：绑定到布尔值（Boolean），表示 “选中” 状态。
    • 多个复选框组：绑定到数组（Array），数组元素为选中项的 value。

    单个复选框

    <template>
  <div>
    <input type="checkbox" v-model="isAgree" id="agree">
    <label for="agree">同意协议</label>
    <p>是否同意：{{ isAgree ? '是' : '否' }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      isAgree: false // 初始未选中
    }
  }
}
</script>

    多个复选框

    <template>
  <div>
    <p>选择爱好（可多选）：</p>
    <input type="checkbox" v-model="hobbies" value="reading" id="read">
    <label for="read">阅读</label>

    <input type="checkbox" v-model="hobbies" value="coding" id="code">
    <label for="code">编程</label>

    <input type="checkbox" v-model="hobbies" value="sports" id="sport">
    <label for="sport">运动</label>

    <p>选中的爱好：{{ hobbies }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      hobbies: [] // 初始为空数组，选中项的value会被添加到数组
    }
  }
}
</script>

    多个复选框必须绑定到同一个数组，且每个复选框需设置 value（表示选中时存入数组的值）。

  • 3、单选按钮(input[type=”radio”])
    • 绑定值类型：字符串 / 数值（String/Number），表示选中项的 value。
    • 用法：多个单选按钮绑定到同一个变量，通过 value 区分选项。
    <template>
  <div>
    <p>选择性别：</p>
    <input type="radio" v-model="gender" value="male" id="male">
    <label for="male">男</label>

    <input type="radio" v-model="gender" value="female" id="female">
    <label for="female">女</label>

    <p>选中的性别：{{ gender }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      gender: '' // 初始为空，选中后会被赋值为对应value
    }
  }
}
</script>
  • 4、下拉选择器(select+option)

    下拉选择器分单选和多选（添加 multiple 属性），绑定值类型不同：

    • 单选：绑定到字符串 / 数值（String/Number），值为选中项的 value。
    • 多选：绑定到数组（Array），数组元素为选中项的 value（按住 Ctrl 键多选）。

    单选下拉

    <template>
  <div>
    <select v-model="city">
      <option value="">请选择城市</option>
      <option value="beijing">北京</option>
      <option value="shanghai">上海</option>
      <option value="guangzhou">广州</option>
    </select>
    <p>选中的城市：{{ city }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      city: '' // 初始为空
    }
  }
}
</script>

    多选下拉(multiple)

    <template>
  <div>
    <select v-model="cities" multiple size="3">
      <option value="beijing">北京</option>
      <option value="shanghai">上海</option>
      <option value="guangzhou">广州</option>
      <option value="shenzhen">深圳</option>
    </select>
    <p>选中的城市：{{ cities }}</p>
    <p>提示：按住Ctrl键可多选</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      cities: [] // 初始为空数组
    }
  }
}
</script>

    size 属性用于设置下拉框可见选项数量，multiple 开启多选模式。

  • v-model修饰符

    1、.lazy：失去焦点后同步数据

    默认情况下，v-model 会在 input 事件触发时同步数据（实时更新）。.lazy 修饰符会将同步时机改为 change 事件（失去焦点或按下回车时）。

    <input type="text" v-model.lazy="username">
<p>用户名（失去焦点后更新）：{{ username }}</p>

    2、.number：自动转为数字类型

    默认情况下，表单输入的值会被当作字符串处理。.number 修饰符会自动将输入值转为数字（若无法转换，仍为字符串）。

    <input type="number" v-model.number="age">
<p>年龄（数字类型）：{{ age }}，类型：{{ typeof age }}</p>

    3、.trim：自动去除手尾空格

    .trim 修饰符会自动过滤输入值的首尾空格（中间空格保留）。

    <input type="text" v-model.trim="username">
<p>用户名（去空格后）："{{ username }}"</p>
  • 自定义表单组件v-model绑定

    在自定义组件中使用 v-model 时，需要明确值的传递和事件的触发规则：

    • 默认情况下，组件的 v-model 会将 value 作为 prop，将 input 作为事件（即 v-model 等价于 :value="xxx" @input="xxx = $event"）。
    • 可通过组件的 model 选项自定义 prop 名称和事件名称。
    <!-- 父组件 -->
<template>
  <div>
    <custom-checkbox v-model="isChecked"></custom-checkbox>
    <p>是否选中：{{ isChecked }}</p>
  </div>
</template>

<script>
import CustomCheckbox from './CustomCheckbox.vue';
export default {
  components: { CustomCheckbox },
  data() { return { isChecked: false }; }
}
</script>


<!-- 子组件 CustomCheckbox.vue -->
<template>
  <label>
    <!-- 绑定内部选中状态 -->
    <input 
      type="checkbox" 
      :checked="checked" 
      @change="$emit('change', $event.target.checked)"
    >
    自定义复选框
  </label>
</template>

<script>
export default {
  // 自定义v-model的prop和事件（默认是value和input）
  model: {
    prop: 'checked', // 自定义prop名称
    event: 'change'  // 自定义事件名称
  },
  props: {
    checked: { type: Boolean, default: false }
  }
}
</script>

    原理：

    • 父组件通过 v-model="isChecked" 绑定数据，等价于 :checked="isChecked" @change="isChecked = $event"。
    • 子组件接收 checked prop 作为选中状态，当复选框状态变化时，通过 $emit('change', 新值) 通知父组件更新数据。
  • 核心原理总结

    双向绑定本质：v-model 是 v-bind:value 和 v-on:input 的语法糖，例如：

    <!-- 等价于 -->
<input v-model="username">
<input :value="username" @input="username = $event.target.value">

    值类型适配：v-model 会根据表单元素类型自动适配绑定值的类型（字符串、布尔值、数组等），简化开发。

    灵活性：通过修饰符（.lazy/.number/.trim）可处理特殊场景，通过自定义组件的 model 选项可扩展 v-model 功能。

  • 注意事项
    • 复选框组忘记用数组：多个复选框必须绑定到数组，否则只会生效最后一个复选框。
    • 单选按钮未设置 value：多个单选按钮需通过 value 区分选项，否则无法正确绑定。
    • 自定义组件未触发事件：子组件需通过 $emit 触发对应事件（默认 input），否则父组件数据不会更新。
  • 购物车功能实现
    <!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Vue购物车</title>
  <script src="./node_modules/vue2/dist/vue.js"></script>
  <style>
    li {
      list-style: none;
    }
  </style>
</head>
<body>
  <div id="app" class="container">
    <h2>购物车</h2>
    <div>
      <ul>
        <!-- 全选功能 -->
        <div class="select-all">
          <input type="checkbox" v-model="isChecked" @change="handleSelectAll"> 全选
        </div>
        
        <!-- 商品列表 -->
        <li v-for="(item, index) in products" :key="item.name">
          <input 
            type="checkbox" 
            v-model="item.selected" 
            @change="handleSelectOne"
          >
          
          <div class="product-info">
            <span class="product-name">商品名称: {{item.name}}</span>
            <span class="product-price">价格: ¥{{item.price}}</span>
            <span style="margin-left: 10px; color: #7f8c8d;">(库存: {{item.count}})</span>
          </div>
          
          <!-- 数量控制 -->
          <div class="quantity-control">
            <button 
              class="quantity-btn"
              :disabled="item.currentNum === 0" 
              @click="handleDec(index)"
            >-</button>
            <span>{{item.currentNum}}</span>
            <button 
              class="quantity-btn"
              :disabled="item.currentNum >= item.count" 
              @click="handleAdd(index)"
            >+</button>
          </div>
          
          <!-- 删除按钮 -->
          <button class="delete-btn" @click="handleDelete(index)">删除</button>
        </li>
      </ul>
      
      <!-- 总金额 -->
      <div class="total-section">
        总金额: ¥{{sumValue.toFixed(2)}}
      </div>
      
      <!-- 购物车已选商品 -->
      <div class="cart-section">
        <h3>已选商品</h3>
        <div v-if="selectedProducts.length === 0" class="empty-cart">
          购物车为空
        </div>
        <div v-for="(item, index) in selectedProducts" :key="index" class="cart-item">
          {{item.name}} - 数量: {{item.currentNum}} - 小计: ¥{{(item.price * item.currentNum).toFixed(2)}}
        </div>
      </div>
    </div>
  </div>

  <script>
    new Vue({
      el: "#app",
      data: {
        products: [
          { name: "iphone", price: 4000, count: 7, currentNum: 0, selected: false },
          { name: "ipad", price: 5000, count: 4, currentNum: 0, selected: false },
          { name: "ipod", price: 2000, count: 8, currentNum: 0, selected: false },
          { name: "imac", price: 3000, count: 4, currentNum: 0, selected: false },
        ],
        isChecked: false, // 全选状态
      },
      computed: {
        // 计算总金额
        sumValue() {
          return this.products.reduce((pre, cur) => {
            // 只计算选中商品的金额
            return pre + (cur.selected ? cur.price * cur.currentNum : 0);
          }, 0);
        },
        // 获取已选中的商品
        selectedProducts() {
          return this.products.filter(item => item.selected && item.currentNum > 0);
        }
      },
      methods: {
        // 增加数量
        handleAdd(index) {
          this.products[index].currentNum++;
          // 数量大于0时自动选中该商品
          this.products[index].selected = true;
          // 检查是否需要更新全选状态
          this.checkAllSelected();
        },
        // 减少数量
        handleDec(index) {
          if (this.products[index].currentNum > 0) {
            this.products[index].currentNum--;
            // 如果数量变为0，取消选中状态
            if (this.products[index].currentNum === 0) {
              this.products[index].selected = false;
            }
            // 检查是否需要更新全选状态
            this.checkAllSelected();
          }
        },
        // 删除商品
        handleDelete(index) {
          this.products.splice(index, 1);
          // 检查是否需要更新全选状态
          this.checkAllSelected();
        },
        // 全选/取消全选
        handleSelectAll() {
          if(this.isChecked) {
            // 全选时, 没有数量的商品设置为1
            this.products.forEach(item => {
              if (item.currentNum ===0 && item.count > 0) {
                item.currentNum = 1;
              }
              item.selected = item.currentNum > 0;
            })
          } else {
            // 取消全选
            this.products.forEach(item => {
              item.selected = false;
            })
          }
        },
        // 单个商品选择变化时检查是否全选
        handleSelectOne() {
          this.checkAllSelected();
        },
        // 检查是否所有可选商品都被选中
        checkAllSelected() {
          // 过滤出有数量的商品
          const hasQuantity = this.products.filter(item => item.currentNum > 0);
          // 如果没有有数量的商品，全选框不勾选
          if (hasQuantity.length === 0) {
            this.isChecked = false;
            return;
          }
          // 检查所有有数量的商品是否都被选中
          this.isChecked = hasQuantity.every(item => item.selected);
        }
      }
    })
  </script>
</body>
</html>
