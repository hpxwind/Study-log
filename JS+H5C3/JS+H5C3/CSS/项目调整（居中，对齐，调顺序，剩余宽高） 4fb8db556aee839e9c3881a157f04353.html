项目调整（居中，对齐，调顺序，剩余宽高）

  • 居中实现方式
    方法 1：使用 Flexbox（推荐） 子元素
<style>
  .nav-item {
    display: flex;            /* 启用 Flex 布局 */
    justify-content: center;  /* 水平居中 */
    align-items: center;      /* 垂直居中 */
    height: 50px;             /* 设置高度，否则内容高度会压缩 */
    background-color: #f0f0f0;
  }
  .nav-item a {
    text-decoration: none;
    color: #333;
  }
</style>

<div class="nav-item">
  <a href="#">导航1</a>
</div>
    方法 2：使用 Grid（简洁方案）
<style>
  .nav-item {
    display: grid;            /* 启用 Grid 布局 */
    place-items: center;      /* 水平和垂直同时居中 */
    height: 50px;
    background-color: #f0f0f0;
  }
</style>

<div class="nav-item">
  <a href="#">导航1</a>
</div>
    方法 3：使用 line-height（仅适用于单行文本）
<style>
  .nav-item {
    height: 50px;             /* 设置容器高度 */
    line-height: 50px;        /* 行高等于容器高度 */
    text-align: center;       /* 水平居中 */
    background-color: #f0f0f0;
    white-space: nowrap;      /* 防止文字换行 */
  }
</style>

<div class="nav-item">
  <a href="#">导航1</a>
</div>
    方法 4：使用绝对定位 + transform
<style>
  .nav-item {
    position: relative;       /* 父元素设为相对定位 */
    height: 50px;
    background-color: #f0f0f0;
  }
  .nav-item a {
    position: absolute;       /* 绝对定位 */
    top: 50%;                 /* 顶部偏移50% */
    left: 50%;                /* 左侧偏移50% */
    transform: translate(-50%, -50%); /* 向上/左回退自身宽高的50% */
    text-decoration: none;
  }
</style>

    方法 4： table-cell （兼容性极好） 
.container {
  display: table;        /* 父级设为表格 */
  width: 100%;
  height: 100%;          /* 需要指定高度才能让垂直居中生效 */
}
.content {
  display: table-cell;
  vertical-align: middle;
  text-align: center;
}

1. 垂直居中：vertical-align: middle; 是实现垂直居中的关键。
在普通块级元素（如 div）中，vertical-align 通常无效，
但一旦元素变为 table-cell，该属性就会生效，使其内部内容（文本或内联元素）在垂直方向上居中。
2. 水平居中：text-align: center; 负责水平居中。
3. 依赖父容器：table-cell 元素必须位于一个 display: table; 
的父容器中才能完美工作。如果缺少父级 table 或 table-row，
浏览器会隐式生成匿名表格结构，但为了兼容性，通常需要配合如下结构：
  • 对齐方式
    主轴对齐（justify-content）
flex-start：项目靠主轴对齐。默认
flex-end：项目靠主轴终点对齐。
center：项目在主轴上居中对齐。
space-between：项目两端对齐，中间间距平均分配（首项靠起点，末项靠终点）。
space-around：项目两侧间距相等，相邻项目间距为两侧间距之和（整体左右有留白）。
space-evenly：项目之间及与容器边缘的间距完全相等。
    交叉轴对齐（align-items）
stretch（默认）：项目拉伸至填满交叉轴高度（需项目未设置固定高度）。
flex-start：项目靠交叉轴起点对齐。
flex-end：项目靠交叉轴终点对齐。
center：项目在交叉轴上居中对齐（垂直居中常用）。
baseline：项目沿基线对齐（如文字基线）。
    单个项目交叉轴对齐（align-self）
作用于 单个项目，覆盖容器的 align-items 设置。
stretch：项目拉伸至填满交叉轴高度（需项目未设置固定高度）。
flex-start：项目靠交叉轴起点对齐。
flex-end：项目靠交叉轴终点对齐。
center：项目在交叉轴上居中对齐（垂直居中常用）。
baseline：项目沿基线对齐（如文字基线）。
  • 调整顺序
    .sonDiv {
		order: 4; /*默认为0，如果多个order相同，按照html顺序，越大，越在后面*/
}
  • 剩余宽高
    .sonDiv {
	flex:1; /* 自动占满多余的空间 所有同级改值都相同的化，就是平均分 */
}
在flex布局下，如果不设置宽高，默认是以父元素为基准拉伸。
