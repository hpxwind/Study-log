CSS变量

CSS 变量（也称为 “自定义属性”）是 CSS 中用于存储和复用值的强大功能，允许你在样式表中定义一个值，然后在多个地方引用，大幅提升样式的可维护性和灵活性。

  • 定义CSS变量
    • 变量名必须以 - 开头（区分大小写，-color 和 -Color 是两个不同变量）。
    • 通常定义在 :root 伪类中（全局变量，作用于整个文档），也可定义在特定选择器中（局部变量，仅作用于该选择器及其子元素）。
    /* 全局变量（:root 等价于 <html>，但优先级更高） */
:root {
  --primary-color: #4285f4; /* 谷歌蓝 */
  --font-size: 16px;
  --spacing: 20px;
}

/* 局部变量（仅 .card 及其子元素可用） */
.card {
  --card-bg: #fff;
  --border-radius: 8px;
}
  • 使用CSS变量

    通过 var() 函数引用变量，语法：var(变量名, 默认值)（默认值可选，当变量未定义时使用）。

    body {
  color: var(--primary-color); /* 使用全局变量 */
  font-size: var(--font-size);
  padding: var(--spacing);
}

.card {
  background: var(--card-bg); /* 使用局部变量 */
  border-radius: var(--border-radius);
  margin-bottom: var(--spacing); /* 混合使用全局变量 */
}

/* 带默认值的引用（若 --unknown 未定义，则使用 #ccc） */
.box {
  border: 1px solid var(--unknown, #ccc);
}
  • 作用域与继承
    • 全局变量：定义在 :root 中，全文档可访问。
    • 局部变量：定义在特定选择器中，仅该选择器及其后代可访问。
    • 继承性：子元素会继承父元素的变量，若子元素有同名变量则覆盖。
    :root { --color: red; }
.parent { --color: green; }
.child { color: var(--color); } /* 继承父元素的 --color: green */

  • 动态性（可通过 JS 修改）

    CSS 变量是动态的，可通过 JavaScript 读取或修改，实现样式的实时更新（这是它相比预处理器变量的核心优势）。

    <div class="box"></div>

<script>
  // 读取变量
  const rootStyles = getComputedStyle(document.documentElement);
  const primaryColor = rootStyles.getPropertyValue('--primary-color');
  console.log(primaryColor); // 输出 #4285f4

  // 修改变量（会实时更新所有引用该变量的样式）
  document.documentElement.style.setProperty('--primary-color', '#ea4335'); // 改为谷歌红
</script>
  • 与其他CSS特性结合

    媒体查询

    ：在不同屏幕尺寸下修改变量值，实现响应式设计。

    :root { --font-size: 16px; }
@media (max-width: 768px) {
  :root { --font-size: 14px; } /* 小屏幕缩小字体 */
}

    伪类 / 伪元素

    ：在状态变化时（如 :hover）修改变量。

    .button {
  --btn-color: #4285f4;
  background: var(--btn-color);
}
.button:hover {
  --btn-color: #3367d6; /*  hover 时加深颜色 */
}

    计算属性（calc()）

    ：与变量结合实现动态计算。

    :root {
  --base-size: 10px;
  --double-size: calc(var(--base-size) * 2); /* 20px */
}
  • 案例：切换主题

    通过 JS 动态修改根元素变量，实现一键切换主题。

    /* 浅色主题（默认） */
:root {
  --bg-color: #fff;
  --text-color: #333;
}

/* 深色主题（通过 JS 给 html 加 class 激活） */
:root.dark {
  --bg-color: #333;
  --text-color: #fff;
}

body {
  background: var(--bg-color);
  color: var(--text-color);
  transition: all 0.3s; /* 平滑过渡 */
}
    // 切换主题按钮
document.getElementById('themeToggle').addEventListener('click', () => {
  document.documentElement.classList.toggle('dark');
});
  • 配合@property 增强类型（CSS Houdini）

    通过 @property 定义变量的类型、初始值和继承性，支持更复杂的动画和计算（现代浏览器支持）。

    /* 定义一个带类型的颜色变量 */
@property --primary-color {
  syntax: '<color>'; /* 变量类型为颜色 */
  initial-value: #4285f4;
  inherits: true; /* 允许继承 */
}

/* 可对该变量使用过渡动画 */
.box {
  background: var(--primary-color);
  transition: --primary-color 0.3s;
}
.box:hover {
  --primary-color: #ea4335; /* 颜色变化会平滑过渡 */
}
  • 兼容性与降级处理

    支持浏览器：Chrome 49+、Firefox 31+、Edge 15+、Safari 9.1+（IE 完全不支持）。

    降级方案：

    1. 对不支持的浏览器提供默认值：
    .element {
  color: #4285f4; /* IE 等不支持变量的浏览器使用该值 */
  color: var(--primary-color); /* 现代浏览器覆盖 */
}
    1. 通过 JS 检测支持性，动态加载降级样式：
    if (!window.CSS || !CSS.supports('--test', 0)) {
  // 不支持 CSS 变量，加载降级样式表
  const link = document.createElement('link');
  link.rel = 'stylesheet';
  link.href = 'fallback.css';
  document.head.appendChild(link);
}
