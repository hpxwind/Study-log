洛谷P2241 统计方形（数学方法）

  • 题目

    题目描述

    有一个 n×m 方格的棋盘，求其方格包含多少正方形、长方形（不包含正方形）。

    输入格式

    一行，两个正整数 n,m（n≤5000,m≤5000）。

    输出格式

    一行，两个正整数，分别表示方格包含多少正方形、长方形（不包含正方形）。

    输入输出样例

    输入 #1复制运行

    2 3

    输出 #1复制运行

    8 10
  • 公式推理

    1）正方形个数怎么算？

    在 n 行 m 列 的格子里：

    • 边长为 1 的正方形：横向能放 m 个，纵向能放 n 个→ n * m
    • 边长为 2 的正方形：横向剩 m-1 个位置，纵向剩 n-1 个位置→ (n-1)*(m-1)
    • 边长为 3 的正方形：→ (n-2)*(m-2)

    ……

    一直到边长 k = min(n,m)，不能再大了。

    所以总数：

    正方形总数 =
    n*m
    + (n-1)*(m-1)
    + (n-2)*(m-2)
    + ...
    + (n-k+1)*(m-k+1)

    2）所有矩形（包括正方形）公式

    一个矩形由 选两条横线 + 选两条竖线 确定。

    • 横线有 n+1 条选 2 条：C(n+1,2) = n*(n+1)/2
    • 竖线有 m+1 条选 2 条：C(m+1,2) = m*(m+1)/2

    所以：

    总矩形数 = [n(n+1)/2] * [m(m+1)/2]

    3）长方形（不含正方形）

    长方形 = 总矩形数 − 正方形数

import java.util.*;

public class Main {
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);
    int n = sc.nextInt();
    int m = sc.nextInt();

    long z = 0;
    long c = 0; 
    int len = Math.min(n, m);
    for (int i=1; i<=len; i++) {
      z += (long) (n -i + 1) * (m - i + 1);
    }

    long all = (long) n*(n+1)/2 * m*(m+1)/2;
    c = all - z;

    System.out.println(z + " " + c);
  }
}
