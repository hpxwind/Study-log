模板语法和指令

  • 文本插值（{{}}）

    用于将数据渲染到模板中，支持简单表达式：

    <!-- Vue2 和 Vue3 通用 -->
<template>
  <div>
    <!-- 直接渲染数据 -->
    <p>{{ message }}</p>
    
    <!-- 简单表达式（运算、三元等） -->
    <p>{{ count + 1 }}</p>
    <p>{{ isActive ? '激活' : '未激活' }}</p>
    
    <!-- 不支持流控制语句（if/for等） -->
  </div>
</template>
  • v-bind 绑定属性

    用于动态绑定 HTML 属性、组件 props 等，可简写为 :：

    <!-- Vue2 和 Vue3 通用 -->
<template>
  <!-- 绑定 HTML 属性 -->
  <img :src="imageUrl" :alt="imageAlt">
  
  <!-- 绑定 class（对象/数组语法） -->
  <div :class="{ active: isActive, 'text-danger': hasError }"></div>
  <div :class="[activeClass, errorClass]"></div>
  
  <!-- 绑定 style（对象/数组语法） -->
  <div :style="{ color: textColor, fontSize: '16px' }"></div>
  <div :style="[baseStyles, overrideStyles]"></div>
  
  <!-- 绑定 props 到子组件 -->
  <ChildComponent :user="currentUser" :is-admin="isAdmin" />
</template>
  • v-on 绑定事件

    用于绑定 DOM 事件或组件自定义事件，可简写为 @，支持事件修饰符

    <!-- Vue2 和 Vue3 通用 -->
<template>
  <!-- 绑定 DOM 事件 -->
  <button @click="handleClick">点击</button>
  <input @input="handleInput" @keyup.enter="handleEnter">
  
  <!-- 绑定自定义事件（子组件触发） -->
  <ChildComponent @custom-event="handleCustomEvent" />
  
  <!-- 事件修饰符（.stop/.prevent/.self等） -->
  <button @click.stop="handleClick">阻止冒泡</button>
  <form @submit.prevent="handleSubmit">阻止默认提交</form>
</template>
  • v-for 循环

    用于遍历数组 / 对象渲染列表，基础用法一致

    <!-- Vue2 和 Vue3 通用 -->
<template>
  <!-- 遍历数组 -->
  <ul>
    <li v-for="(item, index) in list" :key="index">
      {{ index }}: {{ item.name }}
    </li>
  </ul>
  
  <!-- 遍历对象 -->
  <div v-for="(value, key, index) in user" :key="key">
    {{ index }}. {{ key }}: {{ value }}
  </div>
</template>
  • vue3新增/调整的模板语法

    Vue3 增强内容：

    多根节点（Fragments）消除冗余包裹。

    v-model 语法统一，支持多双向绑定。

    移除 v-on.native，事件处理更直观。

    支持可选链、空值合并等现代语法。

    v-if 与 v-for 优先级调整，更符合逻辑。

    • 1、支持多根节点

      Vue2 要求模板必须有唯一根节点（否则会报错）；Vue3 支持多根节点（无需额外包裹 div）：

      <!-- Vue2（错误：必须有唯一根节点） -->
<template>
  <p>节点1</p>
  <p>节点2</p> <!-- 报错：Multiple root elements found in template -->
</template>

<!-- Vue2（正确：用 div 包裹） -->
<template>
  <div>
    <p>节点1</p>
    <p>节点2</p>
  </div>
</template>

<!-- Vue3（正确：支持多根节点） -->
<template>
  <p>节点1</p>
  <p>节点2</p>
</template>
    • 2、v-model 语法调整

      Vue2 中 v-model 与 .sync 修饰符并存，容易混淆；Vue3 统一了语法

      场景Vue2 用法Vue3 用法
      原生表单元素<input v-model="value">完全一致（语法不变）
      自定义组件双向绑定<Child :value.sync="parentValue"> 或 <Child v-model="parentValue">（需子组件配合）<Child v-model:value="parentValue">（统一用 v-model:prop 形式）

      自定义组件实例：

      <!-- Vue2：.sync 修饰符 -->
<ChildComponent :title.sync="pageTitle" />
<!-- 等价于：<ChildComponent :title="pageTitle" @update:title="pageTitle = $event" /> -->

<!-- Vue3：统一用 v-model:prop -->
<ChildComponent v-model:title="pageTitle" />
<!-- 等价于：<ChildComponent :title="pageTitle" @update:title="pageTitle = $event" /> -->

      Vue3 中，v-model 可同时绑定多个属性（多双向绑定）：

      <!-- Vue3 支持多 v-model -->
<FormComponent 
  v-model:username="user.name" 
  v-model:email="user.email" 
/>
    • 3、v-if 与 v-for 优先级变化
      • Vue2：v-for 优先级高于 v-if（遍历后再判断）。
      • Vue3：v-if 优先级高于 v-for（先判断再遍历，更符合逻辑）。

      注意： 不建议在同一元素上同时使用 v-if 和 v-for（可读性差），Vue3 会报警告。推荐在父元素用 v-if 控制，或用计算属性过滤列表。

    • 4、移除v-on.native 修饰符

      Vue2 中，给自定义组件绑定原生事件需用 .native 修饰符；Vue3 中，自定义组件的事件默认会被当作原生事件处理，若需监听组件内部的自定义事件，由子组件通过 emits 声明：

      <!-- Vue2：监听自定义组件的原生事件 -->
<ChildComponent @click.native="handleClick" />

<!-- Vue3：无需 .native，默认监听原生事件 -->
<ChildComponent @click="handleClick" />

<!-- Vue3：子组件声明自定义事件后，父组件可监听 -->
<!-- 子组件 -->
<script>
export default {
  emits: ['custom-event'] // 声明自定义事件
}
</script>
<!-- 父组件 -->
<ChildComponent @custom-event="handleCustom" />
    • 5、支持动态指令参数的复杂表达式

      Vue2 中动态指令参数（如 v-bind:[key]）只能是简单表达式；Vue3 支持更复杂的表达式：

      <!-- Vue3 支持 -->
<template>
  <!-- 动态参数可包含运算 -->
  <button @[`click${postfix}`]="handleClick">点击</button>
  
  <!-- 动态参数可调用方法 -->
  <div :[getAttrName()]="value"></div>
</template>

<script>
export default {
  data() {
    return { postfix: 'Me', value: 'test' }
  },
  methods: {
    getAttrName() {
      return 'data-id';
    }
  }
}
</script>
    • 6、支持模板中的可选链（?.）和空值合并（??）

      Vue3 模板中默认支持 JavaScript 的可选链（?.）和空值合并（??）运算符，无需额外配置：

      <!-- Vue3 支持 -->
<template>
  <!-- 可选链：安全访问嵌套属性 -->
  <p>{{ user?.address?.city }}</p>
  
  <!-- 空值合并：默认值只在左侧为 null/undefined 时生效 -->
  <p>{{ username ?? '未知用户' }}</p>
</template>
    • v-for支持解构

      Vue3 中 v-for 可直接解构遍历的对象 / 数组，简化代码：

      <!-- Vue3 支持 -->
<template>
  <div v-for="({ name, age }, index) in users" :key="index">
    {{ name }} - {{ age }}
  </div>
</template>

Vue 2 和 Vue 3 中的指令（Directive）是带有 v- 前缀的特殊属性，用于在模板中操作 DOM、绑定数据或监听事件

  • 指令

    以 v- 开头的特殊属性，用于控制 DOM 行为：

    指令作用Vue2/Vue3 一致性
    v-text渲染文本（类似 {{ }}）完全一致
    v-html渲染 HTML（注意 XSS 风险）完全一致
    v-show基于条件显示元素（CSS 切换）完全一致
    v-if/v-else/v-else-if基于条件渲染元素（DOM 增删）完全一致
    v-for循环渲染列表基础用法一致，优先级有差异（见下文）
    v-bind绑定属性（可简写为 :）完全一致
    v-on绑定事件（可简写为 @）基础用法一致，修饰符有调整（见下文）
    v-model双向数据绑定基础用法一致，自定义组件用法有调整
    v-pre跳过编译（直接显示原始内容）完全一致
    v-cloak防止未编译模板闪烁完全一致
    v-once只渲染一次（后续不更新）完全一致
  • 自定义指令（2/3通用）

    作用：封装 DOM 操作逻辑，让模板更简洁，逻辑更复用。

    使用场景：聚焦输入框、格式化日期、权限控制、滚动监听、拖拽等。

    形式：以 v- 为前缀的自定义属性（如 v-focus、v-permission）。

    Vue2:

    // main.js
import Vue from 'vue';
import App from './App.vue';

// 注册全局指令 v-focus
Vue.directive('focus', {
  // 元素插入 DOM 时自动聚焦
  inserted(el, binding) {
    // 根据绑定值决定是否聚焦（如 v-focus="true" 才聚焦）
    if (binding.value) {
      el.focus();
    }
  },
  // 当指令绑定值更新时
  update(el, binding) {
    if (binding.value && !binding.oldValue) {
      el.focus(); // 从 false 变为 true 时聚焦
    } else if (!binding.value && binding.oldValue) {
      el.blur(); // 从 true 变为 false 时失焦
    }
  }
});

new Vue({
  render: h => h(App)
}).$mount('#app');

    局部注册：

    <!-- MyComponent.vue -->
<template>
  <input v-focus="isFocused" type="text">
  <button @click="isFocused = !isFocused">切换聚焦</button>
</template>

<script>
export default {
  data() {
    return { isFocused: true };
  },
  // 局部注册指令
  directives: {
    focus: {
      inserted(el, binding) {
        if (binding.value) el.focus();
      },
      update(el, binding) {
        if (binding.value) el.focus();
        else el.blur();
      }
    }
  }
};
</script>

    使用：

    <!-- 在模板中使用 -->
<input v-focus="true" type="text"> <!-- 初始聚焦 -->
<input v-focus="false" type="text"> <!-- 初始不聚焦 -->

    Vue3:

    // main.js
import { createApp } from 'vue';
import App from './App.vue';

const app = createApp(App);

// 注册全局指令 v-focus
app.directive('focus', {
  // 元素插入 DOM 时（替代 Vue 2 的 inserted）
  mounted(el, binding) {
    if (binding.value) el.focus();
  },
  // 元素更新时（替代 Vue 2 的 update + componentUpdated）
  updated(el, binding) {
    if (binding.value !== binding.oldValue) {
      binding.value ? el.focus() : el.blur();
    }
  }
});

app.mount('#app');

    局部注册：

    <!-- MyComponent.vue -->
<template>
  <input v-focus="isFocused" type="text">
  <button @click="isFocused = !isFocused">切换聚焦</button>
</template>

<script setup>
import { ref } from 'vue';

const isFocused = ref(true);

// 局部注册指令（在 <script setup> 中需用 defineDirective）
const vFocus = {
  mounted(el, binding) {
    if (binding.value) el.focus();
  },
  updated(el, binding) {
    if (binding.value !== binding.oldValue) {
      binding.value ? el.focus() : el.blur();
    }
  }
};
</script>

    简化写法：（仅需一个钩子）

    如果指令逻辑简单，可直接传递一个函数（在 mounted 和 updated 时触发）：

    // Vue 3 简化版 v-focus
app.directive('focus', (el, binding) => {
  binding.value ? el.focus() : el.blur();
});
