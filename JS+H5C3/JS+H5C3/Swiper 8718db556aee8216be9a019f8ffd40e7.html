Swiper

Swiper 是一款 强大的移动端 / PC 端滑动组件库，支持轮播图、滑动列表、Tab 切换、手势操作等多种场景，兼容性好（支持 IE9+ 及主流浏览器），且提供丰富的配置项和扩展插件，是前端开发中实现滑动交互的首选工具。

官方中文网站：Swiper使用方法 - Swiper中文网

  • 1、核心结构（3层嵌套）
    <!-- 1. 最外层容器：Swiper 根容器（自定义类名，需初始化时指定） -->
<div class="my-swiper">
  <!-- 2. 滑块容器：固定类名 .swiper-wrapper -->
  <div class="swiper-wrapper">
    <!-- 3. 滑块单元：固定类名 .swiper-slide（每个滑块对应一个轮播项） -->
    <div class="swiper-slide">滑块1</div>
    <div class="swiper-slide">滑块2</div>
    <div class="swiper-slide">滑块3</div>
  </div>
</div>

    注意：.swiper-wrapper 和 .swiper-slide 是 Swiper 内置的固定类名，不可修改；根容器（如 .my-swiper）可自定义，初始化时需通过选择器指向它。

  • 4、初始化Swiper实例
    <script>        
  var mySwiper = new Swiper ('.swiper', {
    direction: 'vertical', // 垂直切换选项
    loop: true, // 循环模式选项
    
    // 如果需要分页器
    pagination: {
      el: '.swiper-pagination',
    },
    
    // 如果需要前进后退按钮
    navigation: {
      nextEl: '.swiper-button-next',
      prevEl: '.swiper-button-prev',
    },
    
    // 如果需要滚动条
    scrollbar: {
      el: '.swiper-scrollbar',
    },
  })        
  </script>
    // 等待 DOM 加载完成（确保 Swiper 容器已存在）
document.addEventListener('DOMContentLoaded', function() {
  // 创建 Swiper 实例
  const mySwiper = new Swiper('.my-swiper', {
    // 1. 基础配置
    direction: 'horizontal', // 滑动方向：'horizontal'（水平，默认）/ 'vertical'（垂直）
    loop: true, // 是否循环播放（滑动到最后一张后，继续滑动会回到第一张）
    speed: 500, // 滑动动画时长（毫秒，默认 300）

    // 2. 自动播放配置（需启用 autoplay 模块）
    autoplay: {
      delay: 3000, // 自动播放间隔（毫秒，默认 3000）
      stopOnLastSlide: false, // 循环模式下无效，非循环模式下是否停止在最后一张
      disableOnInteraction: true, // 用户交互（如点击、滑动）后是否暂停自动播放（默认 true）
    },

    // 3. 分页器配置（需启用 pagination 模块）
    pagination: {
      el: '.swiper-pagination', // 分页器容器的选择器（对应 HTML 中的 .swiper-pagination）
      clickable: true, // 是否允许点击分页器切换滑块（默认 false）
      type: 'bullets', // 分页器类型：'bullets'（圆点，默认）/ 'fraction'（数字，如 1/3）/ 'progressbar'（进度条）
    },

    // 4. 导航按钮配置（需启用 navigation 模块）
    navigation: {
      nextEl: '.swiper-button-next', // 下一张按钮的选择器
      prevEl: '.swiper-button-prev', // 上一张按钮的选择器
    },
  });
});

  • 轮播图场景
    配置项类型说明
    loopboolean是否循环播放（默认 false）
    autoplayobject自动播放配置（delay 为间隔时间，disableOnInteraction 控制交互暂停）
    paginationobject分页器配置（el 指向容器，clickable 允许点击切换）
    navigationobject导航按钮配置（nextEl/prevEl 指向按钮）

    1、多滑块场景（一次显示多个）

    new Swiper('.my-swiper', {
  slidesPerView: 3, // 一次显示的滑块数量（默认 1，可设为小数如 2.5）
  spaceBetween: 20, // 滑块之间的间距（单位 px，默认 0）
  loop: true,
  // 响应式配置：不同屏幕宽度显示不同数量的滑块
  breakpoints: {
    768: { slidesPerView: 2 }, // 屏幕宽度 ≤ 768px 时，一次显示 2 个
    480: { slidesPerView: 1 }, // 屏幕宽度 ≤ 480px 时，一次显示 1 个
  }
});

    2、垂直滑动场景（垂直轮播，滑动列表）

    new Swiper('.my-swiper', {
  direction: 'vertical', // 垂直滑动（默认 horizontal）
  slidesPerView: 'auto', // 自动适应滑块高度（显示完整滑块）
  freeMode: true, // 自由滑动（松开手指后不自动对齐到滑块，类似滚动列表）
  scrollbar: { // 显示滚动条
    el: '.swiper-scrollbar',
    draggable: true, // 允许拖动滚动条
  }
});

    3、手势控制场景（放缩，触摸）

    new Swiper('.my-swiper', {
  zoom: {
    maxRatio: 2, // 最大缩放比例（默认 3）
    minRatio: 1, // 最小缩放比例（默认 1）
  },
  touchRatio: 1, // 触摸灵敏度（默认 1，值越大越灵敏）
  preventClicks: true, // 防止滑块内点击事件被触发（默认 true）
});
  • Swiper事件与方法（交互控制）

    1、监听状态

    通过 on 方法绑定事件，监听滑块的滑动、切换等行为：

    const mySwiper = new Swiper('.my-swiper', { loop: true });

// 1. 滑块切换结束时触发（slideChange 是事件名）
mySwiper.on('slideChange', function() {
  console.log('当前滑块索引：', this.realIndex); // this 指向 Swiper 实例
});

// 2. 滑动开始时触发
mySwiper.on('touchStart', function() {
  console.log('开始滑动');
});

// 3. 自动播放暂停时触发
mySwiper.on('autoplayPause', function() {
  console.log('自动播放已暂停');
});

    2、常用方法（主动控制）

    通过 Swiper 实例调用方法，主动控制滑块（如切换到指定滑块、暂停自动播放）：

    const mySwiper = new Swiper('.my-swiper', { autoplay: { delay: 3000 }, loop: true });

// 1. 暂停自动播放
document.querySelector('#pause-btn').addEventListener('click', function() {
  mySwiper.autoplay.pause();
});

// 2. 恢复自动播放
document.querySelector('#play-btn').addEventListener('click', function() {
  mySwiper.autoplay.start();
});

// 3. 切换到指定滑块（index 为索引，1 表示第 2 张，loop 模式下有效）
document.querySelector('#go-to-2').addEventListener('click', function() {
  mySwiper.slideTo(1, 1000); // 第二个参数是动画时长（毫秒）
});

// 4. 切换到下一张
document.querySelector('#next-btn').addEventListener('click', function() {
  mySwiper.slideNext();
});
  • Swiper扩展插件（高级功能）

    Swiper 提供多个官方扩展插件，需额外引入或启用（部分已包含在 swiper-bundle.min.js 中）：

    插件名称功能说明
    Autoplay自动播放（基础插件，bundle 已包含）
    Pagination分页器（基础插件，bundle 已包含）
    Navigation导航按钮（基础插件，bundle 已包含）
    Scrollbar滚动条（基础插件，bundle 已包含）
    Zoom图片缩放（支持双击 / 手势缩放，bundle 已包含）
    Lazy图片懒加载（延迟加载未显示的滑块图片，优化性能）
    EffectFade淡入淡出切换效果（替代默认的滑动效果）
    Parallax视差效果（滑块内元素随滑动产生不同速度的位移，增强视觉体验）

    淡入淡出效果：

    new Swiper('.my-swiper', {
  effect: 'fade', // 启用淡入淡出效果（默认 'slide' 滑动效果）
  fadeEffect: {
    crossFade: true, // 交叉淡入淡出（前后滑块同时显示过渡，默认 false）
  },
  autoplay: { delay: 3000 },
  pagination: { el: '.swiper-pagination', clickable: true },
});
  • 兼容性与优化

    兼容性

    • 浏览器支持：Swiper 11+ 支持 IE11+、Chrome、Firefox、Safari 及主流移动端浏览器；若需支持 IE9/10，需使用 Swiper 6 及以下版本。
    • 移动端适配：建议在 head 中添加视口配置，确保滑动体验正常：
      <meta name="viewport" content="width=device-width, initial-scale=1, minimum-scale=1">

    2. 性能优化

    • 图片懒加载：启用 Lazy 插件，避免一次性加载所有图片：
      <!-- HTML：图片 src 改为 data-src，添加 .swiper-lazy 类 -->
<img class="swiper-lazy" data-src="https://picsum.photos/800/400?1" alt="轮播图1"><!-- 加载占位图（可选） -->
<div class="swiper-lazy-preloader"></div>
      // JS：启用 Lazy 插件
new Swiper('.my-swiper', {
  lazy: {
    loadPrevNext: true, // 预加载前后一张图片
  },
});
    • 减少滑块数量：若轮播图滑块过多（如 100+），建议使用 loop: false 并配合分页器，避免 DOM 元素过多导致性能问题。
