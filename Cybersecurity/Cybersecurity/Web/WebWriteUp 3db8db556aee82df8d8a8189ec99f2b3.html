    import requests
url = "https://example.com"
headers = {
    "X-Forwarded-For": "127.0.0.1"  # 伪造的IP
}
response = requests.get(url, headers=headers)
print(response.text)

    3、使用curl命令

    Windows
curl -s -H "X-Forwarded-For: 1.2.3.4" https://example.com | findstr "target_string"

Linux/mac
# 发送请求并查找包含 "target_string" 的内容
curl -s -H "X-Forwarded-For: 1.2.3.4" https://example.com | grep "target_string"

    为什么可以通过. 和 ../修改路径？

    当路径中 .. 的数量超过可回溯的目录层级时，路径会尝试访问 根目录之上 的位置（如 Linux 的 / 或 Windows 的 C:\），这是文件系统不允许的。

    大多数操作系统会忽略多余的 .. ，最终停留在根目录（如返回 /etc 或 C:\Windows），但某些严格校验的API会直接报错。

      • MIME

        MIME：多用途互联网邮件扩展类型，是设定某种扩展名的文件用一种应用程序来打开的方式类型，当该扩展文件名被访问的时候，浏览器会自动使用指定应用程序来打开，多用于指定一些客户端自定义问文件名，以及一些媒体打开方式。

        每个MIME类型由两部分组成，前面是数据的大类别，例如生音audio，图像image等，后面定义具体类型。

        常见MIME类型：

        超文本标记语言文本 .html/ .html text/html

        普通文本 .txt text/plain

        RTF文本 .rtf application/rtf

        GIF图形 .gif image/gif

        JPEG .ipeg .jpg image/jpeg

        $_FILES() 可以从客户端计算机向远程服务器上传文件

        第一个参数是表单的 input name

        第二个可以是 name, type, size, tmp_name 或 error

        $_FILES['file']['name'] # 被上传文件的名称
$_FILES['file']['type'] # 被上传文件的类型
$_FILES['file']['size'] # 被上传文件的大小（字节）
$_FILES['file']['tmp_name'] # 存储在服务器的文件的临时副本名称
$_FILES['file']['error'] # 由文件上传导致的错误代码
