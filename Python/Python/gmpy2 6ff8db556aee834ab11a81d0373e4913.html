gmpy2

用于高精度数论计算的核心库，基于 GMP（GNU Multiple Precision Arithmetic Library）、MPFR、MPC 等底层高性能库封装，专为大整数、高精度浮点数、数论运算（如模逆、素性检测、幂模运算）设计

  • 高精度整数运算

    gmpy2 的 mpz 类型是高性能大整数类型，兼容 Python 原生 int，但运算速度更快：

    import gmpy2

# 1. 创建 mpz 类型（大整数）
a = gmpy2.mpz("123456789012345678901234567890")  # 字符串转大整数
b = gmpy2.mpz(98765432109876543210)               # 原生 int 转 mpz

# 2. 基本运算（+、-、*、/、% 等，与原生一致）
print(a + b)  # 高精度加法
print(a * b)  # 大整数乘法（无溢出）
print(a % b)  # 模运算

# 3. 幂运算（比 pow 更快）
print(gmpy2.powmod(a, 100, b))  # (a^100) mod b（高效幂模，RSA 核心）
  • 逆元计算（gmpy2.invert）

    gmpy2.invert(a, m) 是 RSA 加密中计算私钥 d 的核心函数，等价于求解 a*x ≡ 1 (mod m)，仅当 a 和 m 互质时存在逆元。

    import gmpy2

# 你的 RSA 代码场景（计算私钥 d）
p = 1325465431
q = 152317153
e = 65537
phi_n = (p-1) * (q-1)

# 计算 e 在模 phi_n 下的逆元（即 RSA 私钥 d）
d = gmpy2.invert(e, phi_n)
print(f"私钥 d = {d}")  # 输出：大整数结果

# 验证：(e*d) mod phi_n == 1
print(gmpy2.powmod(e, d, phi_n))  # 输出 1，验证成功

    异常处理：若 a 和 m 不互质，gmpy2.invert 会抛出 ZeroDivisionError，需提前判断：

    if gmpy2.gcd(e, phi_n) == 1:
    d = gmpy2.invert(e, phi_n)
else:
    raise ValueError("e 和 phi_n 不互质，无逆元")

  • 素性检测（gmpy2.is_prime/gmpy2.next_prime）

    密码学中生成 RSA 质数 p/q 的核心功能，比 Python 原生素性检测更精准、更快：

    import gmpy2

# 1. 素性检测（Miller-Rabin 算法，默认 25 轮检测，越高越精准）
p = 1325465431
print(gmpy2.is_prime(p))  # 输出 True/False（判断是否为质数）
# 自定义检测轮数（CTF 常用 64 轮，平衡速度与精度）
print(gmpy2.is_prime(p, 64))

# 2. 生成下一个质数（RSA 密钥生成必备）
start = 1000000000000  # 起始数
next_p = gmpy2.next_prime(start)
print(f"大于 {start} 的最小质数：{next_p}")

# 3. 生成指定位数的随机质数（密码学标准）
# 生成 1024 位随机质数（RSA 推荐 2048/4096 位）
random_prime = gmpy2.mpz_urandomb(gmpy2.random_state(), 1024)  # 生成 1024 位随机数
random_prime = gmpy2.next_prime(random_prime)  # 转为质数
print(f"1024 位质数：{random_prime}")
  • 高精度浮点数（mpfr 类型）

    突破 Python 原生 float 的精度限制（原生仅 64 位），支持自定义精度：

    import gmpy2

# 1. 创建 mpfr 类型（指定精度，单位：比特）
pi = gmpy2.const_pi(1000)  # 1000 位精度的 π
print(f"π（1000 位精度）：{pi}")

# 2. 自定义精度运算
gmpy2.set_context(gmpy2.context(precision=200))  # 全局精度设为 200 位
a = gmpy2.mpfr("1.23456789012345678901234567890")
b = gmpy2.mpfr("9.87654321098765432109876543210")
print(a / b)  # 高精度除法（无精度丢失）
  • 数论中的方法
    函数功能说明示例
    gmpy2.gcd(a, b)计算最大公约数gmpy2.gcd(12, 18) → 6
    gmpy2.lcm(a, b)计算最小公倍数gmpy2.lcm(12, 18) → 36
    gmpy2.factorial(n)计算 n!（阶乘），支持超大 ngmpy2.factorial(1000) → 大整数
    gmpy2.legendre(a, p)勒让德符号（判断 a 是否为 p 的二次剩余）gmpy2.legendre(5, 7) → -1
    gmpy2.powmod(a, b, m)高效幂模运算 (a^b) mod m（比 pow 快）gmpy2.powmod(2, 100, 7) → 2
    gmpy2.is_square(n)判断 n 是否为完全平方数gmpy2.is_square(16) → True
    gmpy2.sqrt(n)高精度平方根（返回 mpfr/mpz 类型）gmpy2.sqrt(2, precision=100) → 高精度√2
