BeautifulSoup

BeautifulSoup（通常简称为bs）是Python中一个功能强大且易于使用的库，主要用于从HTML或XML文档中提取数据。

  • 安装

    使用前需要先安装BeautifulSoup库，推荐同时安装一个高性能解析器如lxml：

    pip install beautifulsoup4 lxml

  • 初始化BeautifulSoup对象
    from bs4 import BeautifulSoup
import requests

# 从网络获取HTML
resp = requests.get('https://example.com')
soup = BeautifulSoup(resp.text, 'lxml') # 推荐使用lxml解析器

# 从本地文件或字符串获取
# soup = BeautifulSoup(open("index.html", encoding="utf-8"), 'lxml')
# soup = BeautifulSoup('<html>data</html>', 'lxml')

    🧬 理解四种核心对象

    BeautifulSoup将HTML文档转换为树形结构，主要由四种对象组成：

    1. BeautifulSoup：代表整个文档树，是一个特殊的Tag对象，可当作根节点使用
    1. Tag（标签）：对应HTML中的标签（如<div>、<a>），是操作最多的对象
    1. NavigableString：标签内的文本字符串，不包含标签本身
    1. Comment：一种特殊的NavigableString，表示HTML注释
    # 获取标签对象
tag = soup.a  # 获取文档中第一个<a>标签

# 获取标签名称
print(tag.name)  # 输出: 'a'

# 获取标签属性
print(tag['href'])  # 直接通过属性名获取
print(tag.get('href'))  # 更安全的获取方式
print(tag.attrs)  # 获取所有属性的字典

# 获取标签文本
print(tag.string)  # 如果标签内只有文本，返回该文本
print(tag.get_text())  # 获取标签内所有文本（包括子孙标签的文本）
  • 三种数据提取方式
    1. 文档树遍历与节点导航

    这种方法按照HTML的嵌套结构进行导航

    # 父子节点关系
soup.head.title  # 嵌套选择：获取head标签下的title标签
soup.p.contents  # 获取p标签的所有直接子节点列表
soup.p.children  # 获取直接子节点的迭代器
soup.p.descendants  # 获取所有子孙节点的迭代器（包括嵌套标签）
soup.a.parent  # 获取直接父节点
soup.a.parents  # 获取所有祖先节点的迭代器

# 兄弟节点关系
soup.a.next_sibling  # 下一个兄弟节点
soup.a.previous_sibling  # 上一个兄弟节点
soup.a.next_siblings  # 后面所有兄弟节点的迭代器
soup.a.previous_siblings  # 前面所有兄弟节点的迭代器
    1. 搜索文档树（find与find_all）
    # 基本用法
soup.find_all('a')  # 查找所有<a>标签
soup.find('div')  # 查找第一个<div>标签

# 多种过滤方式
soup.find_all(['a', 'div'])  # 查找所有<a>和<div>标签
soup.find_all(id='header')  # 按id属性查找
soup.find_all(class_='item')  # 按class查找（注意class后的下划线）
soup.find_all(attrs={'data-type': 'news'})  # 按自定义属性查找

# 文本内容过滤
soup.find_all(text='首页')  # 按文本内容查找
soup.find_all(text=re.compile('Python'))  # 使用正则表达式匹配文本

# 更精细的控制
soup.find_all('li', limit=3)  # 只返回前3个结果
soup.find_all('div', recursive=False)  # 只查找直接子节点
    1. css选择器（select）
    # 基本选择器
soup.select('div')  # 所有<div>标签
soup.select('.item')  # class为item的所有元素
soup.select('#header')  # id为header的元素

# 组合选择器
soup.select('div.item')  # class为item的<div>标签
soup.select('li > a')  # 所有直接是<li>子标签的<a>标签
soup.select('div p')  # 所有在<div>内的<p>标签（不限嵌套层级）

# 属性选择器
soup.select('a[target="_blank"]')  # 所有target为_blank的链接
soup.select('input[type="text"]')  # 所有类型为text的输入框

# 获取单个元素
soup.select_one('.content')  # 只返回第一个匹配的元素
  • 不同解释器的不同点

BeautifulSoup自动将输入文档转换为Unicode，输出转换为UTF-8，一般无需手动处理编码。

案例：

  • 提取豆瓣电影信息
    import requests
from bs4 import BeautifulSoup

def scrape_douban_movies():
    url = "https://movie.douban.com/chart"
    headers = {
        'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36'
    }
    
    # 获取网页内容
    resp = requests.get(url, headers=headers)
    soup = BeautifulSoup(resp.text, 'lxml')
    
    movies = []
    
    # 查找所有电影条目（根据实际页面结构调整选择器）
    items = soup.find_all('div', class_='pl2')  # 示例选择器，需根据实际情况调整
    
    for item in items:
        # 提取电影名称
        name_tag = item.find('a')
        name = name_tag.get_text().strip() if name_tag else "未知"
        
        # 提取链接
        link = name_tag['href'] if name_tag and name_tag.has_attr('href') else "未知"
        
        # 提取评分
        rating_tag = item.find('span', class_='rating_nums')
        rating = rating_tag.get_text() if rating_tag else "暂无评分"
        
        movies.append({
            'name': name,
            'link': link,
            'rating': rating
        })
    
    return movies

# 使用示例
movies = scrape_douban_movies()
for i, movie in enumerate(movies[:5], 1):  # 只显示前5部电影
    print(f"{i}. 电影名称: {movie['name']}")
    print(f"   链接: {movie['link']}")
    print(f"   评分: {movie['rating']}\n")
