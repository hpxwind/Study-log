  • 关键帧动画
    • animation复合属性

      animation 是多个动画属性的简写

      元素 { /* 语法 */
  animation: 名称 时长 速度曲线 延迟 播放次数 方向 填充模式 播放状态;
}
如：
.box {
  animation: move 2s ease 0.5s infinite alternate forwards running;
}

      可拆分为8个单独属性，每个属性控制动画的不同维度：

      1、animation-name：关联关键帧动画
.box {
  animation-name: move; /* 关联名为 "move" 的 @keyframes 动画 */
}

2、animation-duration：动画持续时间
.box {
  animation-duration: 2s; /* 动画持续2秒 */
}

3、animation-timing-function：动画速度曲线
.box {
  animation-timing-function: ease-out; /* 结束时减速 */
}
ease：默认，先慢→快→慢（自然过渡）。
linear：匀速运动（适合机械感动画）。
ease-in：开始慢，逐渐加快（如物体下落）。
ease-out：开始快，逐渐减慢（如物体减速停止）。
ease-in-out：开始和结束慢，中间快。
cubic-bezier(n,n,n,n)：自定义贝塞尔曲线（通过 cubic-bezier.com 生成）。
steps(n, start/end)：分步动画（如帧动画，n 为步数）。

4、animation-delay：动画延迟时间
.box {
  animation-delay: 0.5s; /* 延迟0.5秒后开始动画 */
}

5、animation-iteration-count：动画播放次数
.box {
  animation-iteration-count: infinite; /* 无限循环 */
}
数字（如 3 表示播放 3 次）。
infinite：无限循环播放。

6、animation-direction：动画播放方向
.box {
  animation-direction: alternate; /* 正向→反向交替播放 */
}
normal：默认，每次循环都从 0% 到 100% 正向播放。
reverse：每次循环都从 100% 到 0% 反向播放。
alternate：奇数次正向播放，偶数次反向播放（如 “去→回→去→回”）。
alternate-reverse：奇数次反向播放，偶数次正向播放。

7、animation-fill-mode：动画结束后的状态
.box {
  animation-fill-mode: forwards; /* 动画结束后保持最后一帧状态 */
}
none：动画结束后，元素回到初始样式（不保留关键帧状态）。
forwards：动画结束后，元素保留最后一帧（100% 或 to）的样式。
backwards：动画延迟期间，元素提前应用第一帧（0% 或 from）的样式。
both：同时应用 forwards 和 backwards 的效果。

8、animation-play-state：动画播放状态
.box:hover {
  animation-play-state: paused; /* 鼠标悬停时暂停动画 */
}
running：播放动画。
paused：暂停动画（暂停时保留当前状态）。

    @keyframes规则，其用于定义动画的关键帧状态。

    /* 定义动画名称为 "move" */
@keyframes move {
  /* 开始状态（0% 或 from） */
  0% {
    transform: translateX(0);
    opacity: 1;
  }
  /* 中间状态（可选，可多个） */
  50% {
    transform: translateX(100px);
    opacity: 0.5;
  }
  /* 结束状态（100% 或 to） */
  100% {
    transform: translateX(200px);
    opacity: 1;
  }
}

1、动画名称：自定义名称（如 move、fadeIn），后续通过 animation-name 关联到元素。
2、关键帧百分比：0%（开始）到 100%（结束），表示动画进度，from 等价于 0%，to 等价于 100%。
3、样式规则：每个关键帧内定义元素在该时间点的 CSS 属性（如 transform、opacity、background 等），浏览器会自动计算中间帧的过渡效果。
  • transition()
    transitoin: [属性名] [过渡时间] [过渡函数] [延迟时间];
对应单个属性：
transition-property: width;        /* 要过渡的属性 */
transition-duration: 0.3s;         /* 过渡持续时间 */
transition-timing-function: ease;  /* 过渡速度曲线 */
transition-delay: 0.1s;            /* 延迟开始时间 */
.box {
	transition: width 0.3 ease 0.1;/* 属性名为all ，除了display: none -> block 其余都行*/
}
    单个解释：
1、transition-property：指定过渡的属性
具体属性名（如 width、background-color、transform 等）；
all：所有可过渡的属性都应用
过渡（常用，但性能略差，建议指定具体属性）；
none：不应用过渡（默认）。
.box {
  transition-property: width, background-color; /* 仅 width 和 background-color 有过渡效果 */
}

2、transition-duration：过渡持续时间
.box {
  transition-duration: 0.3s; /* 过渡持续 0.3 秒 */
}

3、transition-timing-function：过渡速度曲线
ease：默认值，先慢→快→慢（最自然）；
linear：匀速运动（适合机械感动画）；
ease-in：开始慢，逐渐加快（如物体下落）；
ease-out：开始快，逐渐减慢（如物体减速停止）；
ease-in-out：开始和结束都慢，中间快；
cubic-bezier(n,n,n,n)：自定义贝塞尔曲线（通过工具生成，如 cubic-bezier.com）。
.box {
  transition-timing-function: ease-out; /* 结束时减速 */
}

4、transition-delay：过渡延迟时间
.box {
  transition-delay: 0.2s; /* 延迟 0.2 秒后开始过渡 */
}
    多属性过度：
/* 同时过渡宽度（0.3s 匀速）和背景色（0.5s 先慢后快） */
.box {
  transition: 
    width 0.3s linear,
    background-color 0.5s ease;
}

.box:hover {
  width: 300px;
  background-color: red;
}

  • transform()

    1、平移

    函数说明示例
    translateX(x)沿 X 轴（水平方向）平移translateX(50px) 向右移动 50px
    translateY(y)沿 Y 轴（垂直方向）平移translateY(-20%) 向上移动自身高度的 20%
    translateZ(z)沿 Z 轴（纵深方向）平移（需配合 3D 场景）translateZ(100px) 向屏幕外移动 100px
    translate(x, y)同时沿 X、Y 轴平移translate(20px, 30%)
    translate3d(x,y,z)3D 空间内同时沿 X、Y、Z 轴平移translate3d(10px, 20px, 50px)

    百分比参数基于元素自身的宽高

    常用于元素居中（如 position: absolute; left: 50%; top: 50%; transform: translate(-50%, -50%);）

    2、放缩

    函数说明示例
    scaleX(x)沿 X 轴（水平方向）缩放scaleX(1.5) 宽度放大至 1.5 倍
    scaleY(y)沿 Y 轴（垂直方向）缩放scaleY(0.5) 高度缩小至 0.5 倍
    scaleZ(z)沿 Z 轴缩放（3D 场景，影响纵深方向大小）scaleZ(2) 纵深放大 2 倍
    scale(x, y)同时沿 X、Y 轴缩放（y 默认等于 x）scale(0.8) 宽高均缩小至 0.8 倍
    scale3d(x,y,z)3D 空间内同时缩放 X、Y、Z 轴scale3d(1.2, 1.2, 0.5)
    transform-origin指定中心点位置，默认为center transform-origin: left top; 从左上角开始

    缩放倍数 >1 为放大，0~1 为缩小，<0 为缩放并翻转（如 scaleX(-1) 水平翻转）
    缩放会影响元素的视觉大小，但不会改变其在文档流中占据的空间（布局不变）

    3、旋转

    函数说明示例
    rotate(angle)沿 Z 轴（平面内）旋转（默认）rotate(45deg) 顺时针旋转 45 度
    rotateX(angle)沿 X 轴（水平轴线）旋转（3D 效果）rotateX(90deg) 向前翻转 90 度
    rotateY(angle)沿 Y 轴（垂直轴线）旋转（3D 效果）rotateY(180deg) 左右翻转 180 度
    rotateZ(angle)同 rotate(angle)，沿 Z 轴旋转rotateZ(-30deg) 逆时针旋转 30 度
    rotate3d(x,y,z,angle)3D 空间内沿自定义轴旋转（x,y,z 为 0~1 的权重）rotate3d(1,1,0, 45deg) 沿对角线旋转 45 度
    transform-origin指定中心点位置，默认为center transform-origin: left top; 从左上角开始

    角度为正数时顺时针旋转，负数时逆时针旋转

    3D 旋转（rotateX/rotateY）需配合 perspective 属性（定义 3D 透视效果）才能显示立体感

    4、倾斜

    函数说明示例
    skewX(angle)沿 X 轴（水平方向）倾斜skewX(30deg) 上边缘向右倾斜 30 度
    skewY(angle)沿 Y 轴（垂直方向）倾斜skewY(-10deg) 右边缘向下倾斜 10 度
    skew(x, y)同时沿 X、Y 轴倾斜skew(20deg, 5deg)

    倾斜会改变元素的形状（如矩形变平行四边形），但不会改变元素的位置和大小。

    角度过大会导致元素严重变形，通常用于特殊视觉效果。

一个transform中可以写多个属性值。

/* 悬停时同时放大、上移、轻微旋转 */
.box {
  width: 100px;
  height: 100px;
  background: aquamarine;
  transition: transform 0.3s; /* 配合过渡实现平滑动画 */
}
.box:hover {
  transform: scale(1.2) translateY(-10px) rotate(5deg);
  /* 顺序：先放缩，再平移，最后旋转 */
}
